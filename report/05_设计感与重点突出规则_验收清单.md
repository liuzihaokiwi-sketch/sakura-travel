# 设计感与重点突出规则 + 验收清单

## 1. 情绪价值怎么来
在这套攻略里，情绪价值不主要靠多写文案，而主要靠：
- 用户先被理解
- 重点先被看见
- 重要对象被认真展开
- 页面层次有呼吸感
- 章节与重点有“被策展过”的感觉

---

## 2. 全书统一的排版原则

### 原则 1：一页一个主焦点
每页必须能回答：这页最重要的东西是什么。
如果答不上来，这页就还没设计好。

### 原则 2：视觉层级只保留 3 层
- 主焦点
- 次焦点
- 辅助信息
不要做 5 层以上的层级混战。

### 原则 3：重点不是靠加粗堆出来
应主要通过：
- 位置
- 尺寸
- 留白
- 对比
- 分组
来形成视觉层级。

### 原则 4：重要判断尽量卡片化
“为什么是它”“为什么这样排”“先保什么先砍什么”都更适合做成判断卡，而不是长段。

### 原则 5：留白是价值感的一部分
用户感到“高级”，很多时候不是因为花哨，而是因为信息没有挤在一起。

---

## 3. 哪些地方必须有设计感

### A. 封面 Hero
必须有仪式感。

### B. 目录
必须让用户有“这本东西结构很清楚”的感觉。

### C. 偏好兑现页
必须一眼看出“这是为我定的”。

### D. 章节开场页
必须有阶段氛围感。

### E. 主要活动页
必须让用户一眼知道“这就是重点”。

### F. 主要酒店 / 主要餐厅 / 主要拍摄页
必须让用户感到这些选择是经过判断的，不是拼凑的。

### G. 每日执行页的 Day 标题区
必须有每天不同的节奏感和情绪目标。

---

## 4. 每类重点怎么突出

### 封面
靠：
- 大标题
- 图像裁切
- 低噪音背景
- 关键词轻点缀

### 偏好兑现页
靠：
- 对照式布局
- 明确因果关系
- 每条结论单独成块

### 大路线页
靠：
- 路线图作为视觉主角
- 文本退后
- 阶段分色 / 分段

### 每日执行页
靠：
- Day Hero 区
- 今日主线摘要
- timeline 中的关键锚点

### 活动 / 酒店 / 餐厅 / 拍摄页
靠：
- 对象标题 + 一句定性
- 选择理由块
- 时间窗 / 位置 / 体验卖点中的一项作为绝对主焦点

### Plan B 页
靠：
- 决策顺序
- 先保 / 先砍对照
- fallback 路径可视化

---

## 5. 不同页型适合的版式倾向

### 更适合“英雄区 + 内容区”
- 封面
- 章节页
- 主要活动页
- 拍摄主题页

### 更适合“信息块分层”
- 偏好兑现页
- 酒店策略页
- 酒店详情页
- 餐厅详情页
- Plan B 页

### 更适合“流程 / 轨道”
- 大路线页
- 每日执行页
- 交通页

### 更适合“卡片网格”
- 目录页
- 主要活动总表
- 顺路补充页
- 次要对象合并页

---

## 6. Web 与 PDF 的排版差异处理

### Web
- 允许轻交互
- 允许 anchor 跳转
- 允许 lazy render 非首屏页
- 可以有 sticky 目录或章节导航

### PDF
- 重点是页内结构稳定
- 所有重点必须直接可见
- 不依赖 hover、展开、点击

### 共同要求
- 先有稳定 page plan
- 再有各页 page template
- 不允许 screen 和 print 内容顺序不同

---

## 7. 实现完成后必须检查的内容

### A. 结构检查
- 固定前置页是否完整
- 每日执行页是否每天都有
- 主要活动是否都被单独展开
- 章节页是否只在需要时出现
- 是否存在阅读任务混页

### B. 视觉检查
- 每页主焦点是否明确
- 是否有页面过满
- 是否有多页连续长段文字
- 目录是否好扫
- 章节页是否与普通页有区分感

### C. 内容检查
- 偏好兑现是否真实对应用户输入
- 每一类详页是否真讲了“为什么是它”
- 是否出现只是重复 daily timeline 的专题页
- 是否有重要对象被埋没在补充页

### D. 技术检查
- Web 首屏渲染是否稳定
- PDF 是否分页稳定
- 页码与目录是否对应
- 图片是否裁切正常
- 超长标题是否破版
- 中英文混排是否溢出
- print CSS 是否与 screen 结构一致

### E. 体验检查
- 7 天版本是否显得完整但不累
- 14 天版本是否显得层次清晰而不重复
- 用户是否能在 30 秒内理解整本攻略结构
- 用户是否能在 10 秒内看到“这是按我偏好做的”

---

## 8. 最后的取舍原则
当时间不够、资源不够时，优先保：
1. page plan 稳定
2. 每页主焦点明确
3. 偏好兑现页做强
4. 每日执行页做稳
5. 主要活动 / 酒店 / 餐厅 / 拍摄至少各有 1 个优秀模板

最后再做：
- 更复杂的装饰
- 更多动画
- 更花的背景
- 更重的视觉实验
