# 产品范围

## 三档定价详情

### 免费试用（¥0）
- **用户获得**：3 天粗略行程概览（Day 1 完整 + Day 2-3 摘要）
- **不包含**：详细时间安排、餐厅推荐、交通指引、PlanB、酒店推荐
- **目的**：让用户感受到行程质量 → 引导加微信 → 转化为付费
- **交付方式**：网页在线浏览（`/plan/[id]`）

### 标准版（¥248，首发原价 ¥368）
- **用户获得**：完整 5-7 日行程 PDF + H5 在线版
- **包含**：每天时间轴（09:00-21:00）、景点推荐理由、餐厅精选、交通路线、PlanB 雨天方案
- **精调**：1 次结构化修改（替换景点/餐厅/调整节奏）
- **交付方式**：微信发送 PDF + 在线链接

### 尊享版（¥888）
- **用户获得**：标准版全部 + 深度管家服务
- **额外包含**：酒店推荐（含预订链接）、特色体验预约指导、紧急联络卡、行李清单
- **精调**：3 次结构化修改
- **专属客服**：旅行期间微信在线答疑
- **交付方式**：微信 + 专属群

## 用户流程

```
1. 首页了解服务 (/)
2. 填写 5 步问卷 (/quiz)
   - 出行城市 + 天数
   - 同行人（情侣/家庭/闺蜜/独旅）
   - 偏好标签（美食/购物/文化/自然...）
   - 旅行日期
   - 微信号
3. 提交成功页 (/submitted)
   - 扫码加客服微信
4. 微信沟通 → 确认需求 → 付款
5. 规划师生成行程 → 内部审核
6. 微信交付链接 (/plan/[id])
7. 用户反馈 → 精调（如有）
```

## 不做的事

- ❌ 不做机票/酒店预订（只推荐）
- ❌ 不做实时导航
- ❌ 不做多国支持（仅日本）
- ❌ 不做 B2B 旅行社合作
