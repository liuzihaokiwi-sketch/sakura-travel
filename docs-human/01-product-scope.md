# 产品范围

## 三档定价与交付物

### 一日体验版（免费）
- **交付物**：3-5 页预览，包含第一天完整行程
- **内容**：Day 1 逐时间段路线、2-3 个景点推荐理由、当天交通指引
- **目的**：让用户零成本体验攻略质量和细致程度
- **限制**：不含剩余天数、餐厅推荐、酒店建议

### 完整攻略（¥248 首发价，原价 ¥368）
- **交付物**：30-40 页完整行程手册（H5 网页 + 可导出长图/PDF）
- **内容**：
  - 全程逐日行程，精确到每个时间段
  - 每餐推荐 + 备选餐厅（含人均、是否需预约）
  - 酒店区域建议 + 选择理由
  - 交通最优方案 + 省钱技巧（Suica/JR Pass 计算）
  - 拍照攻略 + 最佳时段
  - 避坑指南 + 出行前准备清单
  - Plan B 备选方案（雨天/排队/临时关闭）
  - 预订优先级提醒 + 全程预算参考
- **精调**：含 2 次免费行程微调

### 尊享定制版（¥888）
- **交付物**：40-50 页深度手册 + 全程 1v1 服务
- **额外包含**：1v1 需求沟通、不限次精调、出行期间实时答疑、隐藏小众目的地、高端餐厅酒店精选、蜜月/纪念日特别安排

## 用户流程

```
首页(page.tsx) → 问卷(quiz) → 提交成功页(submitted) → 微信联系规划师
   → 免费预览(plan/[id]?mode=preview) → 付费 → 完整交付(plan/[id])
   → 可选精调 → 可选升级管家版(¥888)
```

## 支持的目的地

东京、大阪+京都、东京+大阪+京都、北海道、冲绳等 12 个城市，支持 3-8+ 天行程。

## 同行类型

独行、情侣/夫妻、带孩子、带父母、朋友闺蜜——每种类型有不同的行程风格和节奏调整。
