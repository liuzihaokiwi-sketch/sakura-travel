# Travel AI — 项目概览

## 项目定位

Travel AI 是面向中国出境游客的日本旅行定制服务。用户填写 3 分钟问卷，规划团队（AI + 人工审核）生成个性化行程攻略，以杂志级 PDF/H5 交付。

## 解决的痛点

1. **信息过载**：小红书/马蜂窝攻略碎片化，自己拼行程耗时 8-20 小时
2. **模板化陷阱**：传统旅行社套餐千篇一律，不匹配个人偏好
3. **时效性差**：网上攻略大量过期信息（闭店/翻修/季节变化）
4. **语言障碍**：日文地名/交通/预约让新手望而却步

## 目标用户

- 首次赴日的中国情侣/家庭/闺蜜团
- 有日本自由行经验但想省时间的回头客
- 樱花季/红叶季等旺季出行者

## 商业模式

- **免费试用**：3 日粗略行程，引导加微信
- **标准版 ¥248**：完整 5-7 日行程 + 1 次精调
- **尊享版 ¥888**：深度行程 + 3 次精调 + 人工审核 + 专属客服

## 交付流程

```
用户填问卷 → 加客服微信 → 规划师生成 → 内部审核 → 微信交付 → 用户反馈精调
```

## 核心理念

**用户感知是"人做的"而非"AI 生成的"**，所有对外文案不出现"AI"字样。
