## 根目录文件

- `CLAUDE.md` — **AI 每次会话必读**的总入口(项目基本信息 + 工作类型导航 + 红线)
- `README.md` — GitHub 首页展示
- `目录说明.md` — 你在看的这份

其他根文件(Dockerfile / docker-compose.yml / pyproject.toml / alembic.ini / .env* / .gitignore 等)都是**工具约定必须在根目录**的,不能搬。
