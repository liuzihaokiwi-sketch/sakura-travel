## S+ 级活动

### 樱花满开（4月初，约10天窗口）

**加法：0额外时间，大阪城自动升级**

大阪城本来就在Day 2，春天去就是赏樱版。唯一要加的是**夜樱**（18:00-21:00），可以放Day 2晚上，砍P5晚间选项。

| 时间 | 安排 |
|------|------|
| Day 2中午 | 大阪城赏樱（自动升级，西之丸庭园350円，300棵染井吉野） |
| Day 2晚上 | 大阪城夜樱（18:00亮灯，至21:00，最晚20:30入园）— 砍P5晚间选项 |

- 怎么去：JR大阪城公园站步行即到
- 注意：满开期日均数万人，17:30前到比18:00后好很多

---

### 天神祭（7/24-25，固定日期）

**加法：加一天，或占Day 4**

日本三大祭，100万人，下午到晚上（14:00-21:00）。不可能塞进Day 2——上午去完黑门市场+大阪城，下午不够时间赶到天满桥占位。

**方案A（推荐）：加一天**，大阪从4天变5天。Day 2照常，天神祭作为额外Day。

**方案B：占Day 4**，人群版内容整体后移或砍。

| 时间 | 安排 |
|------|------|
| 上午 | 正常排别的（黑门市场/购物等） |
| 14:00 | 到达天满桥/樱之宫站，先吃屋台（500-1000家） |
| 15:30 | 占好河边花火观赏位（17:00后好位置全没） |
| 15:30 | 看陆渡御（3000人陆上大游行） |
| 18:00 | 船渡御（100余艘船队沿大川巡游） |
| 19:30-21:00 | 奉纳花火（约3000发） |
| 20:45 | 提前撤离（21:00散场各站极度拥挤） |

- 怎么去：地铁「南森町站」或JR「大阪天满宫站」；花火看JR「桜ノ宮站」
- 最佳观赏位：桜之宫公园（开阔）、天满桥附近川边（近距离看船）
- 付费席：JTB运营，船上席每年秒空，4月上旬开售
- 屋台：天神橋筋商店街+天满宫境内+桜之宮公园沿线，19:00花火开始时买东西最不挤
- **注意：花火结束后各站等30min+，建议20:45提前走或22:00再走**
- **注意：当天桥梁18:00-23:00关闭，必须坐电车**

## S 级活动

### 大相扑春场所（3/8-22，15天）

**加法：加到Day 2下午，砍P4新世界+通天阁**

大相扑12:00-18:00在EDION Arena，难波站步行4分钟。Day 2上午黑门市场+大阪城照常，下午去相扑代替新世界。新世界移到Day 4上午或砍掉。

| 时间 | Day 2变为 |
|------|----------|
| 上午 | 黑门市场 → 大阪城（不变） |
| 12:00-18:00 | 大相扑（砍新世界+通天阁） |
| 傍晚 | 法善寺横丁（不变） |

- 会场：EDION Arena Osaka，难波站步行4分钟（就在酒店旁边！）
- 票价：椅子席3,500-9,000円，枡席4人约40,000円
- 买票：1月底-2月初开售，官网 sumo.pia.jp，热门日秒光
- 只看顶级：13:30-14:00入场即可，不需要12:00就到
- 遇力士：会场走廊容易擦肩而过；场所前2周各部屋来大阪训练，部分开放早朝稽古观看（8-9点）
- 不买票替代：相扑堂Hirakuza（Namba City 8F）每晚有真力士表演
- **小技巧：平日工作日票便宜且好买，周末和千秋楽最先售罄**
- **被砍的新世界+通天阁可以移到Day 4上午**

### 造幣局桜の通り抜け（4/9-15，仅1周）

**加法：塞进Day 2上午，不需要砍任何东西**

只要30-40min，黑门市场吃完早餐→地铁去天满桥→造幣局→再去大阪城，时间完全够。

| 时间 | Day 2上午变为 |
|------|-------------|
| 9:00 | 黑门市场早餐（不变） |
| 10:00 | 造幣局通り抜け（加入，30-40min） |
| 11:00 | 大阪城（不变） |

- **必须网上预约**，不接受现场排队。专用网站 sakura-osaka.mint.go.jp
- 免费入场
- 140种347棵八重樱（不是普通染井吉野，是稀有品种）
- 怎么去：京阪/地铁「天满桥站」步行15min
- **与大阪城赏樱时间错开**：大阪城染井吉野4月初满开，造幣局八重樱4月中旬满开，正好接力

### 淀川花火大会（8月第1个周六 或 10月中旬）

**加法：加到当天晚上，砍P5晚间选项**

花火17:30入场、19:30-20:30放，只占晚上。Day 2或Day 4白天照常，晚上去花火。

> 注意：2025年起因万博影响改为秋季（10月），2026年具体日期待确认

| 时间 | 安排 |
|------|------|
| 白天 | 正常行程不变 |
| 17:30前 | 到达会场（18:30后可能无法入场） |
| 19:30-20:30 | 花火（梅田楼群为背景，500m星型连发） |
| 20:30-21:00 | 原地等20min再移动（散场十三站极挤） |

- 有料席：十三会场右岸，パノラマ席5,000円 / プレミア席18,000円 / 纳凉船27,000円
- 免费区：梅田侧全面禁止，只能十三侧
- 怎么去：JR塚本站东口或阪急十三站西口，步行15min
- **如果是Day 4情侣版，花火代替蓝天大厦夜景（花火更好）**

### 岸和田花车祭（9/19-20）

**加法：加一天，或占Day 4**

要去岸和田（难波25min），下午+晚上，回来已经很晚。

**方案A（推荐）：占Day 4**，人群版内容砍掉，全天给花车祭。

**方案B：加一天**，大阪变5天。

| 时间 | 安排 |
|------|------|
| 上午 | 正常排别的/睡懒觉 |
| 12:00 | 南海线到岸和田站（难波出发约25min） |
| 14:00-16:00 | "やりまわし"急转弯高潮（4吨花车时速20-30km冲弯） |
| 19:00-22:00 | 提灯曳行（夜间版，视觉唯美） |

- 最佳观看：カンカン場（急转弯处），有料席3,000-8,000円
- **安全注意：不要站路中间，花车极重极快；不可打伞（穿雨衣）；带小孩/老人买有料席**
- 怎么去：南海本线「岸和田站」

### 红叶季 — 箕面（11月中旬-12月初）

**加法：加到Day 4上午，砍Day 4第一个slot或压缩时间**

往返2h（含赏景），从梅田出发25min到箕面站。

**Day 4情侣版为例：** 上午去箕面红叶（10:00-12:00），下午14:00海游馆照常。砍掉的是睡懒觉时间。

**Day 4闺蜜版为例：** 箕面红叶代替中之岛上午（红叶季中之岛也有银杏但不如箕面），下午美国村照常。

| 时间 | 安排 |
|------|------|
| 10:00 | 阪急箕面站出发 |
| 10:00-10:40 | 滝道散步（沿溪谷2.8km） |
| 10:40-11:00 | 箕面大瀑布+红叶 |
| 11:00-11:30 | 回程，吃もみじの天ぷら（红叶天妇罗，名物必吃） |
| 12:00 | 回到市区，继续Day 4下午行程 |

- 怎么去：梅田乘阪急宝塚线→石橋阪大前换箕面线，约25min
- 注意：红叶期周末停车场早晨即满，坐电车去

### OSAKA光のルネサンス（12/14-25）

**加法：加到当天晚上，砍P5晚间选项**

只占晚上17:00-19:00。Day 2或Day 4白天照常。

**Day 4闺蜜版最适合：** 下午美国村→傍晚直接走到中之岛看灯光（美国村到中之岛步行15min），代替梅田购物+夜景。

| 时间 | 安排 |
|------|------|
| 白天 | 正常行程不变 |
| 17:00 | 到淀屋桥站/从美国村步行到中之岛 |
| 17:00-17:30 | 市役所正面灯光 |
| 17:30-18:00 | 中之岛欅树大道illumination（150米） |
| 18:00-18:15 | 中央公会堂投影映射（每12min一场，每场约9min） |
| 18:15-19:00 | 沿中之岛散步，拍照 |

- 全程免费
- 怎么去：地铁/京阪「淀屋桥站」步行1-5min
- 注意：12/24-25极度拥挤，平日17:00-18:00到场最舒适
- 雨天投影正常，荒天暂停

## AI匹配逻辑

```
输入：用户旅行日期范围 + 大阪天数
基础天数：2.5天（不含USJ）/ 3.5天（含USJ）

匹配规则：
1. 检查日期是否落在S+/S/A级活动窗口内
2. 不需要额外时间的（樱花满开、造幣局、淀川花火、光のルネサンス）→ 直接加进日程
3. 需要半天的（大相扑、箕面红叶）→ 砍Day 2的P4（新世界+通天阁），被砍内容移到Day 4上午
4. 需要整天的（天神祭、岸和田花车祭）→ 加一天或占Day 4
5. 时间不够时，按优先级从P5→P4→P3依次砍常规景点腾时间
6. A级作为推荐项告知用户，不强制加入
7. B级不主动推荐，住难波自然路过的（道顿堀灯光秀）自动享受
```
