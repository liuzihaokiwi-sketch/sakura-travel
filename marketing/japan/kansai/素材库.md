# 第一部分:说明书(必读)

## 这份文档是什么

做关西模板和调研时,我们会搜到/想到**一些小红书攻略号没写过、但用户会觉得"哇原来这样"的素材**。这些素材是产品的护城河,但以前散落在 Claude Code 会话里,做完模板就丢了。

**这份文档 = 所有这类素材的统一收纳地**。

## 为什么需要这份

- **调研/做模板时产生素材** → 当时不记录,几天后忘
- **小雅做内容时需要素材** → 她不会翻模板 JSON / 会话记录
- **素材分散在 fun_fact / curators_notes / photo_spots** → 散在几十个模板里,没法按主题找

把素材**从创作过程中剥离出来,统一索引**,两边都方便。

## 谁在什么时候该写这份

### 写的时机(3 种场景)

**场景 1:调研时看到"这个我好像没在攻略里见过"**
- 典型:搜小红书爆款看到某个小众视角;WebSearch 到日本本地杂志的洞察
- 不用等完整搜索,**当下就往这份文档里丢一行**

**场景 2:做模板时写 photo_spots / fun_fact / curators_notes / nearby_shops 时的灵光一现**
- 典型:写清水寺 photo_spots 时想到"子安塔背后视角游客不知道";写 fun_fact 时查到金阁寺三岛由纪夫纵火案
- 写完那个字段后,**顺手复制一份丢素材库**(字段里留在那,素材库这边换成内容运营视角写)

**场景 3:数据采集时发现某景点/餐厅有可发小红书的冷知识**
- 典型:核查月桂冠酒藏营业时间时发现"2026/7/1 起改无现金支付";核查后发现某景点季节限定菜品
- **带强传播性的冷知识丢素材库**(常规事实留 live_facts 即可)

### 不写的内容

- ❌ 常识性的(早起避人潮 / 关西机场到京都怎么走) — 所有攻略都有
- ❌ 产品硬事实(票价/开门时间) — 那是 live_facts 的事
- ❌ 模板执行细节(几点去几点离开) — 那是模板的事
- ❌ 用户私人体验 — 这份是通用素材,不是游记

**判断标准**:一个月后某 AI 打开这份文档,**能立刻用这条素材起一个小红书选题**的,才是合格素材。

## 每条素材的标准格式

```markdown
### {素材标题一句话}
**主题**: {京都小众视角 / 祭典深度 / 出片机位 / 本地冷知识 / 餐厅隐藏 / ...}
**来源**: {调研什么的时候 / 哪个模板 / 搜了什么}
**为什么好**: {1-2 句解释为什么这条是"攻略没写过的"}
**配图方向**: {小雅拍/找图时的画面建议}
**可延展选题**: {一条 → 可以写成什么样的小红书标题}
**关联**: {关联的模板 ID / entity / 爆款参考编号}
```

**字段解释**:
- **主题**:用下面"素材主题词表"的现有分类,避免每人一套
- **来源**:后来的人想追溯能找到
- **为什么好**:**这是最重要的字段**——说不清楚为什么好,这条就不是好素材
- **配图方向**:小雅拍图时的具体画面,不是"拍得好看"这种废话
- **可延展选题**:点明它能撑起什么内容,不只是"一条冷知识"
- **关联**:反向查找用(做类似模板时能再次捡起)

## 素材主题词表(避免每人一套分类)

必须用下面这些现有主题,不要自创:

| 主题 | 用于什么素材 |
|---|---|
| `京都小众视角` | 大家都去的景点但有本地人才知道的角度/玩法 |
| `大阪小众视角` | 同上,大阪版 |
| `关西小众城市` | 奈良/神户/宇治/姬路等不在主骨架的城市深度 |
| `祭典深度` | 祇园祭/天神祭/五山送火等祭典的"进阶玩法" |
| `季节限定` | 只在特定季节/日期才有的体验(梅花季/红叶夜景/蟹会席) |
| `出片机位` | 摄影视角/光线/时段的具体建议 |
| `本地冷知识` | 历史故事/文化背景/本地人才知道的来龙去脉 |
| `餐厅隐藏` | 餐厅的"招牌菜之外的小众单品/点菜诀窍/预约技巧" |
| `酒店体验` | 某个酒店的独特体验点(不是基础信息) |
| `动线优化` | 避开人流/避开排队/顺路小发现 |
| `文化礼仪` | 寺院规矩/温泉礼仪/交通礼仪的有趣点(不是硬约束) |
| `避坑警示` | 某个常见攻略推荐但其实不值得的,或游客陷阱 |
| `交通窍门` | pass/换乘/打车/巴士的具体小技巧 |

**主题不够用就先丢"其他",等积累到 5+ 条再决定是不是新开一个主题**。

## 怎么用这份(小雅 / AI 做内容时)

**起选题**:
1. 按主题浏览 → 看有没有"这几条组合起来能写一篇"
2. 按日期(新到旧)浏览 → 最近的素材最新鲜
3. 按关联模板/城市浏览 → 做某个城市主题时用

**写内容时**:
- 从素材库取 3-5 条 → 每条延展成小红书一页
- 配图方向字段直接指导拍图/找图
- 保留"为什么好"作为**自己的文案判断标尺**(这才是产品护城河)

**反向回填**:
- 小雅发出去的小红书爆款了 → 在这条素材下标"已发 xxx 链接,效果 yyy"
- 发现某素材不如预期 → 标"尝试过效果一般",后续同类不用

## 维护规则

- **不删**,素材只增不减(差的素材加"⚠️ 效果一般"标注)
- **追加在对应主题章节底部**,不插入中间(保持按写入时间顺序)
- **一个月梳理一次**:看哪个主题膨胀到 20+ 条,考虑拆子文件
- 素材总数 > 200 时考虑拆分(按主题拆,每主题一文件)

**来源**: 2026-04-20 东山 core 深化调研 / 小红书"山鬼"笔记

### 花见小路"公道 vs 私道"撮影规则——2019 起罚 1 万日元的真相
**主题**: 京都小众视角 / 文化礼仪
**来源**: 2026-04-20 东山 core 事实核查(修正 SOP 规则 16 的错误表述)
**为什么好**: 几乎所有攻略都模糊说"花见小路不能拍艺伎",事实是:**花见小路通本身是公道**,可以走、可以拍街景;**周边小路是私道**(小袖小路等),2019/10/25 起立牌"无许可撮影 1 万日元",2024 加"通り抜け禁止"。罚款无法律拘束力但地元协议会巡逻。真正该强调的是"追拍舞妓 + 进私道"两件事,不是整条花见小路禁拍——写对了才是"懂当地的朋友"
**配图方向**: 花见小路 vs 旁边私道的立牌对比(日英中三语 + 1 万日元警示)
**可延展选题**: "花见小路禁拍是假的——真正罚钱的是这几条路"/"去祇园前必看这张图"
**关联**: kyo_higashiyama__core.json / kyo_gion_kamogawa__slow_day.json(修正 curators_notes)

### 八坂通下坡段——八坂之塔从町屋缝隙"突然出现"的那个转角
**主题**: 京都小众视角 / 出片机位
**来源**: 2026-04-20 东山 core
**为什么好**: Instagram 那张"石板路尽头五重塔"的经典画面就是八坂通下坡段,但攻略常把它和"法观寺塔"混为一谈,其实关键不是塔本身,是"从町屋屋顶缝隙冒出来"的那个突然感——下午 15:00 后游客量下降、顺光。真正的机位在二年坂下来往东走 150m,不是近距离

### 伏见稻荷傍晚6点上山——颠覆"必须清晨去"的最反直觉洞察
**主题**: 京都小众视角 / 动线优化
**来源**: 2026-04-20 调研 / 小红书"月亮"笔记(205赞151收藏)
**为什么好**: 所有攻略都说"伏见稻荷清晨5-7点去,人最少",但傍晚6点上山是另一个完全被忽视的窗口:山脚游客已经散场,山腰观景台可以俯瞰整个京都的夕阳+城景,比清晨多了光线色彩。神社24小时开放,夕阳时段的千本鸟居是橙红色光线打进来,和清晨逆光完全不同的视觉体验。下山时京都夜景也渐渐亮起
**配图方向**: 山腰观景台俯瞰京都盆地夕阳+夜景过渡,鸟居内打进橙色光线
**可延展选题**: "去伏见稻荷不用早起——本地人的时间是傍晚6点"/"清晨vs黄昏,伏见稻荷两种截然不同的体验"

### 伏见稻荷鸟居左边——不用早起也能拍到无人鸟居的秘密
**主题**: 出片机位 / 动线优化
**来源**: 2026-04-20 调研 / 小红书"机智七棒棒哒"笔记(603收藏)
**为什么好**: 入口第一段千本鸟居分左右两边走:右边游客挤满,左边是下山道几乎没人停留。12点去也能拍出无人鸟居。另一个秘密:山腰观景台(上山40分钟)俯瞰全京都视野极佳,大部分人体力不支到这就折返,平台反而是全程人最少+视野最好的地方

### 京都红叶时机真相——山区先红市区后红,宁早不晚
**主题**: 季节限定 / 动线优化
**来源**: 2026-04-20 调研 / 小红书"一葱"笔记(834收藏,4刷红叶季实战数据)
**为什么好**: 几乎所有攻略说"11月中下旬去京都红叶",但实际规律是:**山区(岚山/大原/贵船)先红约11月末,市区(东福寺/永观堂/南禅寺)后红约12月初**。两个时段各有主场,卡错时间等于全程踩空。另一个洞察:红叶初期红黄绿交织层次感比"全红期"更像油画,去晚了就是枯树枝。"宁早不晚"是4次实战后的结论
**配图方向**: 山区红叶(岚山竹林旁枫树)vs 市区红叶(东福寺通天桥溪谷)对比/初期层次感对比全红期
**可延展选题**: "去京都看红叶,你可能选错了时间"/"4刷红叶季数据:最佳时机真相"
**关联**: kyo_arashiyama__koyo / kyo_higashiyama__koyo / 当前状态.md 季节窗口

### 京都红叶Top12——王道必去vs岚山秘境vs本地私藏
**主题**: 季节限定 / 京都小众视角
**来源**: 2026-04-20 调研 / 小红书"一葱"笔记(1703收藏)
**为什么好**: 从"王道(东福寺/永观堂/南禅寺/醍醐寺)→岚山秘境(祐斋亭圆窗/大河内山荘/二尊院/常寂光寺)→本地私藏(神护寺/真如堂/无邻庵/京都御所银杏)"三层结构,比普通攻略"TOP10景点"深很多。关键洞察:岚山红叶真正好的不是天龙寺,是大河内山荘(旧名演员私人别墅,可喝抹茶俯瞰岚山,游客少很多)和祐斋亭(圆窗红叶需预约,价格贵但体验独一无二)
**配图方向**: 大河内山荘俯瞰岚山全景 / 祐斋亭圆窗红叶框景 / 真如堂三重塔+红枫
**可延展选题**: "岚山红叶不是天龙寺——本地人去哪里"/"京都红叶分三档,你去的是哪档"
