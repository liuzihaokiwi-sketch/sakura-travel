## 一、护城河（按强度+优先级）

### 当前已有（脆弱）

| 护城河 | 现状 | 强度 | 备注 |
|---|---|---|---|
| 速度 | 已有 | 弱 | AI 工具普及·2-3 年内追上 |
| 方法论沉淀 | 已有 | 中 | 14 坑+4 份规范+总流程·别人复制要踩同样坑 |

### 主动构建（按"难复制+用户感知"双维度排）

| 护城河 | 时长 | 强度 | 难度 | 优先级 |
|---|---|---|---|---|
| **品质（"懂当地的朋友"）** | 12-18 个月 | 强 | 高 | **P0** |
| **设计（实体+ kiwi IP）** | 6 个月 | 中 | 低 | **P0**（已启动） |
| **用户路径垄断** | 18-24 个月 | 极强 | 极高 | P1 |
| **内容飞轮**（pilot diary 反馈循环） | 24+ 个月 | 极强 | 中 | P1（pilot 阶段就启动） |

### 核心判断

- **不需要打赢大厂**——大厂卖电子攻略+商家投流·跟实体策展不同物种
- **目标 0.1% 用户**——10 万用户 × ¥298 = 3000 万·够好的小生意
- **真正的护城河是"独立性 + IP 陪伴感"**——大厂有商业利益冲突·做不出"无恰饭推荐"+"P-plan 跟你同行"

### 大厂 vs 我们对比

| 维度 | 大厂（携程/美团/飞猪/马蜂窝） | 我们 |
|---|---|---|
| 形式 | App 内电子攻略 | 纸质手账本+贴纸 |
| 收钱逻辑 | 商家投流抽佣 | 内容付费 |
| 数据立场 | 利益相关（推佣金高的） | 完全独立（不接恰饭） |
| 用户体验 | 用完就关 | 旅行带着+回家保存 |
| 品牌资产 | OTA 通用·无人格 | P-plan IP·拟人陪伴 |

## 三、P-plan 角色（IP）

### 来源

女朋友设计·完整设定见 `marketing/brand/kiwi/角色设定档案.docx`。

### 一句话定位

**"一只不会飞·但用脚把世界踩遍的几维鸟·替别人编织梦境的孤独旅行者。"**

### 核心人设

**身份**：女性·几维鸟·"逃亡之子"（出身桉叶森林·因稀有曾被抓走过）
**职业**：旅行策划师
**装备**：黑色墨镜（装酷+遮泪点）+ 草帽（盖几维秃毛）+ 双肩包

**人格关键词**：
- 防御型外向 — 爱省钱·爱攻略·容易感动到哭（墨镜挡住）
- 反英雄 — 自己不能飞·没钱·孤独·靠"为别人编织梦境"找到价值
- 圆滚滚反差萌 — 圆肚子+长睫毛+墨镜+红帽
- 真心藏在墨镜后 — 摘下墨镜=触发隐藏真情时刻

**梦想 OS**："世界很大·但我的本子能装下它。装下风景·装下机票·装下电影·装下心·那是个我创造的世界。"

### 为什么这个 IP 强

**1. 反英雄设定**
- 不是俯视的"专家"·是同行的"小伙伴"
- 用户买的不是攻略·是「跟一只懂你的小鸟同行」

**2. 视觉反差萌**
- 圆滚滚+长睫毛+墨镜+红帽——一眼记住·复刻难
- 不像泛 IP 的鸽子/熊·有独特记忆点

**3. 互动有钩**
- 「摘墨镜」=触发隐藏内容
- 「死亡气息感知」=Plan B 的拟人化
- 「冷笑话安慰」=出错时的情绪缓冲

**4. 故事内核能延伸**
- 每开一个新城市圈·P-plan 都有"第一次走访"的故事
- 用户买第二本·第三本·跟着 P-plan 走遍世界
- 这是**用户路径垄断**的载体

## 四、P-plan 在手账本中的互动设计

### 出现规则

**不是装饰性吉祥物·是用户的旅行同伴**——这是产品最重要的差异化。

### 出现位置 + 形态对照

| 场景 | 推荐动作 | 用户感受 |
|---|---|---|
| 封面 | kiwi_action_01（拍照） | "这是 P-plan 给你的本子" |
| 首次打开 | kiwi_action_17（挥手） | "Hi·我陪你这趟" |
| 章节首页 | kiwi_action_02（写字板） | "我帮你做的功课" |
| 标准 slot 页 | kiwi_action_08（行走背包） | "下一站..." |
| 出片机位 | kiwi_action_01（拍照） | "我推荐的角度" |
| 用心推荐 | kiwi_action_05（抱爱心·唯一彩色） | "这家是我专门挑的" |
| 高端餐厅 | kiwi_action_13（端酒杯） | "今晚仪式感" |
| 咖啡店 | kiwi_action_09（端咖啡） | "歇一歇" |
| 自由漫游段 | kiwi_action_15（躺平摸鱼） | "你慢慢逛" |
| 章节结束 | kiwi_action_11（躺平星星） | "今天圆满" |
| 旺季提醒 | kiwi_action_19（演讲台正式） | "重要提示" |
| 路线建议 | kiwi_action_16（手帕指路） | "我建议..." |
| Plan B / 雨备 | （待补图）「红帽歪一边·拿伞」 | "没事·还有 Plan B" |
| 旅程结束 | kiwi_action_18（蹲坐思考） | "下次去哪？" |

完整动作清单 → `marketing/brand/kiwi/动作清单.md`

### 互动哲学

**3 条原则**：

1. **P-plan 不说话·靠表情和姿态传达**
   - 不要"P-plan 提醒你..."这种说教口吻
   - 用 kiwi 的姿态+一句简短文案配合（如 kiwi 摸鱼图 + "你慢慢逛·我陪你"）

2. **隐藏内容用"摘墨镜"触发**
   - 标准页 P-plan 戴墨镜·有酷酷距离感
   - 冷知识/特别推荐页可用"摘下墨镜的 P-plan"（19 张里目前没·下批补）
   - 视觉暗示：这是 P-plan 真心推荐的

3. **彩色是稀缺·只在关键情绪点用**
   - 19 张里只有 kiwi_action_05（抱爱心）有珊瑚粉
   - 用在"用心推荐 / 重点强调 / 收藏"·不滥用

### 不做的事

- ❌ **不让 P-plan 推销**（"P-plan 推荐你买..."）— 违反"独立"品牌资产
- ❌ **不让 P-plan 抢用户主体感**（每页都有 P-plan = 喧宾夺主）
- ❌ **不用 P-plan 做营销噱头**（"P-plan 限定款"打折促销 = 廉价化 IP）

## 五、跨产品延伸（路径垄断）

### 用户买第二本的理由

不是"你产品好"·是"我已经习惯了 P-plan 的视角"。具体设计：

**1. 跨本子贴纸联动**
- 关西本配的贴纸·能贴到"下一本"日本/欧洲本上
- 用户收集 P-plan 旅行印章（每本一个）

**2. P-plan 故事跨本延伸**
- 关西本：P-plan 第一次走京都
- 北海道本：P-plan 在札幌跟松鼠交朋友
- 欧洲本：P-plan 第一次坐邮轮
- 用户跟着 P-plan 一站站走

**3. 老用户折扣**（已在业务流.md）
- 二刷新地区 8 折
- 集齐 N 本送限定贴纸

### 内容飞轮（pilot 阶段就启动）

```
pilot 用户 diary 反馈
    ↓ 沉淀爆款洞察
小红书种草（用户真实照片+P-plan 互动）
    ↓ 拉新
更多 pilot
    ↓ 越多用户越准
```

详见 [上线后/pilot反馈.md](../操作SOP/上线后/pilot反馈.md)。

## 六、品牌口吻（用户可见文字基线）

所有用户可见文字（手账正文 / 文案 / 客服话术 / 小红书运营）都呼应：

> **"一只懂当地的朋友·替你做完所有判断·也替你藏起所有担心。"**

### 口吻 dos

✅ **温柔·不卖弄**："今天去的这家·8:30 前到·能拍空舞台"
✅ **替用户决策完**："不用查·按这个走"
✅ **小心思**："如果你愿意为出片早起一次·就选这个变体"
✅ **真情藏在细节**："这家是 P-plan 摘下墨镜推荐的"

### 口吻 don'ts

❌ 攻略口吻："必去 TOP10 / 错过后悔"
❌ 销售口吻："限时 / 抢购 / 秒杀"
❌ AI 口吻："为您智能推荐 / 基于您的偏好"
❌ 教师口吻："请注意 / 务必 / 切记"
