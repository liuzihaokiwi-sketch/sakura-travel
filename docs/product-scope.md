## 2. 免费版预览边界

- **展示**: Day 1 完整内容（含时间线、景点推荐理由、餐厅、交通、避坑、Plan B）
- **锁定**: Day 2-7 只展示钩子信息（如"第 2 天将带你去一个连本地人都排队的地方"），不透露具体景点/餐厅/路线
- **目的**: 让用户"先爽到"→"意犹未尽"→ 付费解锁
- **CTA**: "先免费看一天 →"，点击跳转 `/quiz`

---

## 3. 修改规则

| 档位 | 次数 | 修改范围 | 不支持 |
|------|------|---------|--------|
| 免费 | 0 次 | 只读 | — |
| 标准版 | 1 次 | 替换景点 / 替换餐厅 / 调整节奏 | 不支持增减天数、不支持改城市 |
| 尊享版 | 3 次 | 同上 + 更灵活的调整 | — |

---

## 4. 天数与定价浮动

- 基准为 7 天版本
- 增加或减少天数，价格相应浮动（具体浮动规则待定）
- 免费版固定 1 天，不支持天数调整

## 5. PDF 交付规则

- **格式**: PDF + H5 在线版
- **内容**: 每天时间轴（09:00-21:00）、景点推荐理由、餐厅精选、交通路线、PlanB
- **交付渠道**: 微信发送链接
- **交付时间**: 待定

## 6. 人工介入条件

- 尊享版: 100% 人工审核
- 标准版: 自动质检 + 40-60% 人工抽检
- 免费版: 纯自动
- 退款/投诉: 人工处理

## 8. 核心用户流程

```
首页(/) → 问卷(/quiz) → 提交成功(/submitted)
→ 扫码加微信 → 确认需求 → 付款
→ AI 生成行程 → 审核
→ 微信交付 /plan/[id] 链接
→ 用户反馈 → 精调（标准版 1 次 / 尊享版 3 次）
```
