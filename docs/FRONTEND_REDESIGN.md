## 二、UI 设计参考

产品是旅行手账本——温暖、有质感、像翻开一本精心制作的书。不是冰冷的 SaaS，也不是廉价的电商。

### 推荐参考网站

| 网站 | 参考什么 | 链接 |
|------|---------|------|
| **Airbnb** | 温暖的色调、大图片、信任感构建、清晰的步骤引导 | airbnb.com |
| **Headspace** | 温和的入门引导、一步一步不急不躁的感觉 | headspace.com |
| **Notion** | 干净的排版、留白、信息层次分明 | notion.so |
| **Apple** | 产品展示的高级感、大面积留白、文案精炼 | apple.com |
| **Craft.do** | 文档/笔记类产品的质感、纸质感设计 | craft.do |
| **MUJI** | 简洁温暖、自然材质感、不花哨但有品味 | muji.com |
| **Kinfolk Magazine** | 杂志排版美学、摄影+留白+字体的平衡 | kinfolk.com |
| **Wanderlog** | 旅行规划产品的交互设计 | wanderlog.com |

### 设计原则

```
1. 温暖 > 酷炫（暖色系为主，不用深黑科技风）
2. 留白 > 堆满（每屏只说一件事）
3. 实物感 > 数字感（让人联想到纸质手账本）
4. 真实图片 > 通用素材（用真实的城市/美食照片）
5. 一句话 > 一段话（文案精炼，不废话）
```

### 色彩建议

```
主色：暖米白 #FBF7F0（纸张感底色）
强调：赤茶色 #C65D3E（温暖有力，用于 CTA）
辅助：墨绿 #2D4A3E（沉稳，用于标题）
文字：深棕 #3D3029（不用纯黑，更柔和）
点缀：樱粉 #E8A0BF / 金色 #D4A855（特殊元素）
```

### 4.3 V3 八屏表单 `/form/[code]`

按 `docs/product/travel_form_multistep_frontend_v3.md` 实现。

关键交互：
- 每屏一个主问题
- 必填（1-4屏）+ 选填（5-8屏）+ 确认（第9屏）
- 顶部进度条
- 可跳过的页面明确标注
- 避雷选项在第5屏
- 提交后30分钟可撤回
- 动态表单，支持草稿保存
