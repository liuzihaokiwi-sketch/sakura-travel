# 4 屏表单

> 迁移自 `content/kansai/form_design.md` 第一部分。

## 屏1:你们什么时候去关西?

- 出发日期
- 返回日期
- (系统自动算出总天数)

## 屏2:这趟旅行

**这趟更像哪种感觉?**

- ○ 经典版 — 把最值得的都安排好,轻松出发不纠结
- ○ 约会感 — 浪漫·夜景·有情调的餐厅·安静的两人时光
- ○ 出片感 — 好看的地方·咖啡甜品·古着·逛逛拍拍
- ○ 亲子感 — 孩子玩得开心最重要·轻松不赶

**几位出行?**

成人 [  ] 位　儿童 [  ] 位　老人 [  ] 位

> 设计说明:风格和人数是两个独立维度。情侣可以选"出片感",朋友也可以选"约会感"。系统拿到风格信号+人数构成,不互相绑死。

## 屏3:这趟想走什么节奏?

- ○ 能走多少走多少
- ○ 平衡一点
- ○ 悠闲度假不想赶

## 屏4:还有什么想告诉我们的?(跳过也完全没关系)

□ 有些酒店或门票已经订好了
　└ 展开:什么时候 + 订了什么 + 备注(可添加多条)

□ 有些地方去过了,或这次不想去
　└ 展开:点选或输入(多选)

□ 还有别的特别要求
　└ 展开:文本框
　　比如:不吃生食 / 有老人同行 / 不想太早起 / 想多留购物时间

> 大部分人直接跳过这页。但如果你已经订了酒店或去过某些地方,告诉我们能让方案更准。
