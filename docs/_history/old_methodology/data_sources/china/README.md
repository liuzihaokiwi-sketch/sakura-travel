## 开城流程

1. 确定第一个国内城市圈（如广府圈 = 广州+佛山+深圳）
2. 调研该圈的本地攻略站和博主
3. 按 `sources/japan/` 的文件结构复制一份
4. 按品类填充数据源（P0/P1/P2 分级 + 三轴分配）
5. 更新 `sources/README.md` 索引
