## 一、欧洲研究的"国情层 4 份"标配

### R1 · 节奏灵魂
在欧洲研究里必须**分区域**,因为欧洲国家内部多元度高:
- **西班牙**: 安达露西亚 / 加泰 / 巴斯克 / 加利西亚 / 马德里-Castilla(5 区)
- **意大利**: 北意 vs 南意(经济+文化+食物完全不同)
- **德国**: 巴伐利亚 vs 普鲁士(啤酒 vs 啤酒 + 天主教 vs 新教)
- **法国**: 巴黎 vs 普罗旺斯 vs 诺曼底 vs 阿尔萨斯(4+ 区)
- **英国**: 英格兰 vs 苏格兰 vs 威尔士 vs 北爱(身份认同)

**写作要求**:R1 必须有 3-5 条"结构性洞察"(打破"这国家是 X 样"的粗略印象)。

### R2 · 顶级操盘手逻辑(欧洲专用)
欧洲有**高端英文旅行顾问市场**,不抄白不抄:
- **Black Tomato**(英国,$10k+/人)
- **Exeter International**(欧洲定制)
- **Butterfield & Robinson**(徒步/骑行)
- **Abercrombie & Kent**(英国老牌)
- **Condé Nast Johansens**(酒店推荐)

**研究动作**:
1. 访问他们的官网,找目标国家的 sample itinerary
2. 看他们"多收的钱买了什么"(通常是:私人导游 / 秘密俱乐部 / 顶级住宿 / 真人大师体验)
3. 判断哪些能抄进手账本(**抄思路,不抄路线**),哪些留给 Tier 3 加钱定制

**西班牙范本**:Paradores 网(国营古迹酒店)是 Black Tomato 的核心武器,中国攻略零覆盖 = 信息差金矿。

### R3 · 中国人痛点(欧洲专用 11 大主题)
欧洲国家 R3 必须覆盖:
1. **签证**(申根 / 非申根,材料 / 出签率 / 多次签策略)
2. **骗局 6 大类**(迷迭香 / 假警察 / 3 人扒手 / 椅背包 / 街头赠品 / DCC 陷阱)
3. **支付**(欧元 / 非欧元 + Visa vs 银联接受度 + 现金偏好国)
4. **退税 VAT**(流程 / 盖章 / 现金 vs 信用卡退)
5. **作息 / 餐食时间**(晚餐时间 + 周日闭店)
6. **厕所**(公厕少 + 收费 + Caballeros/Señoras)
7. **楼层**(1 楼 = 0 楼 欧洲标准)
8. **语言**(基础 5 词 + 区域独立语言)
9. **购物 / 退税限额 / 品牌**
10. **交通**(机场入 / 城际 / 打车软件)
11. **紧急联络**(112 + 大使馆)

**西班牙范本**:`research_iberia/R3_chinese_traveler_pain_points.md` 200+ 条,可直接作为模板。

### R4 · 区域补完(可选)
如果 R1 没覆盖所有独立区域,补一份:
- 西班牙 R4 = Galicia(加利西亚,不自认西班牙人)
- 意大利可能需要 = 西西里 / 撒丁岛(独立身份)
- 法国可能需要 = 科西嘉 / 布列塔尼(凯尔特身份)

## 三、欧洲跨国联游的"组合规律"

### 天然组合(文化 / 交通 / 体验连续)
| 组合 | 天然在哪 | 推荐天数 |
|------|---------|---------|
| 西班牙 + 葡萄牙 | 伊比利亚同文化圈,AVE 连通 | 14-17 天 |
| 意大利 + 希腊 | 地中海古文明双核 | 14 天 |
| 奥地利 + 捷克 + 匈牙利 | 哈布斯堡遗产(日耳曼+斯拉夫) | 10-12 天 |

| 挪威 + 瑞典 + 丹麦 | 北欧维京圈 | 12-14 天 |
| 克罗地亚 + 斯洛文尼亚 + 黑山 | 巴尔干西岸海岸线 | 10 天 |

### 勉强组合(别拼)
| 组合 | 为什么勉强 |
|------|-----------|
| 西班牙 + 法国 | 文化断层太大,混走串味 |
| 意大利 + 瑞士 | 气质反差(热烈 vs 精致),但短途相邻可点缀 |
| 德国 + 英国 | 跨海+跨圈,不值 |
| 希腊 + 北欧 | 地理方向完全相反 |

### 组合策略
- **12 天极限**: 一国深度 或 天然组合 2 国蜻蜓点水
- **14-17 天**: 天然组合 2-3 国
- **20+ 天**: 天然组合 3-4 国深度

## 六、欧洲文案红线(跨国通用)

### 身份红线(最重要,每国必查)
- **西班牙**: 加泰 / 巴斯克 / 加利西亚 不自认 Spanish
- **英国**: 苏格兰不爱被叫 English
- **比利时**: 弗拉芒 vs 瓦隆 语言战争
- **西班牙+法国**: Basque Country(Euskal Herria)横跨两国
- **意大利**: 南意 vs 北意 经济/文化区别
- **德国**: 巴伐利亚人有独立文化(啤酒文化 vs 普鲁士严谨)
- **瑞士**: 四语区(德/法/意/Romansh),不是"瑞士人说瑞士语"
- **爱尔兰**: 不是英国,共和国独立 / 历史敏感

### 语言红线
每国必须给 5-10 个基础词(不是只西班牙语/法语,要覆盖区域语言):
- 加泰语 / 巴斯克语 / 加利西亚语(西班牙)
- 威尔士语 / 盖尔语(英国 / 爱尔兰)
- 巴伐利亚方言(德国)
- 普罗旺斯语 / 布列塔尼语(法国)

### 饮食红线
- 西班牙: Sangría 是游客饮料 / Paella 是午餐
- 意大利: Cappuccino 只在早餐 / 意面不剪断
- 法国: 不问服务员菜好不好吃(自行研究)
- 德国: 啤酒杯碰杯要看对方眼睛(不看 = 失礼)
- 瑞士: Fondue 不掉面包进锅(有趣惩罚但别忘规则)

## 九、典型坑(欧洲专有)

### 坑 1: 把申根签证当"轻松签证"
真相:2026 年 EES 生物识别全面铺开,首次申请必须录指纹;中国护照申根依然"申请国必须是主要目的地"或"首站国"。

### 坑 2: 把欧洲当"天然串联"
真相:文化圈跨度大,西班牙 → 德国不是"便利邻居",是"两个星球"。混走串味。

### 坑 3: 忽略非申根国
真相:英国 / 爱尔兰 / 克罗地亚(2025 加入)/ 塞浦路斯(2026 待定)/ 保加利亚(2025 陆路加入)都要单独处理。

### 坑 4: 只看国家不看大区
真相:北意 vs 南意 / 巴伐利亚 vs 普鲁士 / 苏格兰 vs 英格兰,国内差异往往 > 邻国差异。

### 坑 5: 忽略欧元 vs 非欧元国
真相:英国英镑 / 瑞士法郎 / 北欧三国独立货币 / 捷克匈牙利独立货币 / 保加利亚 2026 入欧元...这些都是手账本必记。

### 坑 6: 小费文化一刀切
真相:法国含服务费 / 意大利 10% / 德国四舍五入 / 北欧不给 / 英国 10-12.5%。

### 坑 7: 建筑只说"教堂"
真相:哥特 vs 罗马 vs 拜占庭 vs 巴洛克 vs 新艺术,每种背后是完全不同的时代精神。
