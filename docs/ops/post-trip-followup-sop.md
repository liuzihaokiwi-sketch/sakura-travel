# 旅行后回访话术模板（T17）

> 版本：v1.0 | 最后更新：2026-03-21
> 目标：旅行结束后 3-7 天，通过企微/短信触达用户，收集真实体验反馈并入库。

---

## 一、触达时机

| 触达节点 | 距出发日期 | 渠道 | 优先级 |
|----------|-----------|------|--------|
| 第一次 | 出发后 +3 天（约到家） | 企微文字 | 高 |
| 第二次 | 出发后 +7 天（若第一次未回复） | 企微/短信 | 中 |
| 收集完成 | 反馈提交后 | 企微致谢 | 低 |

---

## 二、企微回访话术

### 话术 A（首次触达，3 天后）

```
旅行结束顺利了吗？🎌

[姓名/称呼] 好，我是你的日本行程顾问～

你的东京 5 日行程应该接近尾声了，想在你还沉浸在旅途中的时候问几个小问题：

1. 行程整体节奏还好吗？有没有哪天感觉太赶或太松？
2. 推荐的餐厅有没有去成？有什么出乎意料的好吃？
3. 有哪个地方超出预期、或者没想到这么棒？

你的反馈对我们帮助非常大，以后会用来帮助下一个客人安排更好的行程 🙏

（如果方便的话，能否发几张旅途中的照片给我们？欢迎你分享～）
```

### 话术 B（第二次触达，7 天后）

```
Hi [称呼]，上次发的问题不知道你是不是还没来得及回复 😊

回来有段时间了，日本的记忆慢慢沉淀，现在正是最好的时候聊聊感受。

只有这 2 个最重要的问题（真的只要 1-2 分钟）：
① 行程推荐最满意的是哪个地方？
② 如果能改变一件事，你会改什么？

感谢你的反馈，下次来日本欢迎再找我们！🌸
```

### 话术 C（反馈收集完成致谢）

```
谢谢你！你的反馈已经记录 ✅

你提到的「[用户提到的景点/问题]」我们会认真研究，下一版行程里会改进这块。

你的旅程是我们最好的老师 🙏

如果有朋友要去日本，欢迎推荐我们，他可以用你的专属链接享受 ¥50 优惠～
[INVITE_LINK]
```

## 三、问卷链接（H5 表单）

短链格式：`https://trip.ai/feedback/{order_id}`

问卷字段（对应 `user_entity_feedback` 表）：
- 每个已推荐景点：**去了吗？（是/否）**
- 实际去了：**评分（1-5 星）**、**推荐准确度（1-5）**、**排队程度（1-5）**
- 餐厅类：**食物评分（1-5）**
- 拍照类：**拍摄效果（1-5）**
- 开放文字：**最惊喜的地方** / **希望改进的地方** / **标签补充**

---

## 四、验证标记规则（自动入库）

满足以下条件的反馈自动标记 `verified = true`：
1. 评分维度填写 ≥ 3 项
2. `visited = true`（实际到访）
3. 非重复提交（同 order_id + entity_id 只保留最新）

`verified` 反馈会：
- 更新 `entity_time_window_scores.crowd_score`（实际人流数据）
- 触发 `entities.verified_visit_count` +1
- 若与现有数据偏差 ≥ 2 分，触发编辑团队人工审查标记
