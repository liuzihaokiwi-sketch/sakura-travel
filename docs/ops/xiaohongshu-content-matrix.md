# 小红书内容矩阵启动计划（T15）

> 版本：v1.0 | 最后更新：2026-03-21
> 目标：建立每日1篇攻略型笔记发布节奏，CTA 引流至问卷页，目标 CPL ≤ ¥50

---

## 一、账号定位

- **账号名称**：「日本行程实验室」或「7天说走就走」
- **账号人设**：资深日本旅游规划师（曾帮300+旅行者规划，数据说话）
- **内容调性**：干货优先、数据说话、避坑专家、不说废话

## 三、标题公式模板库

### 数字+具体结论型（CTR 最高）
- 「上野公园→浅草寺→晴空塔，步行{N}分钟全搞定」
- 「东京第1天怎么安排：{N}个景点，{N}顿饭，花费明细」
- 「去日本必须知道的{N}件事（第{N}个99%的人不知道）」

### 对比/揭秘型（分享率高）
- 「同样的东京，普通游客vs行程规划后的差距」
- 「道顿堀章鱼烧哪家好吃？我吃了{N}家给你排名」
- 「日本旅游攻略APP的局限性（以及我们怎么解决的）」

### 痛点共鸣型（评论互动多）
- 「看完小红书300篇攻略还是不知道怎么排行程？」
- 「第一次去日本最怕的不是语言，是……」
- 「和行程规划师沟通5分钟，就知道你适合哪种玩法」

## 五、CTA 策略

### 层级设计
1. **低摩擦 CTA**（最优先）：「评论「行程」，我发给你」→ 收集互动
2. **中摩擦 CTA**：「点击主页链接，填3分钟问卷」→ 直接引流
3. **高摩擦 CTA**：「私信我你的出行日期和人数」→ 高意向用户

### UTM 参数规范
- 问卷链接：`https://trip.ai/quiz?utm_source=xhs&utm_campaign={content_type}&utm_content={post_id}`
- 转化追踪：在 conversion admin 看板查看「渠道分析」
