# 全站文案审查报告

> 审查日期：2026-03-21
> 审查规则：
> 1. 不能出现"AI生成"、"自动生成"等字眼 → 应改为"规划师"、"专业团队"
> 2. 不能讲功能 → 要讲"用户痛点 → 你获得的结果"
> 3. 不能有"限制"、"上限"、"不能" → 用正向表达
> 4. CTA 要有行动感，不要"了解更多"这种弱 CTA
