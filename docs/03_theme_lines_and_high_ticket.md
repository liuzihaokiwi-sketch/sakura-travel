# 03 主题线与高客单升级方向

> 版本：v2.0  
> 更新时间：2026-03-21  
> 角色：产品 / 内容 / 增长 / 方案设计  
> 权威性：高

## 总原则
“同一个地区 + 不同主题” = 不同产品。  
真正拉开差异的不是城市，而是**这次想获得什么体验**。

---

## 一、固定主题家族（先只做 6 类）
当前阶段不扩到更多主题，先固定：

1. **首次经典**
2. **情侣审美 / 出片**
3. **美食购物**
4. **温泉疗愈**
5. **文化深度 / 小众体验**
6. **亲子轻松**

### 为什么只做 6 类
- 足够覆盖主要需求
- 便于做 route family
- 便于内容、问卷、交付统一
- 不会让方案树失控

---

## 二、主题层的真正作用
主题层不是“多一个标签”，而是直接影响：
- 候选点位召回
- 餐厅权重
- 酒店区域与类型选择
- 节奏安排
- 亮点解释方式
- 高光预告写法
- 页面与内容话术

---

## 三、两条高价值升级线

### A. 第二次日本
不是“更详细的第一次日本”，而是：
- 降低经典大路货权重
- 增加本地感、小众感、熟人感
- 避免重复体验
- 更强调“你没必要再去那个热门点”的判断

#### 适合人群
- 去过 1–2 次日本
- 已经打过部分经典卡
- 想玩得更特别，但不想踩雷

### B. 一地深玩
不是多城压缩，而是：
- 降低跨城
- 增强区域层次感
- 更重视街区、夜晚、餐厅、步行体验、留白
- 更像“生活在这里几天”，而不是“打卡一串点”

#### 适合人群
- 不想折腾跨城
- 想慢慢玩
- 对审美、餐厅、街区体验更在意
- 重复赴日用户尤其适合

---

## 四、高客单最值得做的方向
高价不是靠“更多页数”，而是靠“更值的体验组合”。

### 当前最值得做的高客单方向
1. 温泉酒店路线
2. 审美 / 出片路线
3. 工艺匠人 / 文化深度路线
4. 纪念日 / 蜜月路线
5. 一地深玩升级版

---

## 五、系统分流信号
主题线不是靠人工猜，系统应使用以下信号做分流：

### 基础信号
- 同行结构：情侣 / 闺蜜 / 带父母 / 带孩子 / 独自
- 日本熟悉度：第一次 / 去过 1–2 次 / 去过很多次
- 玩法偏好：多城顺玩 / 一地深玩 / 还没想好
- 风格偏好：经典 / 出片 / 吃 / 温泉 / 深度 / 亲子
- 预算偏向：住 / 吃 / 玩 / 均衡 / 性价比

---

## 六、内容与销售表达方式
前台不要平铺一堆主题页。  
更合理的是：
- 首页只放轻入口
- 问卷中做主题识别
- 内容和专题页承担“教育 + 分流”
- 后台用主题线驱动方案树

---

## 七、现在不做的事
- 不为每个主题单独起一个前台产品
- 不把主题做成复杂商品矩阵
- 不在首页把所有主题平铺
- 不为小众需求无限扩主题
