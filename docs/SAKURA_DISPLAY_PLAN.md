**别展示"你抓了多少数据"，要展示"我现在该不该去、去哪最稳、为什么信你"。**

## 二、5 个核心指标（面向普通人）

### 1. 能冲指数（0-100）
```
90-100 🟢 现在就冲      — 正在最佳期，确认度极高
70-89  🟡 本周可以冲    — 接近最佳期 or 多源预测一致
50-69  🟠 再等等        — 还没到最佳期
30-49  🔴 还早/已过     — 花苞阶段 or 已经在落
```
映射关系:
- confidence=100 + stage=best_viewing → 能冲指数 95-100
- confidence≥80 + stage=full_bloom → 能冲指数 85-95
- confidence≥80 + stage=starting → 能冲指数 50-65
- confidence≥80 + stage=falling → 能冲指数 40-55

### 2. 最佳观赏窗口
不给单日，给范围: "3/28-4/2"
在时间轴上标注用户的假期日期 → 显示重叠度

## 七、能冲指数计算公式

```python
def calc_rush_score(confidence: int, stage: str, days_to_best: int) -> int:
    """
    能冲指数 = 基础分(来自stage) × 置信度修正 × 时间修正
    """
    # 基础分 (当前状态)
    stage_base = {
        "bud": 20,
        "starting": 45,
        "30_percent": 55,
        "50_percent": 65,
        "70_percent": 75,
        "80_percent": 82,
        "best_viewing": 95,
        "full_bloom": 90,
        "falling_starts": 55,
        "green_leaves": 10,
    }.get(stage, 30)
    
    # 置信度修正 (confidence 100=1.0, 35=0.7)
    conf_factor = 0.7 + 0.3 * (confidence / 100)
    
    # 时间修正 (距最佳期越近越高)
    if days_to_best is None:
        time_factor = 0.8
    elif days_to_best <= 0:  # 已在最佳期内
        time_factor = 1.0
    elif days_to_best <= 3:
        time_factor = 0.95
    elif days_to_best <= 7:
        time_factor = 0.85
    elif days_to_best <= 14:
        time_factor = 0.7
    else:
        time_factor = 0.5
    
    return min(100, int(stage_base * conf_factor * time_factor))
```
