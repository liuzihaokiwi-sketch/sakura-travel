# 05｜北九州城市圈系统结构表

面向：程序 AI / 城市圈建模 / 前端配置层  
目标：把“北九州城市圈”固化成可排程、可编辑、可扩展的数据结构。  
默认版本：`northern_kyushu_circle_v1`

---

## 1. 圈定义

```ts
interface CityCircle {
  circle_id: string;
  name: string;
  region: string;
  core_bases: BaseNode[];
  daytrip_nodes: DaytripNode[];
  major_activity_clusters: MajorActivityCluster[];
  secondary_activity_clusters: SecondaryActivityCluster[];
  default_hotel_strategies: HotelStrategyPreset[];
  recommended_day_range: [number, number];
  expansion_rules: ExpansionRule[];
}
```

### 1.1 建议圈边界

| 字段 | 值 |
|---|---|
| `circle_id` | `northern_kyushu_circle_v1` |
| `name` | 北九州城市圈 |
| `region` | Kyushu |
| `core_bases` | 福冈 / 别府由布院(条件型) / 长崎(扩展型) |
| `daytrip_nodes` | 太宰府 / 柳川 / 别府 / 由布院 / 长崎 / 熊本 |
| `optional_expansion_nodes` | 阿苏 / 佐贺 / 门司港 |
| `recommended_day_range` | `5~9` 天 |
| `hard_cap_without_expansion` | `8` 天 |

### 1.2 默认策略

| 天数 | 默认住法 | 默认核心 |
|---|---|---|
| 4–5 天 | 单基点（福冈）+ 1 个温泉/历史节点 | 福冈 + 太宰府 + 别府/长崎择一 |
| 6–7 天 | 双基点（福冈 + 温泉/长崎） | 福冈 + 别府由布院 + 长崎/熊本择优 |
| 8–9 天 | 双基点 / 三基点条件开放 | 可加入熊本或阿苏扩展 |

## 5. 北九州圈活动兼容矩阵

| 主活动 | 最适配次要活动 | 不建议强拼 |
|---|---|---|
| 福冈都市美食线 | 屋台、车站补餐、轻量购物 | 同日再拼长崎/别府 |
| 太宰府文化线 | 甜点、轻量参道购物 | 晚上再拼远距城市 |
| 别府温泉地狱线 | 温泉街散步、旅馆晚餐 | 当晚回福冈强夜生活 |
| 由布院温泉线 | 咖啡、散步、旅馆休息 | 同日再拼熊本 |
| 长崎港城历史线 | 夜景、港城晚餐 | 清晨再回福冈跑满日 |
| 熊本城线 | 小吃、城下町慢走 | 同日再拼阿苏 |
| 阿苏自然线 | 观景补位、轻食 | 雨天硬跑长户外 |

## 6. 风险标签系统（给活动簇）

### 6.1 字段

```ts
interface ActivityRiskTags {
  precheck_risk: RiskLevel;
  live_disruption_risk: RiskLevel;
  execution_friction: RiskLevel;
  user_condition_risk: RiskLevel;
  known_schedule_dependency: boolean;
  day_of_disruption_possible: boolean;
  backup_same_corridor_required: boolean;
}
```

### 6.2 分类原则

- 温泉酒店售罄、展望设施维护、列车/主题车预约，属于前置可规避风险。  
- 火山地区天气、山地能见度、列车大面积晚点，属于实时扰动风险。  
- `aso_nature_drive / beppu_onsen_hells / yufuin_retro_onsen` 默认需要 fallback。

## 8. 北九州圈默认生成模板

### 4–5 天

- 必选主活动：`福冈都市美食 / 太宰府或柳川`
- 画像命中后再加入：`别府由布院 or 长崎`
- 不默认同时加入：`长崎 + 熊本 + 别府`

### 6–7 天

- 默认 4–5 个主活动簇
- 福冈必须承接 2–3 天
- `别府由布院 / 长崎 / 熊本` 三者择二

### 8–9 天

- 可开放 `福冈 + 温泉 + 长崎` 或 `福冈 + 温泉 + 熊本`
- 阿苏只在明确自然/自驾画像下开放
- 不建议 9 天内把北中南九州拉成一条线

## 10. 来源锚点（供程序维护）

- JR Kyushu 的 Northern Kyushu Rail Pass 覆盖福冈、大分、熊本、长崎、佐贺，适合做“北九州城市圈”边界。
- JNTO 的经典九州线路也以福冈、长崎、熊本、别府/由布院为常见骨架。
