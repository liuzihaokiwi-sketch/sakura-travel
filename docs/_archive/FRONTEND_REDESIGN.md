## 二、UI 设计参考

产品是旅行手账本——温暖、有质感、像翻开一本精心制作的书。不是冰冷的 SaaS，也不是廉价的电商。

### 推荐参考网站（实际看过的）

#### 产品/工具类（功能+质感）

| 网站 | 实际设计特点 | 参考什么 |
|------|------------|---------|
| [Craft.do](https://craft.do) | 暖米色底色、圆角元素、纸质纹理感、充分留白、过渡动画慢而优雅 | **底色和纸质感**——最接近手账本的数字表达 |
| [Headspace](https://headspace.com) | 暖米白 #F9F4F2 底色、自定义圆润字体、装饰性SVG点缀、柔和阴影、友好CTA | **一步步引导的温和感**——表单流程参考 |
| [Wanderlog](https://wanderlog.com) | 黑白为主、清晰CTA双入口、功能截图做hero、大量真实用户评价（70+条） | **社会证明的铺量**——评价展示方式 |

#### 高端旅行定制公司（定位+信任+展示）

| 网站 | 实际设计特点 | 参考什么 |
|------|------------|---------|
| [Boutique Japan](https://boutiquejapan.com) | 全幅鸟居hero图、白底+暗文字极简、105条五星评价显眼展示、行程按天数分卡片展示、多个行业认证logo | **信任感构建**——评价+认证的组合方式 |

| 网站 | 实际设计特点 | 参考什么 |
|------|------------|---------|
| [Bungu Store](https://bungu.store) | 纯白底+深棕字、金色点缀（星标/新品）、网格产品展示、极简导航 | **日式克制美学**——少即是多 |

### 从这些网站提取的设计要点

1. **底色用暖色不用纯白** — Craft 用米色、Headspace 用 #F9F4F2、Jacada 用浅棕交替。纯白太冷。
2. **hero 用真实场景而非素材** — Boutique Japan 用鸟居实拍、Jacada 用舞妓视频、不是 Unsplash 通用图。
3. **信任先行，价格后说** — 高端旅行网站都先展示评价/认证/服务理念，价格藏在深处或"联系获取"。
4. **CTA 不急** — "Discover more"、"Start here" 比"立刻购买"更有品味。我们的 ¥198 价格低到可以直接展示但按钮文案要温和。
5. **一屏一件事** — 每个滚动区间只讲一个卖点，不堆砌。
6. **装饰性元素点缀** — Headspace 的 SVG 星星、Jacada 的金色线条图标——轻点缀而非满屏动画。

### 设计原则

```
1. 温暖 > 酷炫（暖色系为主，不用深黑科技风）
2. 留白 > 堆满（每屏只说一件事）
3. 实物感 > 数字感（让人联想到纸质手账本）
4. 真实图片 > 通用素材（用真实的城市/美食照片）
5. 一句话 > 一段话（文案精炼，不废话）
6. 国内外并重 > 日本独占（品牌定位是"旅行手账"不是"日本攻略"）
```

### 国内/海外分区设计

首页和产品展示需要同时覆盖国内和海外，不能让国内用户觉得"这是个日本旅行网站"。

**方案：首页分两个入口**

```
┌────────────────────────────────────┐
│ 一本替你想好一切的旅行手账          │
│                                    │
│ ┌──────────┐    ┌──────────┐      │
│ │ 国内出发  │    │ 海外出发  │      │
│ │          │    │          │      │
│ │ 广府·广东 │    │ 关西·关东 │      │
│ │ ·北疆    │    │ ·北海道   │      │
│ │          │    │          │      │
│ │ ¥198 起  │    │ ¥228 起  │      │
│ └──────────┘    └──────────┘      │
└────────────────────────────────────┘
```

- 点击"国内出发" → 展示广府/广东/北疆的场景图和攻略样本
- 点击"海外出发" → 展示关西/关东/北海道的场景图和攻略样本
- 下单页根据选的城市圈自动判断国内/海外价格
- 所有通用区块（流程、信任、FAQ）两边共用

**不要做成两个独立站点**，就是首页有两个入口 tab/卡片，后续页面根据城市圈自动适配色调和场景图。

### 色彩建议

```
通用底色：暖米白 #FBF7F0（纸张感）

标题：墨绿 #2D4A3E（沉稳）

海外圈点缀：樱粉 #E8A0BF + 靛蓝 #4A6FA5
```
