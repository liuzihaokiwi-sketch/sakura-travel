## 〇、重构背景与设计参考

### 为什么要重构

当前前端和产品定义严重脱节：

| 维度 | 现在 | 应该 |
|------|------|------|
| 品牌 | "Sakura Rush 2026 · 全日本樱花追踪" | 旅行手账定制品牌 |
| 定价 | 免费/248/888三档 | 198基准+每天20 |
| 流程 | 全部走微信（复制微信号） | 先付款→填表→生成→预览→修改→解锁 |
| 市场 | 日本only | 6个城市圈（含国内） |
| 表单 | 6步老版本 | V3八屏设计 |

### 要删除/重建的页面

| 现有页面 | 处理 |
|---------|------|
| `/quiz` | 删除，入口改为 `/order` |
| `/sample` | 删除，不再有免费样本 |
| `/preview` | 合并到 `/guide/[code]` 的前置层 |
| `/pricing` | 合并到 `/order`（动态计价） |
| `/detail-form` | 重建为 `/form/[code]`（V3八屏） |
| `/plan` | 重建为 `/guide/[code]`（三段结构） |
| `/plan/[id]/upgrade` | 删除 |
| `/rush` | 迁移到 `/tools/sakura` |

### 设计原则

```
1. 温暖 > 酷炫（暖色系为主，不用深黑科技风）
2. 留白 > 堆满（每屏只说一件事）
3. 实物感 > 数字感（让人联想到纸质手账本）
4. 真实图片 > 通用素材（用真实的城市/美食照片）
5. 一句话 > 一段话（文案精炼，不废话）
6. 国内外并重 > 日本独占（品牌定位是"旅行手账"不是"日本攻略"）
```


### 圆角/阴影/间距

```css
--radius-sm: 8px;

--shadow-card: 0 1px 3px rgba(61,48,41,0.08);

```

### 动画

```css
/* 页面进入 */
--enter-duration: 600ms;
--enter-ease: cubic-bezier(0.16, 1, 0.3, 1);  /* 慢出 */

/* hover */

/* 整体原则：慢而优雅，参考 Craft.do 的 transition-all duration-500 */
```

## 六、V3 八屏表单 `/form/[code]`

**目标**：温和引导，不给压力。参考 Headspace onboarding。

### 整体布局

```
┌─────────────────────────────────────────────────────────────┐
│ ← 返回                   ●●●○○○○○○  3/9                     │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│                                                             │
│              [当前屏的主问题区域]                              │
│              （详见下方每屏设计）                              │
│                                                             │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  [上一步]                                    [继续 →]        │
│  （第1屏不显示上一步）          （选填屏显示"跳过这一页"）      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**通用交互**：
- 顶部进度：圆点指示器（实心=已填，空心=未填，当前屏高亮）
- 每屏切换时自动保存（PATCH 到后端）+ localStorage 备份
- 返回可以回到上一屏（已填数据保留）
- 动画：屏切换用左右滑动（200ms ease-out）

### 每屏详细设计

#### 屏1：城市圈（已在 /order 选过，此处确认）

```
先告诉我们，这趟旅行想去哪里

[已选：关西城市圈 ✓]  [想换？点击修改]
```
