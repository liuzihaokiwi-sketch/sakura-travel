## 使用规则

1. **模板`meal_area`** 必须从上表餐厅区域里选，不能自创
2. **酒店数据`hotel_area`** 必须从上表酒店区域里选
3. **景区本地吃**（川床/宿坊/吉野山等）用对应特殊区域
4. **晚餐区域** = 回程路上餐厅最密集节点（不是酒店位置）
