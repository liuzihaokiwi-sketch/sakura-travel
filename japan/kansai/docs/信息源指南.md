# 关西模板信息源使用指南

> 做模板/数据采集时应该参考哪些信息源,各自的用途和可信度。
> 最后更新:2026-04-20

---

## 信息源分级

### P0 权威信息源(事实依据,最高优先级)

**官方网站**
- 寺院/神社官网:开门时间/票价/特别公开日期/禁止事项
- 京都市官方观光 Navi: kyoto.travel
- 大阪观光局: osaka-info.jp
- JR/近铁/京阪官网:交通时刻表/票价

**japan-guide.com**
- 用途:景点基本信息/季节时间/人流评估/实况报告
- 特点:每年红叶/樱花季有详细实况报告,可信度极高
- 参考方式:查景点页面+查当年季节实况博客
- URL格式: japan-guide.com/e/e{编号}.html

**Tabelog (食べログ)**
- 用途:餐厅评分/营业时间/预约方式/价格区间
- 可信度:日本最权威餐厅评分,P0级
- 注意:分数 3.5+ 才算值得考虑,3.0以下慎重

---

### P1 高质量参考源

**InsideKyoto.com (Chris Rowthorn)**
- 用途:京都深度体验/小众景点/旅游建议
- 特点:前Lonely Planet作者,被NYT/泰晤士报引用,本地居住多年
- 适合:验证小众景点/找"游客不知道的角度"

**japan-guide.com 的论坛/问答区**
- 用途:真实旅行者的具体体验/当季情报
- 特点:比小红书理性,来自全球游客

**Kyoto Travel (Kyoto City Tourism)**
- 用途:官方视角的体验推荐
- URL: kyoto.travel/en

**VOYAPON / Tokyo Cheapo / japan-travel.com**
- 用途:英文深度攻略,有时比中文攻略更新更快
- 特点:编辑团队常驻日本,信息可靠

---

### P2 辅助参考源

**小红书 (opencli工具)**
- 用途:了解中国用户的真实诉求/出片需求/体验反馈
- **重要限制**:
  - 极端观点(特别负面/特别正面)容易获高收藏,不代表普遍情况
  - 作为"用户感受"的参考,不作为事实依据
  - 用collects>likes的笔记(真正有用)作为参考
  - **一条笔记的结论不能直接采用**,至少3-5条交叉验证
- 最佳用法:搜索"景点+一日游/路线/city walk"看高收藏的行程结构

**Google Maps评价**
- 用途:景点热度/人流时段/照片参考
- 适合:快速判断景点人气和实际状态

**TripAdvisor**
- 用途:西方游客视角/细节体验描述
- 适合:补充中文信息源盲区

---

### P3 灵感来源(不作为事实依据)

**BRUTUS / &Premium / Discover Japan (日本杂志)**
- 用途:本地编辑视角的策展/深度故事/冷知识
- 适合:找fun_fact/curators_notes的素材

**Lonely Planet / Rough Guide**
- 用途:基础信息参考
- 注意:更新较慢,具体数据以官网为准

## 不同类型信息的最佳信息源

| 需要确认什么 | 首选来源 | 备用来源 |
|---|---|---|
| 开门时间/票价 | 官方网站 | japan-guide |
| 禁止事项(和服/拍照) | 官方网站 | 近年小红书帖子交叉验证 |
| 樱花/红叶实况时间 | japan-guide 季节博客 | 京都市官方 |
| 景点体验质量 | japan-guide + InsideKyoto | 小红书3条以上 |
| 餐厅推荐 | Tabelog | 小红书(高收藏餐厅帖) |
| 人流情况/最佳时段 | japan-guide实况 | Google Maps热力图 |
| 用户真实路线 | 小红书高收藏行程帖 | japan-guide论坛 |
| 本地冷知识/故事 | InsideKyoto | BRUTUS/日本杂志 |
| 交通方式/票价 | 官方交通网站 | japan-guide |

---

## 使用原则

1. **事实类信息(票价/时间/政策)必须查官方** — 小红书/攻略号数据经常过时
2. **体验类信息至少3个来源交叉验证** — 一条小红书帖子不能代表全部
3. **极端评价要用权威来源校验** — 特别负面/正面的帖子可信度低
4. **季节实况查japan-guide当年报告** — 每年气候不同,参考往年数据有偏差
5. **餐厅数据以Tabelog为准** — 不以小红书网红评价为准
6. **小红书的价值在于:用户在乎什么/出片在哪里/踩坑在哪里** — 不是在于"这家好不好"
