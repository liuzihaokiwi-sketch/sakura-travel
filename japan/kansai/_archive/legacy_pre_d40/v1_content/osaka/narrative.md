# 大阪叙事种子

> 本文件为手账本文案 AI 提供创作方向与情绪约束。
> 规则：相邻两天的 day_mood 不得重复；每天的 arc 必须有至少一次情绪转折。

---

## 大阪的城市性格

大阪是热闹的、张扬的、属于普通人的城市。它的标签是"吃倒之城（くいだおれ）"——把钱花在吃上花到倒，这就是大阪人的价值观。

跟京都的反差是理解大阪的钥匙：

| 维度 | 大阪 | 京都 |
|------|------|------|
| 气质 | 外向、直率、大声 | 内敛、含蓄、留白 |
| 感官 | 味觉主导——章鱼烧的酱汁味、串炸的油香 | 视觉主导——庭院的苔藓、障子的光影 |
| 底色 | 烟火气、商人文化、庶民能量 | 寺庙气、贵族文化、文人传统 |
| 叙事方式 | 讲"人"——老板的吆喝、食客的笑声 | 讲"物"——建筑、庭院、器物 |

大阪不需要"千年古都"那一套文化沉淀叙事。它的叙事核心是**人**——商人文化造就的直率与热情，"おもろい（有趣）"高于一切的价值观，以及那种"管你怎么看，我就是这样"的自在。

## 全程情绪弧线

大阪在关西旅程中扮演"能量段"的角色。完整的关西情绪弧线：

```
大阪（前段）──→ 京都（中段）──→ 神户/收尾（后段）
热闹·能量·城市感   安静·深度·文化沉浸   放松·味觉·余韵
外向释放           内向吸收             松弛回味
```

大阪段内部也有自己的弧线：到达时的新鲜感 → 核心日的能量爆发 → USJ 的极致刺激 → 深度日的另一面发现 → 离开时带着满足感切向下一城。

## 每天叙事种子

### arrival_day

- **day_mood**: 初遇与兴奋
- **day_arc**: 疲惫落地 → 街头气味唤醒 → 霓虹灯下第一口大阪味
- **theme_carriers**:
  - 道顿堀边走边吃的烟火气——还没放下行李就被拽进大阪的节奏
  - 第一顿晚餐的仪式感——不管吃什么，"终于到大阪了"这个念头本身就是调味料
- **closing_scene**: 道顿堀运河夜景，格力高跑步人和霓虹灯倒映在水面上，手机拍不出来但眼睛记得住
- **contrast_note**: 到达日刻意留白，不塞行程。用"少"来制造饥饿感，跟明天的满日形成反差

---

### core_day

- **day_mood**: 探索与发现
- **day_arc**: 市场早餐慢热 → 大阪城开阔 → 新世界市井热闹 → 法善寺安静过渡 → 夜晚高潮
- **theme_carriers**:
  - 黑门市场（黒門市場）的烟火气开场——海鲜烤串的烟雾和老板的吆喝声，用五感把人彻底拉进大阪
  - 法善寺横丁（法善寺横丁）从热闹到安静的情绪转折——被苔藓覆盖的不动明王，在闹市中心制造一个安静的气泡
- **closing_scene**: 爵士酒吧的微光 or 裏难波小巷的笑声（按人群切换——情侣选前者，朋友选后者）
- **contrast_note**: 全天最满的一天，是大阪能量的集中释放。前面到达日的留白在这里得到回报

---

### module_usj

- **day_mood**: 疯狂与惊奇
- **day_arc**: 排队紧张 → 项目间的肾上腺素 → 夕阳游行感动 → 回程疲惫但满足
- **theme_carriers**:
  - 哈利波特区（ウィザーディング・ワールド）第一眼的震撼——转过弯看到霍格沃茨城堡的那一刻，大人也会倒吸一口气
  - 闭园前灯光秀/游行的感动——一天的疲惫在这个时刻被感动替代
- **closing_scene**: 回难波路上，疲惫但兴奋地复盘今天玩了什么。脚很酸，但嘴角是翘的
- **contrast_note**: 跟前一天的文化探索完全不同，纯粹的快乐和刺激。这是大阪"おもろい"精神的极致体现

---

### audience_day

按人群类型分叉：

#### couple（情侣）

- **day_mood**: 浪漫与余韵
- **day_arc**: USJ 后的慢起 → 海游馆（海遊館）沉浸 → 日落摩天轮升温 → 梅田夜景高潮 → 余韵不再塞点
- **theme_carriers**:
  - 海游馆巨型水槽前的安静——两个人不说话只看鱼，这种安静比任何对话都亲密
  - 天保山摩天轮（天保山大観覧車）日落时分——透明车厢里的紧张感本身就是记忆
- **closing_scene**: 蓝天大厦空中庭园（梅田スカイビル 空中庭園），不再说话只看夜景。风很大，自然靠近
- **contrast_note**: USJ 日是"一起疯"，这天是"一起静"。情侣行程需要这种节奏切换

#### friends（朋友）

- **day_mood**: 美与发现
- **day_arc**: 洋馆咖啡的松弛开场 → 古着巡礼的兴奋 → 甜品喘息 → 夜景收束
- **theme_carriers**:
  - 中崎町/堀江古着店的寻宝感——每家店都可能藏着"就是它"的那件
  - 一起在咖啡店歇脚翻看战利品的满足感
- **closing_scene**: 梅田蓝天大厦或中之岛灯光，集体合照。照片里每个人手上都提着购物袋
- **contrast_note**: 不追热门景点，追体验和发现。朋友行的核心是"一起逛"而不是"一起看"

#### family（家庭）

- **day_mood**: 欢笑与陪伴
- **day_arc**: 海游馆的专注 → 乐高乐园（レゴランド）的释放 → 摩天轮的安静 → 回酒店路上孩子睡着
- **theme_carriers**:
  - 海游馆里孩子趴在玻璃上看鲸鲨的专注眼神——大人看孩子比看鱼更感动
  - 摩天轮上全家人挤在一起的体温——不用说什么，这就是旅行的意义
- **closing_scene**: 摩天轮上，孩子趴在玻璃上看港口灯光。或者更真实的版本：孩子已经在推车里睡着了
- **contrast_note**: 家庭日的节奏要慢，留够"孩子闹了需要暂停"的 buffer

#### default（通用）

- **day_mood**: 自在与余韵
- **day_arc**: 自由节奏，不追赶。想逛就逛，想坐就坐
- **theme_carriers**:
  - 没有计划本身就是计划
  - 在喜欢的地方多待一会儿的自由
- **closing_scene**: 梅田或难波，自由收尾。可以是一杯酒，可以是便利店门口的台阶
- **contrast_note**: 这天存在的意义是给前几天的密度做减法

---

### deep_dive_day

- **day_mood**: 悠然与发现
- **day_arc**: 不赶时间的探索 → 在一个区域深入停留 → 发现"大阪还有这一面"
- **theme_carriers**:
  - 中崎町（中崎町）老巷里偶遇的小店——这个藏在梅田背后的文艺角落，是大阪不常被看见的那一面
  - 天保山（天保山）的海风——从市中心的密度里抽离出来，看到大阪也有开阔的一面
- **closing_scene**: 中崎町老巷里的咖啡余香 or 天保山的海风吹干汗水
- **contrast_note**: 跟核心日的满和赶形成反差，这天是"呼吸"。证明大阪不只有道顿堀和通天阁

---

### recovery_day

- **day_mood**: 慢与回味
- **day_arc**: 自然醒 → 不急不赶 → 在熟悉的区域闲逛 → 用身体记住大阪的节奏
- **theme_carriers**:
  - 酒店附近那家昨天路过但没进去的店——今天终于有时间了
  - 在便利店买了冰咖啡坐在河边发呆的十五分钟
- **closing_scene**: 难波某个安静角落的午后阳光。不拍照，只晒太阳
- **contrast_note**: USJ 后的身体需要恢复，心理也需要从极致刺激切回日常。这天的"无聊"是刻意设计的

---

### departure_half

- **day_mood**: 告别与期待
- **day_arc**: 松弛醒来 → 最后一次逛街的从容 → 带着满足感切向下一城
- **theme_carriers**:
  - "最后再吃一次"的心态——明知道吃不下了还是想再来一份章鱼烧
  - 拖着行李箱走过昨天走过的路，视角突然不同了
- **closing_scene**: 车窗外大阪城市轮廓渐远，京都/神户在前方等着。不是告别，是"下次见"
- **contrast_note**: 不赶行程，纯粹的收尾留白。大阪段的句号要轻，因为关西的旅程还在继续

## 文案风格指引

### 基本语调

不是旅行百科，是"有个懂大阪的朋友在跟你说话"。这个朋友去过很多次大阪，知道哪里好玩，但不会炫耀，只是自然地分享。

### 示范

| 类型 | 示例 |
|------|------|
| 错误 | 大阪城，1583年丰臣秀吉所建，为日本三名城之一，现为国家特别史迹 |
| 正确 | 你第一眼看到天守阁（天守閣）从樱花里冒出来的时候，会明白为什么大阪人管这儿叫"太阁さんのお城"——不是历史课本里的城堡，是他们家的骄傲 |
| 错误 | 道顿堀是大阪最著名的美食街，拥有众多餐厅和小吃摊 |
| 正确 | 道顿堀（道頓堀）不用找，鼻子会带你去。章鱼烧的酱汁味飘过三条街，跟着味道走就对了 |

### 负面信息处理

让用户有心理准备但不受伤：

| 类型 | 示例 |
|------|------|
| 排队 | 不说"排队很长"，说"这家值得等，带本书或者先去旁边逛一圈回来" |
| 贵 | 不说"价格偏高"，说"花这个钱的意义在于……" |
| 人多 | 不说"游客太多"，说"早上9点前到是本地人的节奏，10点后是游客的节奏" |

### 名称规范

重要地名、店名首次出现时中日双语，格式为"中文名（日文名）"。后续出现只用中文名。

示例：黑门市场（黒門市場）、法善寺横丁（法善寺横丁）、蓝天大厦（梅田スカイビル）
