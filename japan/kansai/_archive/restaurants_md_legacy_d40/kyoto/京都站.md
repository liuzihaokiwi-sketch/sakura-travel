## 区域基因

**中转快食枢纽**。京都旅的**起点 / 终点 / 切换点**都在这里——到达日早晨找地方放行李 + 一顿简饭回血、离开日新干线前最后一杯茶、跨区动线白天从岚山换车奔伏见时的中转午餐。20 家源 JSON 餐厅里 economy 13 家、mid 5 家、showcase 2 家——**基因与东山轴、烏丸御池相反**，怀石/割烹不是这里的主角，拉面·乌冬·喫茶·和菓子和到达日升档和牛/天妇罗才是。

与其他区对比：
- **东山轴**「专程赴约·怀石重镇+老铺白昼」，重仪式感
- **河原町-先斗町**「今晚就来·中端夜食+町家夜景」，重热闹
- **烏丸御池**「白天办事顺便·现代怀石+老铺咖啡」，重效率与精度
- **京都站**「下车先吃·回家前再来一口」，重时效与站距

装配原则：
- **到达日早餐/午餐**：INODA 八条口支店「京の朝食」(1F ASTY·八条口 1 分) / つくもうどん (站构内地下东口·7 点开) / 本家第一旭 (南口 5 分·6 点开·1947 老铺)
- **离开日末餐/买手信**：宝泉 (新干线改札内 2F·130 日元入场券或车票即可) / 中村藤吉 (西改札口前 3F) / 京林泉 (中央改札 1 分·到达 / 离开日升档)
- **跨区中转午餐**：山本まんぼ (京风铁板·京都独有粉もの) / つくも (地下乌冬·7 点起快食)
- **京都站一夜宿的晚餐**：へんこつ (居酒屋·牛尾关东煮) / 一石三鳥 (和牛·新开店)
- **早餐 7 点开门的仅 3 家**：INODA 八条口 / つくも / 第一旭——到达日早航班刚需

**Full 目标**：10 家（京都站基因偏 economy·倒金字塔变形为 showcase 2 + mid 2 + economy 6，覆盖"早到 / 站内 / 夜宿 / 离开"四时段）。

> **基因决定配比偏 economy**：index.md §1.4-1.5 的菱形（showcase 1 / high 3 / mid 3 / economy 2）是**基因中性区**的基准（如东山轴 / 河原町），**不是铁律**。京都站是中转/到达日快食枢纽、用户场景天然集中在经济档与站内快食，所以 economy 占 60%、省略 high 档是有意为之——**基因优先于菱形**。装配 AI 读到此处时默认沿用这个变形，不要质疑「为什么京都站没凑齐 high 3 mid 3」。
