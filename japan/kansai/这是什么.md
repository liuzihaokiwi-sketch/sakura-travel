## 一、关西是什么

**关西 = 京都 + 大阪 + 奈良 + 神户 + 周边**——日本最经典旅行圈·中国游客赴日入门首选。

**核心构成**（按产品权重）：

| 圈内位置 | 代表城市 | 产品权重 |
|---|---|---|
| 主骨架（住宿链） | **京都 + 大阪** | 80% 内容 |
| 日归插件 | 奈良 / 神户 / 姬路 / 宇治 / 彦根 | 5+ 天行程挂 1-2 天 |
| 住宿插件 | 有马 / 城崎 / 高野山 / 吉野山 / 熊野 / 天桥立 | 9+ 天行程加 1-2 晚 |

**特殊地位**：关西机场（KIX）是中国直飞日本的核心门户·南方用户赴日 70%+ 第一次选关西。

**节奏精髓**·"慢-快-慢"：
- **大阪**（前段）热闹/能量/容错率高·适合时差恢复
- **京都**（中段）安静/深度/文化沉浸·体力专注力高峰
- **神户/收尾**（后段）放松/味觉/余韵·体力下降后的温柔结尾

## 二、关西 7 化学反应（跨日组合硬规）

跨日方案必守。装配引擎按这些规则跨城序列：

1. **先京都后大阪** —— 状态好时沉浸京都·末尾切换大阪。例外：特种兵反向（大阪恢复时差）
2. **奈良永远作为「桥」** —— 奈良→大阪 JR 直通 50 分钟·不独立过夜·顺路塞
3. **姬路+神户一天组合** —— 姬路上午、神户下午·铁路顺路
4. **宇治不能和姬路/奈良同一天** —— 宇治要慢·跟大景点组合变走马观花
5. **京都-大阪不需要过渡缓冲** —— 15 分钟新干线·切换快反而产生对比
6. **京都超过 5 天切日归模式** —— 第 4-5 天安排奈良/宇治/嵯峨保持新鲜
7. **神户 2 晚 + 有马温泉** —— 第 1 天港口/牛肉/灘酒造·第 2 天下午有马

## 三、关西餐厅区域基因（采集硬规）

按通用规范 [餐厅规范.md §三.5](../../docs/操作SOP/数据池构建/餐厅规范.md)·关西具体归类：

### 京都

| 区域 | 类型 | 基因偏向 |
|---|---|---|
| 东山轴 | 主场密集 | 怀石重镇·high 偏厚·showcase 集中（菊乃井本店/笹木/未在） |
| 河原町-先斗町 | 主场密集 | 中端主力+夜食+川床·mid 主力·high 也多 |
| 烏丸御池 | 主场密集 | 商务午市+现代咖啡+洋食·high+mid 平衡 |
| 京都站 | 日归动线 | **economy 偏厚约 60%**·中转快食·high 1-2 家足够 |
| 岚山 | 日归动线 | 景区豆腐/会席+川床·high+economy 两极·**no mid** |
| 北区 | 日归动线 | 老铺甜品+乌冬+精进料理+夏川床·跨动线（金阁/北野/下鸭/出町柳/大徳寺/鞍马贵船） |
| 伏见 | 日归动线 | 稻荷寿司+酒蔵·mid 主力·**no showcase** |

### 大阪

| 区域 | 类型 | 基因偏向 |
|---|---|---|
| 梅田 | 主场密集 | 商务+百货美食·mid 偏厚·high 多（怀石小店/老铺洋食） |
| 难波/道顿堀 | 主场密集 | 街食重镇·**economy 偏厚约 60%**·mid 是各种章鱼烧/串炸/拉面名店 |
| 心斋桥 | 主场密集 | 中高端餐饮·high+mid 平衡 |
| 新地（北新地） | 主场密集 | 高端夜食·**showcase 1+/high 主力/mid 0/economy 0** |
| 天王寺-新世界 | 日归动线 | 通天阁周边复古串炸街·economy 主力 |
| 天满 | 日归动线 | 居酒屋天堂·餐厅密度全大阪最高·mid 居酒屋主力 |
| 福岛 | 日归动线 | 夜食酒吧 |
| 池田 | 日归动线 | カップヌードル博物馆周边 |

### 神户/奈良/温泉镇/景点单日

- 神户三宫-旧居留地：和牛仪式感主场·**high 偏厚 35%**
- 神户北野-元町：异人馆动线·咖啡甜品+老铺洋食
- 姬路城周边：日归动线·餐厅薄即可
- 奈良：日归桥定位·正餐 5-8 家够用
- **温泉镇**（有马/城崎/高野山）：旅馆怀石主场·外食池薄（3-5 家正餐 + 少量 stops）
- 高野山：宿坊精进料理为主
- 吉野山：樱花季限定店 + 葛切/柿叶寿司
- 熊野古道：沿途餐厅本来稀少（3-5 家正餐 + 几个茶屋 stops）
- 天桥立-伊根：伊根舟屋海鲜 + 智恩寺周边

## 四、关西酒店区域归类（采集硬规）

按通用规范 [酒店规范.md §三.4](../../docs/操作SOP/数据池构建/酒店规范.md)·关西具体归类：

| 区域类型 | 关西归属 | 偏向 |
|---|---|---|
| **温泉宿坊镇** | 有马 / 城崎 / 高野山 | 几乎全 experience·三星 ≈ 0 正常 |
| **商务城市中心** | 梅田 / 难波 / 心斋桥 / 新地 | city 主力·四星五星偏厚 |
| **古都老镇区** | 京都市内（东山/河原町/北区等） | city + experience 混合·町家民宿独有 |
| **车站周边** | 京都站 / 难波站 / 大阪梅田站 | city 主力·三星四星偏厚·商务客流 |

奢华定制目的地：京都顶级品牌（Aman / Park Hyatt / Bvlgari）—— 仅定制启用·不进自动装配。

## 五、关西装配节奏铁律

详见 [模板写作.md](../../docs/操作SOP/模板写作.md) + 装配引擎实现 `assembly/engine.md`。核心：

- **节奏三类**（adaptive / fixed_early / deep_stay）
- **fixed_early ≤ 1 个/行程**（出片早起·消耗用户体力）
- **deep_stay ≤ 2 个/行程**（temple/nature/温泉等深度停留）
- **showcase 餐厅每城 ≤ 1·全程 ≤ 2**（装配硬顶·池容量可超）

## 六、产品边界（什么不做）

跟 [docs/项目定位.md](../../docs/项目定位.md) 一致·关西特定补充：

- **二刷某地区用户** → 走客服定制
- **奢华游**（酒店 ¥4000+/晚）→ 走定制
- **已订酒店用户** → 酒店反向约束动线·主产品装不了
- **预约失败场景**（任天堂没抽中等）→ 客服处理
