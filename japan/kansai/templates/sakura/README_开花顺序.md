# 关西樱花季模板·按开花顺序排布

> 关西樱花期约 3/15 - 4/20,前后 5 周。不同品种/海拔/城市错峰开放,用户来的时间窗口不同,模板按花期选择。
> 最后更新:2026-04-21

## 开花顺序总览(2026 年预测,以当年气象厅/日本気象協会为准)

| 窗口 | 起止 | 主力品种 | 可用模板 |
|---|---|---|---|
| **早樱期** | 3/15-3/25 | 河津桜·魁桜·彼岸桜 | 淀水路(早春)/平野神社魁桜 |
| **染井吉野初开** | 3/25-3/30 | 染井吉野 开 20-60% | 醍醐寺/哲学之道(开头) |
| **染井吉野满开** | 3/28-4/5 | 染井吉野 满开(关西主流) | 绝大多数京都/大阪模板 |
| **染井吉野散花 + 吉野山中千本** | 4/3-4/10 | 散花+花筏 / 吉野中千本 | 哲学之道散花/吉野山/姬路城 |
| **八重樱期** | 4/10-4/20 | 八重樱·里樱 | 造币局通り抜け/平野神社八重/吉野奥千本 |

## 每个模板必带的 `bloom_window` 字段(执行清单)

计划给每个樱花模板的 `assembly_hints` 补充:
```json
"bloom_window": {
  "primary": "2026-03-28 ~ 2026-04-05",  // 染井吉野满开
  "secondary": "2026-04-06 ~ 2026-04-10", // 散花花筏
  "reliability": "高/中/低",              // 满开可靠度(年度波动)
  "note": "2026 年预测 3/28 开花 / 4/3 满开 / 4/10 散花,当年以气象厅开花预报为准"
}
```

## 用户视角·按来的时间选模板

- **3/20 前到**:早樱窗口(淀水路+魁樱)+ 早春梅模板过渡(北野/城南宫)
- **3/25-4/5**:染井吉野满开,主力 13 个模板任选
- **4/5-4/12**:花筏+吉野山+姬路(满开稍晚)
- **4/10-4/20**:造币局+平野八重樱+吉野奥千本(其他已谢)
- **4/20 后**:樱花基本结束,切换到紫藤花(素材库已记)+新绿季模板

---

## 维护备忘

- 每年 3 月下日本気象協会发布开花预报,更新 `primary` 具体日期
- 吉野山 4 段(下/中/上/奥千本)错开 2 周,`nar_yoshino_sakura_full` 要细分
- 关西樱花年度波动 ±7 天,可靠度标注用于客服提醒"可能早/晚"
