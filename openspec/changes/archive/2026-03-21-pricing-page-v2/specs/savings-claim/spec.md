# Spec: savings-claim

## 概述

定义深度版「深度比价系统」差异化价值主张的展示规范。这是深度版相对标准版最核心的溢价理由，必须让用户在价格卡内一眼看到、秒懂价值。

## 核心主张

深度版包含「深度比价系统」，通过对日本主流餐厅、酒店、景点门票、交通方案的横向对比，为你在相同体验质量下找到性价比最优选项，**帮助节省旅行总开销的 20-30%**。

## 具体化金额说明

用场景化数字让抽象百分比变具体：

> 7 天日本行，两人花销通常在 ¥12,000-¥20,000。节省 20-30% 意味着省下 **¥2,400-¥6,000**——深度版的 ¥888 费用，可以在这笔节省里完全覆盖。

**文案要点：**
- 用"两人"而非"人均"——让读者代入自己的实际旅行场景
- 用具体金额区间，不只写百分比
- 最后一句把"¥888 的成本"和"节省金额"对比，让用户感知"这笔钱花得回来"

## 覆盖范围说明（用于 FAQ/详细说明）

比价系统覆盖四个维度：

| 维度 | 说明 |
|------|------|
| 餐厅选择 | 同品质下，避开游客价区域，推荐本地人价位 |
| 酒店推荐 | 同地段下，性价比最高的住宿方案 |
| 景点门票 | 标注哪些门票可提前网购打折（通常便宜 15-30%） |
| 交通方案 | 根据行程匹配最优交通卡，避免重复购票 |
