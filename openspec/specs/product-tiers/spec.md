## 三档产品定义

### 档位 1：免费预览（Free）
| 维度 | 规格 |
|---|---|
| 价格 | **¥0** |
| 定位 | 秀肌肉、建立信任、激发好奇心、驱动付费转化 |
| 预览深度 | **Day 1 完整内容**（含景点/餐厅/交通/避坑）+ **Day 2-7 仅展示标题+锁定预告** |
| 修改次数 | **0 次**（只读，无精调入口） |
| 个性化程度 | 基于轻问卷（4题）即时生成 |
| 审核 | 自动质检（11条 QTY 规则门控）|
| 转化 CTA | 每个锁定 Day 后面插入付费解锁按钮 |

> 预览逻辑：Day 1 完整展示是最大化用户好奇心+信任的临界点。完整展示 Day 1 让用户"先爽到"，Day 2-7 锁定让用户"意犹未尽"→ 付费解锁。

### 档位 2：行程优化版（Standard / ¥248）
| 维度 | 规格 |
|---|---|
| 价格 | **¥248**（首发限时价，原价¥298） |
| 定位 | 主打产品，覆盖 90% 用户需求 |
| 内容深度 | 全程每日行程（7天）·餐厅精选含预约指引·酒店区域建议·交通最优方案·避坑指南·出片攻略 |
| 修改次数 | **2 次行程精调** |
| 个性化程度 | 基于完整问卷（8题）全个性化生成 |
| 审核 | 自动质检 + 40-60% 人工抽检 |
| 适合人群 | 第一次去日本的旅行者；闺蜜团；情侣 |

---

### 档位 3：深度管家版（Premium / ¥888）
| 维度 | 规格 |
|---|---|
| 价格 | **¥888** |
| 定位 | 锚点产品，衬托¥248性价比；覆盖高需求用户 |

| 个性化程度 | 表单 + 1对1对话细化 |
| 审核 | **100% 人工过稿** |
| 适合人群 | 蜜月/纪念日；家庭出游（含老人/小孩）；高端旅行者 |

## 数据结构

### product_sku 核心字段
| 字段 | 类型 | 说明 |
|---|---|---|
| sku_id | UUID | 主键 |
| tier | VARCHAR | **free / standard / premium**（三档固定值） |
| price_cny | DECIMAL | 0 / 248 / 888 |
| mod_count | INT | **0 / 2 / 5**（B2 裁决固定值） |
| preview_days | INT | **1 / 7 / 7**（免费仅Day1完整，其余7天全开） |
| review_sample_rate | DECIMAL | 0.0 / 0.5 / 1.0 |
| is_active | BOOLEAN | 是否上架 |

### orders（修改次数追踪）
| 字段 | 类型 | 说明 |
|---|---|---|
| order_id | UUID | 主键 |
| sku_id | UUID | 关联 SKU |
| mod_used | INT | 已使用精调次数（初始值 0） |
| mod_quota | INT | 总精调配额（从 sku.mod_count 复制） |
| payment_status | VARCHAR | pending / paid / refunded |
| delivery_status | VARCHAR | pending / generating / delivered / failed |

> `mod_used >= mod_quota` 时禁止创建新修改申请（硬规则 BIZ-07）
