## MODIFIED Requirements

### Requirement: 首页 / 专题页场景化入口文案
首页 hero 区或专题页（/rush）入口卡片 SHALL 呈现至少两个场景化短文案，覆盖「首次去」和「熟客 / 深玩」两个用户心智，引导用户主动点击进入问卷。

#### Scenario: 首次去日本场景文案展示
- **WHEN** 用户访问首页或 /rush 专题页
- **THEN** 页面包含类似「第一次去日本？我们帮你排好，不用查攻略」的短句，正文或副标题区可见

#### Scenario: 熟客 / 深玩场景文案展示
- **WHEN** 用户访问首页或 /rush 专题页
- **THEN** 页面包含类似「去过很多次了？这次换个玩法——本地路线、少人排队、你没去过的区域」的短句

#### Scenario: 文案不新增独立模块
- **WHEN** 场景化文案上线
- **THEN** 文案融入现有 hero / 副标题 / 入口卡片区域，不新增独立 section 或 CTA 模块
