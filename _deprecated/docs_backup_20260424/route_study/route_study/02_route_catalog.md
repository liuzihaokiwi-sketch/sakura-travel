# 关西有意义的路线清单 · 第一版

> 主键是**路线**，不是档期也不是天数。每条路线独立成条，附自己的属性（适用窗口 / 天数 / 硬禁区 / 来源依据）。
> 本文件**不判断路线之间是否平权**——平权与否要用户画像进来后匹配才知道。
> 本文件**不追求一上来覆盖全**，先写 10–15 条主干，后续增删合并。

---

## 关于路线编号

- `K` = 关西
- `01-19`：常规主干路线（全年或淡季可跑）
- `20-39`：季节锚定路线（樱花/红叶/祭典/松叶蟹等）
- `40-59`：主题锚定路线（温泉深度 / 朝圣 / 家庭 / 蜜月）

## 一、常规主干路线

### K01 · 双城经典（大阪 + 京都 + 奈良日归）

**存在理由**：关西首次来访者的"安全默认路线"。双城分住 + 奈良日归是所有中英日语攻略站的最大公约数，不是最惊艳的但不会踩雷。

**最小骨架**：
- 先京都后大阪（KIX→京都 HARUKA 75min 直达，最后大阪方便返程）
- 京都 3 天（东山 / 岚山 / 金阁伏见）
- 奈良日归（从京都出发 1h）
- 大阪 2–3 天（城市 + USJ or 购物）

**适用**：
- **天数**：5–7 天（5 天压缩奈良为半日，7 天最从容）
- **档期**：全年可跑；但樱花 / 红叶窗口会被 K20 / K30 系列"挤下冠军"
- **vibe**：classic / family_fun（romantic 和 photogenic 有更好的选项）

**硬禁区**：
- 樱花满开期（3/28–4/8）清水寺日间必须改早或改夜，否则这条路线体验崩坏
- 盛夏 7/14–7/25 如遇祇园祭，应转 K24

**来源依据**：japan-guide 5d/7d 官方、Funliday、乐吃购 7 天系列、Inside Kyoto 7d、携程攻略前 10 条、Klook 繁中

---

### K02 · 双城深度淡季版（只做大阪 + 京都）

**存在理由**：首次来但时间紧张、或**二回目**游客想把一个城市做深的人。淡季不追季节，把京都/大阪单独做到"每个区域都去过"的密度。

**最小骨架**：
- 京都 3–4 天（南东山 / 岚山 / 金阁+北山 / 伏见+宇治半日 / 大原或鞍马1日）
- 大阪 2–3 天（市井核心 / 城市文化 / 海游馆 或 USJ）
- 不加奈良、不加神户

**适用**：
- **天数**：5–7 天
- **档期**：淡季最佳（4/16–5/25 / 8/26–11/5 的非红叶期 / 3/1–3/24）；花季/红叶季也能跑但会浪费档期
- **vibe**：classic / photogenic / romantic

**硬禁区**：
- 首次来访的主流用户可能觉得"为什么不去奈良"——建议在前端默认**不推荐给首次来访用户**
- 盛夏无纳凉床餐体验会明显打折

**来源依据**：知乎多条"一年去 4 次京都" / Inside Kyoto 京都内部分区指南 / Klook 京都深度 / "10 天只做双城"游记

**不确定点**（需要你拍板）：这条和 K01 的边界是"加不加奈良"。奈良日归其实很近（1h），是否值得单独把"不加奈良"做成独立路线？**倾向是：值得，因为二回目用户主动避奈良**——但请确认。

### K03 · 双城 + 神户一日

**存在理由**：对港口 / 夜景 / 洋馆 / 神户牛有兴趣的人，给 K01 增加一个神户日。和 K01 的差异是"多了一个城市气质"而不是"多玩一天"。

**最小骨架**：
- 京都 3 + 大阪 2.5 + **神户 0.5–1**
- 神户半日版：午后抵达 → 元町/北野异人馆 → 港口夜景 + 神户牛晚餐
- 神户全日版：加三宫购物 / 六甲山展望

**适用**：
- **天数**：6–7 天
- **档期**：全年
- **vibe**：classic / romantic（夜景 + 神户牛对情侣友好）

**硬禁区**：
- 和 K40（有马温泉）冲突：同一趟只选一个（神户一日 vs 有马温泉 1N2D）
- 5 天以下塞不进神户

**来源依据**：KKDAY / 乐吃购"奈良 vs 神户"系列、Skyscanner HK 7 日懒人包

---

### K04 · 双城 + 姬路半日

**存在理由**：对日本第一名城 / 建筑 / 历史感兴趣的人。姬路城世界遗产地位极高，且恰在大阪↔冈山路线上，一日往返效率最高。

**最小骨架**：
- 京都 3 + 大阪 2 + **姬路半日** + 其他补充
- 姬路半日：早晨出发 → 姬路城天守 + 好古园 → 下午返大阪
- 常见变体：姬路上午 + 神户下午（K04.5）

**适用**：
- **天数**：7 天起（6 天以下会挤压大阪/京都）
- **档期**：全年，春樱/秋枫加分（姬路城三の丸、西の丸是樱花/红叶名所）
- **vibe**：classic（历史控加分）

**硬禁区**：
- 非城堡/历史爱好者可能觉得无聊，不要默推
- 首次来且 6 天以下，姬路挤掉奈良不值得

**来源依据**：japan-guide 7d 官方（**姬路是官方标配**）、乐吃购姬路全攻略、MATCHA 姬路城樱花攻略

**不确定点**：`himeji_kobe_day` 这个合并模板是否值得拆成两条独立路线（纯姬路 / 姬路+神户）？**倾向：保留合并作为 K04 主变体**，单独的姬路日只在 9+ 天时出现。

## 二、季节锚定路线

### K20 · 樱花追花双城（标准版）

**存在理由**：樱花窗口下的"时间特权"路线。关西樱花时序有 7–10 天递进，这条路线利用递进延长赏樱窗口，而不是被一城的花期卡住。

**最小骨架**：
- 京都 3（早樱 / 高峰 / 岚山晚樱）→ 奈良 1 → 大阪 2 或吉野山 1–2
- 核心赏樱点：醍醐寺（3/底）→ 哲学之道 + 圆山 + 清水夜樱（3/底–4/初）→ 岚山渡月桥（4/初）→ 背割堤（4/初）→ 奈良公园（4/初）
- 大阪收尾可选大阪城西之丸 + 造币局通抜

**适用**：
- **天数**：6–9 天（5 天太赶、10+ 天接 K21 吉野专项）
- **档期**：**3/25–4/10 硬窗口**（每年 JMA 预测修正 ±5 天）
- **vibe**：classic / photogenic / romantic

**硬禁区**：
- 窗口外这条路线无意义，直接退化为 K01
- 清水寺日间人海，早 6 点进场或转夜间特别拝観
- 体力差的用户注意：满开期很多景点需要排队 30–60min

**来源依据**：JNTO 樱花路线官方、6 人游花见关西 6 日、乐吃购京都赏樱 35 选、知乎赏樱冷门景点

---

### K21 · 吉野山追樱专项

**存在理由**：吉野山的 3 万株山樱 + 海拔递进（下→中→上→奥千本，花期 4/2–4/15 递进）是关西其他景点无法替代的。是"追晚樱"的核心节点。

**最小骨架**：
- 大阪/京都基地 → 吉野山 1 晚或 2 晚
- Day1：下午抵达吉野，下千本 + 中千本观景
- Day2：早晨上千本 + 奥千本 + 水分神社
- 如 2 晚则增加吉水神社 / 金峯山寺深度

**适用**：
- **天数**：加在 K20 之上，需要 +1–2 天；所以最少 8 天
- **档期**：**4/2–4/15 硬窗口**（奥千本可延到 4/20）
- **vibe**：photogenic / romantic / 闺蜜 / 樱花迷（不适合亲子）

**硬禁区**：
- 非樱花季吉野山毫无意义（JNTO 自己都说）
- 带老人 / 小孩不建议（有山路）
- 京都樱花没赏够就去吉野是浪费，先确保京都樱花已够

**来源依据**：吉野山樱花懒人包（乐吃购）、JNTO 吉野路线、KKDAY 吉野樱

---

### K22 · 红叶夜枫双城（标准版）

**存在理由**：红叶窗口下的"日+夜分层"路线。京都 13–15 座寺院的夜间特别拝観是关西独有体验，日间游山水 / 夜间游灯火。

**最小骨架**：
- 京都 3–4（东福寺日间 + 永观堂夜枫 / 高雄三寺 / 岚山红叶 + 大原或琉璃光院）
- 大阪 2（箕面红叶 + 市区）
- 奈良可选（谈山神社、奈良公园红叶）

**适用**：
- **天数**：6–8 天
- **档期**：**11/15–12/5 硬窗口**（早红叶贵船 11/初、晚红叶岚山 12/初）
- **vibe**：classic / romantic / photogenic

**硬禁区**：
- 11/18–11/26 人潮爆满，建议早 7 点前到热门寺
- 不在窗口就退化为 K01
- 亲子不推荐夜枫（孩子 20 点前犯困，但夜间拝観 17–21 点）

**来源依据**：乐吃购京都枫叶 26 选、知乎红叶前线私藏方案、携程关西晚秋红叶游记

---

### K23 · 红叶错峰冷门版

**存在理由**：给二回目 / 不愿挤人潮的人。避开永观堂/东福寺，走圆光寺 / 曼殊院 / 光悦寺 / 毘沙门堂等冷门赏枫点。

**最小骨架**：
- 京都 3–4：修学院 / 一乘寺区域 / 山科毘沙门堂 / 洛北鞍马贵船
- 搭配高雄三寺深度日
- 不走永观堂/东福寺

**适用**：
- **天数**：7–9 天
- **档期**：同 K22（11/15–12/5）
- **vibe**：classic / photogenic（二回目加分）

**硬禁区**：
- 首次来访者不推荐（永观堂不去会遗憾）
- 需要体力（冷门寺地理分散）

**来源依据**：知乎"京都红叶冷门私藏" / 2022 私藏赏秋去处

---

### K24 · 祇园祭锚定路线（7/14–7/17 前祭）

**存在理由**：祇园祭是京都 1000 年的夏季祭典，7 月整月氛围营造但**核心在 7/14–7/17**（宵山三夜 + 17 日山鉾巡行）。这是**日期锚定**而非季节锚定的路线。

**最小骨架**：
- 京都 5 晚（7/14–7/17 必须在京都）
- 7/14-16 晚上宵山（四条烏丸一带山鉾夜间巡观 + 屋台）
- 7/17 山鉾巡行（需提前 5–6h 占位或付费观览席）
- 白天穿插常规京都景点，晚上集中在祭典区
- 大阪 2 天可选在祭典前后

**适用**：
- **天数**：7–10 天
- **档期**：**7/14–7/17 硬窗口**（24 日后祭是子选项）
- **vibe**：classic / photogenic（对文化感兴趣者加分）

**硬禁区**：
- 窗口外没意义
- 40 度高温 + 人挤人，不适合老人 / 小孩
- 宵山 3 晚酒店难订，3 个月前必订

**来源依据**：祇园祭维基 / 京都官方祭典页 / 多条小红书祭典游记

---

### K25 · 天神祭 + 花火盛夏（7/25 前后）

**存在理由**：大阪 7/25 天神祭（日本三大祭之一）+ 船渡御 + 花火，是**大阪独有**的夏季锚点。比祇园祭更"大阪气质"。

**最小骨架**：
- 大阪 3 晚（7/24–7/26 必须在大阪，看陆渡御 + 船渡御 + 天满桥花火）
- 京都 2–3（避开 7/25 那天，其他天常规）
- 可衔接淀川花火（8 月第 1 个周六）做 "双花火" 路线

**适用**：
- **天数**：6–8 天
- **档期**：**7/25 前后 3 天**（若做双花火：7/25 + 8/第一周六，天数需 10+）
- **vibe**：family_fun / friends / photogenic

**硬禁区**：
- 窗口外无意义
- 带老人不推荐（高温 + 人挤人）

**来源依据**：天神祭维基 / Osaka Metro NiNE 官方、多条大阪夏祭游记

---

### K26 · 城崎松叶蟹冬季专项

**存在理由**：**城崎温泉的独立主角窗口只有 11/中–3/初**（松叶蟹季）。这个档期城崎从"可选温泉"升级为**独立路线锚点**，甚至可以不去京都也值得。

**最小骨架**：
- 京都 2–3（轻度京都）+ 城崎 2 晚 + 大阪 2 收尾
- 城崎 2 晚核心：7 外汤巡 / 浴衣夜步 / 松叶蟹怀石（2 晚一顿蟹一顿但马牛）
- 交通：JR 特急 kinosaki 直达 2.5h

**适用**：
- **天数**：7–10 天
- **档期**：**11/15–3/5 硬窗口**（松叶蟹蟹季）
- **vibe**：romantic / 闺蜜 / 美食家

**硬禁区**：
- 窗口外城崎大幅降级（可泡汤但没"独立主角"意义）
- 盛夏去城崎几乎无价值
- 带老人看体力（外汤巡要走路）

**来源依据**：visitkinosaki 官方、乐吃购城崎完整攻略、多条冬季蟹季游记

---

### K27 · 祇园祭后祭 + 五山送火（7/24 + 8/16）

**存在理由**：给能在关西停留 25+ 天的长线旅客，或复购游客。7/24 后祭山鉾巡行 + 8/16 大文字五山送火是两个**7–8 月绝对锚点**。

**最小骨架**：难度高，适合特殊项目型用户。

**适用**：
- **天数**：11+ 天
- **档期**：7/24 + 8/16

**状态**：**候选，优先级低**。产品初期不做。

## 三、主题锚定路线

### K40 · 有马温泉情侣蜜月加成

**存在理由**：有马温泉是**近便型**温泉（京都/大阪 1h 内）。对情侣蜜月来说，在一个普通关西行程里塞 2D1N 有马就完成了"温泉升级"，不需要去城崎的远距离代价。

**最小骨架**：
- 在 K01 基础上用 **2 晚**换掉大阪或京都的 2 晚：
  - 京都 2.5 + 有马 2D1N + 大阪 2.5 + 奈良日归
- 有马核心：金泉/银泉 + 一泊二食旅馆 + 温泉街夜步 + 河童拍照点

**适用**：
- **天数**：7–9 天
- **档期**：全年（冬季金泉最佳，夏季可结合神户六甲山夜景）
- **vibe**：romantic（蜜月）

**硬禁区**：
- 亲子也可，但旅馆选择受限（有马多成人向旅馆）
- 6 天以下塞不进

**来源依据**：有马温泉一日游乐吃购、MATCHA 有马旅馆 10 选、Activity Japan

---

### K41 · 高野山朝圣宿坊

**存在理由**：高野山的**宿坊 + 精进料理 + 早课 + 奥之院**是日本其他地方无法替代的体验。日本 1200 年佛教中心 + 世界遗产。

**最小骨架**：
- 大阪/京都基地 → 高野山 **1N2D（标配）** 或 2N3D（升级）
- Day1：下午抵达 → 宿坊 check-in → 壇上伽藍 → 护摩祈祷 → 精进料理
- Day2：早课勤行 → 奥之院弘法大师御廟 → 金刚峯寺 → 返程
- 升级版增加抄经 + 缆车俯瞰 + 南院深度

**适用**：
- **天数**：9+ 天（宿坊 1 晚需要匹配前后京都/大阪基础行程）
- **档期**：避开梅雨（6 月）；冬季有雪景加分；樱花 / 红叶一般
- **vibe**：classic / photogenic / 深度文化（不适合 family_fun）

**硬禁区**：
- 纯享乐型用户会觉得闷（早 5:30 早课、精进料理清淡）
- 幼童和老人看体力（海拔 900m、宿坊无酒店级服务）

**来源依据**：japan-guide 10d/12d 官方（高野山是官方标配）、Visit Wakayama、ANA 官方

---

### K42 · 吉野山 + 高野山精神线

**存在理由**：两座山都是世界遗产（吉野山 + 熊野古道 + 高野山构成"紀伊山地の霊場と参詣道"）。对精神/宗教/文化深度用户，这是**最高级别的关西深度**。

**最小骨架**：
- 京都 2–3 + 吉野山 1 + 高野山 1N2D + 大阪 1–2
- 可衔接熊野古道 1 日（发心門王子 + 本宫大社）

**适用**：
- **天数**：10–13 天
- **档期**：春季（樱花 + 新绿）或秋季（红叶）
- **vibe**：classic / 深度文化

**硬禁区**：
- 带家庭不推荐
- 9 天以下塞不进

**来源依据**：Visit Wakayama 终极行程、Intrepid Travel Koya-Kumano、游侠客熊野古道攻略

---

### K43 · 白滨海边温泉（家庭 / 情侣）

**存在理由**：关西**唯一海边温泉**，白良滨白沙滩 + 面海温泉 + 当地海鲜。和内陆温泉（有马/城崎）完全不同的度假感。

**最小骨架**：
- 大阪/京都基础 2–3 天 + 白滨 2–3 天
- 白滨核心：白良滨沙滩（夏）/ 圆月島 + 千畳敷 / 一泊二食（蟹季 + 鮪料理）
- 交通：特急くろしお 大阪→白滨 2.5h

**适用**：
- **天数**：8–11 天
- **档期**：**5–9 月**最佳（沙滩 + 海浪）；冬季蟹季也可但不是主场
- **vibe**：family_fun / romantic

**硬禁区**：
- 冬季去白滨体验打折（没海滩）
- 7 天以下塞不进白滨（往返成本高）

## 五、沉淀中产生的疑问（需要确认）

我写的过程中有几个判断点拿不准，想和你确认：

### 疑问 1：K02（双城深度）要不要？
它和 K01 的区别是"不加奈良"。严格来说只是 K01 的减法变体。但搜索里"10 天只做双城"的深度游记非常独立，我倾向保留为独立路线。**你判断呢？**

### 疑问 2：樱花季的 K20 和经典版的 K01 关系？
樱花窗口下，K01 被挤下冠军，K20 接管。这意味着"同一条经典路线在不同档期可能降级"。这个"档期激活/降级"机制要不要显式写进路线属性里？

### 疑问 3：K24 祇园祭是一条独立路线还是 K01 的档期变体？
我目前写成独立路线（K24），因为 7/14–7/17 的行程结构确实和 K01 差别很大（晚间都在四条烏丸）。**但这可能让路线数膨胀**——每个祭典都变独立路线的话，清单会从 15 涨到 25+。你怎么判断？

### 疑问 4：亲子路线 K44 够不够？
我只写了一条标配。但带娃其实分"带 2 岁以下 / 带幼儿园 / 带小学生 / 带青春期"——体验差别很大。现在写成一条可能太粗，但拆成 4 条太细。**倾向：先保留 K44 一条，用户量起来后再拆。**

### 疑问 5：K40 有马 和 K03 神户 是不是应该合并成一条"神户/有马二选一"的路线？
它们共用基础骨架，唯一区别是最后一晚去哪。合并能减路线数，但也可能丢失各自的独立性。**倾向：保留两条**，因为"去神户"和"去有马 2D1N"的用户心态完全不同。
