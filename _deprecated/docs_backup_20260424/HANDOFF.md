## 小雅当前任务（2026-04-08）

### 任务：收集手账本页面设计参考图

**背景：** 我们的旅行手账本需要设计 PDF 页面模板，AI 会根据参考图的风格来生成 HTML/CSS 模板。你来帮忙收集参考图。

**需要收集的页面类型（共20种）：**

#### 一、封面类
- **封面页** — 大标题 + 主图 + 旅行关键词

#### 二、导航/总览类
- **目录页** — 结构清单 + 页码
- **大路线总览页** — 路线图 / 阶段带状图
- **主要活动总表页** — 活动卡片网格
- **酒店策略总览页** — 酒店基点时间轴

#### 三、定制感页
- **偏好兑现页** — "你的偏好 → 我们的安排"对照卡

#### 四、每日执行类（最高频翻阅，重点找）
- **每日执行页** — 时间轴 + 今日情绪主题 + 活动卡片

#### 五、详情页
- **主要活动详情页** — 一个活动深度展开
- **酒店详情页** — 为什么选这家 + 住法细节
- **餐厅详情页** — 为什么推 + 怎么点 + 实用提醒（重点找）
- **拍摄主题页** — 打卡点 + 最佳时段 + 构图建议

#### 六、美食/购物类
- **城市美食图鉴页** — 当地特色菜系介绍（重点找）
- **店铺推荐页** — 药妆 / 伴手礼 / 古着分区推荐

#### 七、实用工具类
- **关键预订窗口页** — 必须订 / 建议订 + 截止时间
- **出发前指引页** — Checklist + App + 行李
- **交通页** — 城市间移动 + JR Pass 说明
- **天气穿衣页** — 按月份 / 季节建议
- **实用信息页** — 紧急联系 + 退税 + Wi-Fi
- **预算追踪页** — 空白表格供用户手填

#### 八、补充/应急类
- **顺路补充页** — 次要景点 / 小店合并
- **Plan B 调整页** — 雨天 / 临时变化应对

#### 九、章节过渡类
- **章节开场页** — 城市圈切换时的氛围页
