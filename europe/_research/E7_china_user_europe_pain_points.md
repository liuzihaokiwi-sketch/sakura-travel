> **核心命题**:中国用户去欧洲的真正痛点,不是"看什么景点",而是**与欧洲运行规则的摩擦**。这份文档专注**跨国通用痛点**,不替代单国 R3。每一条都要求:**中国用户普遍会踩 + 常识攻略不讲清 + 有实操建议**。常识级(不要喝自来水、插头型号)不收录。

## 一、签证与入境(EES/ETIAS 新政)

### 1.1 EES 已全面生效(2026/4/10 起)

**硬事实**(Euronews/ETIAS 官方 2026/3-4):
- 2025/10/12 开始 29 个申根国分阶段铺,**2026/4/10 全面强制**
- 非欧盟公民入境要**采集指纹 + 人脸**,取代护照盖章
- 12 岁以下只采人脸不采指纹
- 已出现 **3 小时排队**(高峰期),ACI Europe 统计部分机场处理时间 **+70%**
- 首次登记在边境现场做,**不需要事先预约**
- 豁免期:2026/4/10 起可暂停 90 天 + 可选 60 天,最晚 2026/9 初结束

### 1.2 ETIAS 2026 Q4 启动(欧洲电子旅行授权)

**硬事实**:
- 预计 **2026 第四季度启动**(欧洲议会批准后定,目前为"预期")
- **€20,3 年有效**,18 岁以下和 70 岁以上免费
- 过渡期至少 **6 个月**,无 ETIAS 不会被拒入境,**仅对免签国家生效**(英美澳等 60+ 国)
- ⚠️ **中国护照不免签欧洲,走申根签渠道,ETIAS 对中国公民无直接影响**,但**持申根签的中国人也要过 EES 生物采集**
- ⚠️ **官方 ETIAS 网站目前未开放**,任何"代办 ETIAS"都是骗局

### 1.3 中国用户实操红线

- **首次抵欧机场留 30-60 分钟缓冲**(EES 首登记慢)
- **中转申根国转机时也要过 EES**(第一脚进申根国就登记)
- **不要通过第三方代办任何"ETIAS"**(中国护照用不到且官方未开放)
- 2026 夏季出行:查一下目的国机场是否启用豁免,豁免期内可能临时跳过 EES

## 二、扒手与治安(真正有信号的场景)

### 2.1 常识不讲清的地方

"巴塞罗那罗马巴黎扒手多" — 这是常识,但**哪个具体地点、什么时间、什么手法**才是真信号。下列是反复出现在领事馆被盗报告 + 本地游客论坛的**高频场景**:

| 城市 | 高风险场景 | 具体手法 |
|------|-----------|---------|
| **巴塞罗那** | La Rambla 大街 / Sagrada Família 地铁 L2/L5 站 / 海滩餐厅 | 分心三人组(一人问路一人碰瓷一人下手) |
| **罗马** | 地铁 A 线(Termini-Spagna-Ottaviano 段)/ 斗兽场外围 / 特米尼火车站 | 地铁门关前最后一秒下手、推挤 |
| **巴黎** | Châtelet-Les Halles 换乘站 / 蒙马特登山车 / 铁塔下草坪 / 凡尔赛 RER C | "签名请愿"女子团伙、假警察查证 |
| **佛罗伦萨** | SMN 火车站广场 / 圣母百花大教堂门前队伍 | 背包割底 |
| **那不勒斯** | 历史中心街区 / 地铁 Linea 1 | 摩托车飞车抢包(Scippo) |

### 2.2 中国游客特有风险

1. **奢侈品购物后的跟踪**(巴黎老佛爷/米兰 Montenapoleone 附近),有专门盯中国客的团伙
2. **"会说中文的同胞"骗局**:借手机打电话、问路、借钱,**欧洲普遍性**
3. **假警察查护照/查钱包**(巴塞罗那/罗马常见),**真警察不会当街查游客现金**
4. **机场打车黑车**(尤其 FCO 罗马菲乌米奇诺/FRA 法兰克福),**必须去官方 taxi 站或用 Uber/FreeNow/Bolt**

### 2.3 实操建议

- **护照复印件 + 数字扫描** 分开放,原件酒店保险箱(护照丢失补办在欧洲耗 1-2 周)
- **钱分三处**:衣兜 €50 现金、腰包/胸包主卡、酒店备用卡
- **单肩小包反斜挎**,包身在前,所有拉链朝胸
- **地铁上手不离包**,尤其车门关的 5 秒
- 万一被盗:立刻去最近警察局开**报案证明**(Denuncia),否则保险不赔

## 三、退税与购物(电子系统全面铺开)

### 3.1 三大电子系统(2026 现状)

| 国家 | 系统 | 特点 |
|------|------|------|
| **西班牙** | **DIVA** | 全电子,商户开电子 ERD 退税单,出境机场扫码;**已取消 €90.16 最低消费**(反直觉,任何金额都可退) |
| **法国** | **PABLO** | 机场自助机扫码,必须**最后离开欧盟前**验证,带货物 |
| **意大利** | **OTELLO** | 电子验证 |
| 其他申根国 | Global Blue / Planet 等第三方处理纸质或电子 | 看商户 |

### 3.2 最容易踩的坑

1. **"最后离境国原则"**:你在西班牙买东西,但从希腊飞回中国,要在**雅典机场**办退税,不是马德里 — 这一条 90% 中国游客不知道
2. **货物必须能检查**:退税时海关可能要看货,**不要放托运行李办完再值机,要先过退税再托运**
3. **退税比率真相**:VAT 20% ≠ 退 20%。扣服务费后实际到手 **12-15%**(最高约 16.66%)
4. **退税有效期**:购物日起 **3 个月内**验证;机场电子盖章/扫码后 **3 个月内拿钱**(信用卡退)
5. **现金退 vs 信用卡退**:现金立等可取但扣大额手续费(€3-5+%),**信用卡退 2-6 周到账但更划算**
6. **英国不再退税**(2021 年起),伦敦购物没退税了

### 3.3 实操建议

- 买之前问清楚"Tax Free?" + "DIVA/PABLO/OTELLO 还是纸质?"
- 收好**所有退税单据 + 购物小票原件**,打包装随身包
- 机场流程:值机前去退税区 → 验证/扫码 → 退现金柜台或邮寄
- 跨国行程:**买在任何申根国都行,办退税统一在最后离境那国**

## 四、支付方式(反直觉:不能一刀切)

### 4.1 欧洲支付地图(2025 ECB 数据)

**高度无现金(带少量现金即可)**:
- **荷兰** 店内 94% 无接触支付,现金份额 17%
- **芬兰 / 卢森堡 / 比利时 / 法国** 现金份额 28-43%

**中等(现金卡并用)**:
- **德国** 店内现金 63%,但无接触卡支付也在涨(69% 卡支付是无接触)— **文化依旧恋现金**
- **奥地利** 类似德国,现金依旧主流

**高现金依赖(必带现金)**:
- **意大利** 现金份额 61%
- **西班牙** 57%(但 **Bizum** 本地手机支付已普及,游客用不到)
- **希腊 / 葡萄牙** 类似

### 4.2 对中国游客的实操提示

1. **Apple Pay / Google Pay 在欧洲西北部全面通用**(荷兰/北欧/法国),绑中国信用卡即可用
2. **意大利/西班牙/希腊小店仍需现金**,尤其传统餐厅、市场、小旅馆
3. **国内"欧元现金需求"被高估**:随身 €200-300 够应急,**大额消费全部走卡**
4. **必须带 Visa/Master 信用卡**:国内"银联 62"卡片大部分欧洲不通
5. **微信/支付宝**:仅部分大城市+中国游客密集景点接受(巴黎老佛爷/米兰 Duomo/巴塞罗那机场等),**不是通用支付**,不能依赖
6. **小费**:详见第六节

### 4.3 ATM 取现实操

- 选**银行自有 ATM**(ING/BBVA/Intesa 等),避开 Euronet/Cardpoint 等第三方(坏汇率)
- **拒绝"DCC 本币结算"选项**(会问要不要按人民币结算,永远选欧元,汇率好 3-5%)
- 出门前查你的国内卡境外取现费(招行/中行/工行各不同,一般 1% + ¥15/笔)

## 五、语言与沟通(英语通用度的真相)

### 5.1 英语通用度梯度(基于本地大学生使用率 + 旅游服务从业英语水平)

| 层级 | 代表国家 | 游客实感 |
|------|---------|---------|
| **高**(英语 = 事实上第二语言) | 荷兰 / 北欧四国 / 德国大城市 / 奥地利维也纳 | 餐厅/酒店/景点英语无摩擦 |
| **中**(旅游业英语 OK,街边弱) | 意大利主城 / 西班牙主城 / 葡萄牙 / 法国巴黎 | 餐厅 OK,问路有困难 |
| **低**(旅游区也不一定) | 意大利南部小镇 / 西班牙非旅游城市 / 法国乡村 / 巴尔干 | 翻译 App 必备 |

### 5.2 中国游客最低工具包

1. **Google 翻译** 离线包(意/西/法/德预下载),拍照翻译最好用
2. **基础问候语**:"你好/谢谢/请问/不好意思"本地语 — 态度差别巨大,尤其法国/意大利
3. **关键场景短语卡**(菜单/药店/警察/医院)
4. **不要默认"欧洲人都会英语"**:法国人/意大利南部/西班牙非游客区,会英语但**故意不用**的也有(文化态度)

### 5.3 方言/身份红线(跨国通用)

- **加泰罗尼亚**(Barcelona 周边)不要说"你们西班牙人",他们用加泰语,自我认同非西班牙
- **巴斯克**(Bilbao/San Sebastián)同理,独立语言 Euskara
- **苏格兰/威尔士**(不要叫"英格兰人")
- **比利时弗拉芒区**(Flanders)说荷兰语,不是法语,别对布鲁日人说"你们法语区"
- **瑞士 4 语区**:苏黎世德语 / 日内瓦法语 / 提契诺意大利语 / 格劳宾登罗曼什语
- **Brittany(布列塔尼)**:凯尔特文化圈,不完全自认法国

## 六、小费与服务费(反常识:很多国家不用给)

### 6.1 跨欧洲小费规则速查

| 国家 | 餐厅小费 | 备注 |
|------|---------|------|
| **意大利** | **服务费已含**(coperto €1.5-3/人),通常不再给 | 特别满意可放 €1-2 |
| **西班牙** | 小费文化弱,**零头留下即可** | 高档餐厅 5-10% |
| **法国** | 账单已含 service compris,**可不给**;满意留 5% | 不像美国 15-20% |
| **德国 / 奥地利 / 瑞士** | 给 **5-10%**,直接告诉服务员总额 | "Zusammen 35 Euro" 意思付 35 |
| **荷兰 / 比利时** | **随意**,5% 或零头 | |
| **北欧** | **不用给**(工资含服务) | 偶尔给咖啡师 |
| **英国** | 账单含 service 12.5% 就不用另给 | 酒吧不给 |
| **希腊 / 葡萄牙** | 10% 左右 | |

### 6.2 反常识总结

- **美国 15-20% 的习惯不要带到欧洲**,多数地方会让服务员尴尬
- **意大利 coperto 不是小费**,是"餐位费/面包费",已经是服务费
- **不要给出租车司机小费**(除非帮搬行李)
- **酒店行李员 €1-2/件**,客房清洁 €1-2/天,放床头
- **导游小费**(如有跟团):一天 €5-10/人

## 七、生活作息(文化节奏撕裂)

### 7.1 跨欧洲作息差异(关键时间表)

| 项目 | 南欧(西/意/葡) | 中欧(德/奥/捷) | 北欧(荷/北欧) |
|------|----------------|----------------|----------------|
| 早餐 | 8-10 点 | 7-9 点 | 7-8 点 |
| 午餐 | **14-16 点**(西班牙最晚) | 12-14 点 | 12-13 点 |
| 晚餐 | **21-23 点**(西班牙最晚) | 18-20 点 | 17-19 点 |
| 商店关门 | 20-22 点 | **18-20 点** | 18-20 点 |
| 周日 | 多数关门 | **几乎全关** | 超市关,小店开 |
| 午休(西欧) | **14-17 点 Siesta** 小店关 | 无 | 无 |

### 7.2 中国游客的三大作息坑

1. **西班牙 20 点找餐厅吃饭扑空** — 晚餐 21 点才开始,**20:30 前到没人**;解决:tapas bar 是唯一早开的选项
2. **德国/奥地利周日全境关门** — 游客周日饿肚子;解决:火车站/机场/加油站便利店是唯一出路,或提前周六囤食
3. **南欧午休 14-17 点小店关** — 尤其 Córdoba/Sevilla/托斯卡纳小镇,博物馆也跟着关;解决:这段时间午休+吃长午餐,与当地节奏同步

### 7.3 楼层定义差异(酒店找房踩坑)

- **欧洲"Ground Floor"= 0 层,"1st Floor" = 中国的 2 层**
- 法国: Rez-de-chaussée = 地面 0 / 1er étage = 2 层 / 2ème étage = 3 层
- 德国: Erdgeschoss = 地面 / 1. Stock = 2 层
- 西班牙: Planta baja = 地面 / 1º = 2 层
- **酒店前台说"3 楼"意味着中国概念的 4 楼**,电梯按钮找不到"1"时按"0/RC/PB/EG"就是地面

### 7.4 公共假期(8 月尤其要避开)

- **法国 / 意大利 / 西班牙 8 月** 大量小店/餐厅老板放假 1-3 周,**尤其 8 月中下旬**
- **罗马 8/15 Ferragosto** 半城关门
- **德语区圣诞市场期** 11 月底-12/24 是旺季,12/25-26 全境关门
- **规划时查目的地 8 月 / 圣诞 / 复活节关门情况**

## 八、医疗与安全网(保险必备)

### 8.1 常识+实操

- **申根签保险** €30,000 保额是底线,**实际建议 €100,000+**
- **欧洲叫救护车 112**(全欧统一紧急号码),不是像美国的 911
- 药房(Pharmacie / Farmacia / Apotheke)**绿十字招牌**,基本药品可购但处方药要医生
- **晚上药房"On Duty"值班制**,门口贴纸告知最近值班药房地址
- **保险不能只买申根签最低,强烈建议加"旅行延误+行李延误"**(EES 导致的新型延误风险)

### 8.2 意外常见场景

- **意大利/西班牙烈日中暑**(7-8 月),带电解质粉
- **冰岛/北欧冬季摔骨折**(冰路),带防滑鞋套
- **阿尔卑斯高反**(少女峰 3454m / Jungfraujoch),有心脏病史须提前问医生
- **街头斗殴 / 足球日暴力**(英国/荷兰/德国周末某些酒吧街避开)

## 九、对产品的启示(跨国通用红线)

> 保留本节仅作**产品设计时的红线提醒**,不展开成"应该做什么产品"。

1. **EES/ETIAS 写进手账本开头**:用户 2026 去欧洲必踩生物采集,不讲清是失职
2. **支付方式按目的国拆**:告诉用户"去意大利带现金 / 去荷兰刷卡"而非一刀切
3. **退税流程独立模块**:最后离境国原则是 90% 用户不知道的盲区
4. **作息时间表给每国明示**:西班牙晚餐 21 点、德国周日关门,写在每日计划旁边
5. **方言身份红线写进文案**:加泰/巴斯克/苏格兰/瑞士 4 语区,用错伤感情
6. **扒手信号写具体场景不写笼统警告**:"Châtelet 换乘小心包" 比"巴黎有扒手"有用 10 倍
