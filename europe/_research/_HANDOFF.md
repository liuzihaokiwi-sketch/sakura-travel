### E5 欧洲三种玩法节奏(终身模式)
- 扫国派 / 一国一次派 / 一国多次派(三子类:近邻往返型 / 文化执念型 / 退休旅居型)
- **中国用户现状:80% 扫国派,2027-2030 主力迁移一国一次派;一国多次派欧洲方向短期不成主力,但日本/东南亚已是雏形**
- 预测:中国欧洲方向一国多次派以文化执念型为主(小众高忠诚)

### E6 跨国联游组合(天然 vs 勉强)
- **四维判断框架**:文化圈连续 × 交通硬连通 × 节奏匹配 × 申根/货币一致
- **6 个天然组合**:西+葡 / 意+希 / 奥捷匈 / 荷比卢 / 挪瑞丹 / 克斯+斯洛文尼亚+黑山
- **5 个勉强组合**:西+法 / 意+瑞 / 德+英 / 希+北欧 / 爱尔兰+欧陆
- **勉强但有真实需求**:法+瑞+意北(中国首刷经典,不劝退要驯化)
- **反直觉硬事实**:Madrid-Lisbon 2034 才通高铁 / 意希无直达雅典船 / 奥捷匈 Railjet 升级 2025/12
- **决策流程**:进欧门户 → 天数 → 人群风格 → 四维兜底

### E7 中国用户欧洲痛点宏观版
- EES 2026/4/10 生效,首次排队 +30-60min,豁免期至 2026/9 初
- VAT 退税 DIVA/PABLO/OTELLO 三大电子系统,**最后离境国原则**(90% 用户不知)
- 支付不能一刀切:荷兰/北欧/法国 Apple Pay 通用,意/西/德仍需带现金
- Sonntagsruhe:德奥瑞周日全境关门(连无人店都要关,写进《基本法》)
- 作息:西班牙晚餐 21 点 vs 德国 18 点关厨房
- 楼层:欧洲 Ground Floor = 0,1st Floor = 中国 2 层
- 扒手真信号:Châtelet / Termini / La Rambla 换乘点,不是城市笼统警告
- 方言/身份红线:加泰/巴斯克/苏格兰/瑞士 4 语区/比利时弗拉芒

2. **"你开太多 agent 了"** — 自己用 WebSearch/WebFetch 做,不要默认甩 agent
3. **事实层独立**(关西 4 层架构)— 订票窗口/Parador 细节不在研究阶段做
4. **E5 是终身玩法模式**,不是单次行程长短

6. **专家式研究法**:边搜边筛,**10 条真盲区 > 100 条常识复述**;每轮搜索之间停下思考
7. **数据真实性红线**:任何文档里写"事实"前问一句"这条是真搜过还是从上下文推的",推的要么去搜要么标"未验证"
