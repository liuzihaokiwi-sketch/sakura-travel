## 一、安全篇:骗术全清单(最关键)

### 1. 迷迭香骗局(吉普赛大妈)
- **手法**:景点 / 教堂门口,一脸热情伸迷迭香,"gratis(免费)",接了就开始念念有词,像算命 / 念咒,念完要钱,给少了会说"这是带来不幸的硬币"恐吓再给
- **防范**:**不搭理是最好的**;万一接了,**态度强硬还给她走开**,她不会上来围攻
- **高发地**:塞维利亚大教堂门口 / 格拉纳达 Alhambra 上下山 / 巴塞 La Rambla / 马德里 Plaza Mayor

### 2. 假警察骗局
- **手法**:2 人穿警服,要求出示护照 + 检查钱包,趁机偷钱 / 真钞掉包
- **升级版**:穿 Iberdrola 电力公司制服 + 工作牌 + Pos 机,借"查管道"骗户主钱;假冒巡警拦车
- **防范**:**真警察不会检查钱包**;要求他们到正规警局再谈;拍照记录工牌

### 3. 三人团伙扒手(Barcelona / Madrid / Sevilla / Granada)
- **经典配置**:一人分散注意(问路 / 地图 / 打翻咖啡) + 一人下手(前袋 / 背包 / 椅背包) + 一人挡路
- **变种**:
 - **清洁鸟屎**(故意弄脏游客衣服,借"帮忙擦"下手)
 - **搂抱接吻情侣**(表面亲热,一个顺手摸包)
 - **问卷调查**(clipboard 签名 = 100% 骗)
 - **时髦女人商场里**(盯目标 + 跟到试衣间外)
 - **地铁挤上车**(站门口最危险 + 过闸时"好心帮忙"= 骗)
- **数据**:巴塞 2025 年 266 个累犯被捕 1776 次(El País 7/21);马德里盗窃率 2014-2024 从 4.6% 涨到 9%

### 4. 餐厅椅背包偷窃
- **手法**:把包挂椅背 / 放地上 → **拿刀切带取走**(连人都感觉不到)
- **高发**:La Rambla 露天座 / 任何游客区露天餐厅
- **防范**:**包放腿上 / 夹腋下 / 两脚之间**,绝不挂椅背

### 5. 街头"赠品 / 服务"骗局
- 绑小绳 / 塞小树枝 / 卖鸽子食 → 讹钱
- 装聋哑人索签名
- "问路"搭讪
- **规则:陌生人递东西一律不接,陌生人搭话一律"No gracias"直接走**

### 6. DCC(动态货币转换)陷阱 — 刷卡时
- **手法**:店员结账时给你看"要用欧元还是人民币/美元结算",选非欧元 = **被加 3-8% 汇率差**
- **防范**:**坚持让店员刷欧元出来**;小票上看货币种类;已刷人民币/美元的**拒绝签字重刷**
- 所有卡都要注意,不限 Visa/MasterCard

### 7. 假出租车(机场 / 火车站)
- **防范**:用官方出租(马德里白底 + 红斜条,巴塞黑黄);或 **AMB app / Cabify / Free Now**(西班牙本地打车软件,**不是 Uber**,西班牙 Uber 大部分城市没有)

## 二、消费篇:钱的管理(中国用户独家需求)

### 支付策略(一句话总结)
**免货币转换费的 Visa/MasterCard 主刷卡 + 银联借记卡 ATM 备用 + 100-200 欧小额现金**

### 详细配比
| 工具 | 用途 | 注意 |
|------|------|------|
| **Visa/MasterCard(欧元)** | 主力刷卡 | **坚持欧元结算**(拒 DCC);带护照(名字核对) |
| **银联借记卡** | ATM 取现备用 | 花旗 / EUFISERV / EURO600 网络都支持银联 + 部分景区有中文界面 |
| **另一张信用卡** | 备用(不放一起) | 丢一张还有一张 |
| **100-200 欧现金** | 打车 / 小店 / 公厕投币 | **小面额**(10/20 欧最好);**避免 200/500 欧纸币**(商家不收) |
| **银联二维码 / 支付宝** | 部分大商户 | 覆盖率远低于国内;不要依赖 |
| **微信支付** | 极少数 | 基本用不上 |

### 入境申报
- **携带现金超过 10,000 欧元必须申报**(中国出境+欧盟入境都要)

### 欧元现金建议
- **在国内换好**(大行预约,手续费低)
- 机场汇率**最差**(应急可用,不要大量换)
- **避开 200/500 欧大面额**

### 信用卡防盗刷
- 每笔交易会 SMS 通知,不认的立即联系银行拒付
- 下好银行 APP,随时查可用额度

## 三、退税篇:流程 + 坑

### 退税基本
- 退"增值税 VAT",约 10-21%
- **满 €90.16 以上**(门槛已降,小额也可)
- 必须在**非欧盟国**(中国)用,货物在西班牙消费不退
- 手续费 + 汇率差后实际到手约 **8-15%**

### 流程(4 步)
1. **购物时**:告诉店员要 Tax Free,店员开免税申请表(粉联 / 客户联);留好发票
2. **离境西班牙机场时**:**先海关柜台盖章**(在行李过关前!)
3. **盖章后到退税柜台**(Global Blue / Premier Tax Free)办退
 - 选 **退现金**(立即,但汇率差大)或 **退信用卡**(1-2 周到账,划算)
4. **寄回表格**:把粉联放入退税专用信封投指定邮筒(已付邮资)

### 踩坑
- **海关盖章必须在过安检前**(过了海关就没法盖)
- **退现金扣手续费更重**,建议退信用卡
- 几笔小额都有单笔手续费 → **不要把小东西拆多单**
- **出境当天至少提前 3 小时到机场**(海关排队 + 退税柜台排队)

## 四、作息 / 餐食篇:适应西班牙节奏

### 吃饭时间表(和中国完全不同)
| 餐 | 西班牙时间 | 中国习惯 |
|----|----------|---------|
| 早餐 | 7:30-10:30(轻,咖啡+面包)| 7-8 |
| **Second breakfast** | 10-11(火腿/鸡蛋补一次)| (没有)|
| **午餐(主餐)** | **13:30-15:30**(Andalucía 14:30)| 12 |
| **Merienda** | **17-18:30**(咖啡+小点)| (没有)|
| **晚餐** | **21:00-23:00**(Andalucía 22-23)| 18-19 |

### 踩坑
- **12 点去吃午饭 → 餐厅要么没开要么在 siesta**(14 点才开)
- **18 点去吃晚饭 → 餐厅全闭,要等到 20:30 后**
- **不吃 Merienda → 晚饭前饿崩**(R1 核心洞察)
- **星期天 / 周一很多小店关**

### 点餐踩坑
- **英文菜单只给整份价,西文菜单给 ración / media ración / tapa 三档**——**指西文菜单点半份最划算**
- **"游客餐厅" = 挂满斗牛/Flamenco/吉他装饰 + 门口拉客 + 照片满墙菜单 + 英文菜单** → **必避**
- **Sangría + Paella 在游客餐厅 = 双重坑**;本地人夏天喝 Tinto de Verano / Cava / Txakoli,Paella 只在午餐吃
- **桌上的面包 / 橄榄 / 酱 = 要收费**,不想要就说 "no, gracias"
- **付账**:向服务员打手势做"签字"动作 = "要账单"(不要自己去吧台)

### 小费
- **不强制,没有美国习惯**
- 账单已含服务费
- 觉得服务好 → **放 1-2 欧小费**在桌子上,不用 5-10%

## 五、物流 / 生活细节篇

### 厕所(中国人最问)
- **只有 KFC / McDonald's / Burger King / Starbucks 有免费厕所**(星巴克可能要小票码)
- **地铁没有厕所**
- 大型百货 / 购物中心 / 大火车站有
- **公厕少 + 可能收费**(€0.50-1)
- **Caballeros = 男 / Señoras = 女**,别走错
- **部分公厕灯声控 / 触控**,蹲太久会黑,**挥手或出声**重亮

### 楼层
- **西班牙"一楼" = 0 楼(B 或 0)**,**中国"一楼"在这里叫二楼**
- **找酒店房间 / 电梯按键别按错**

### 星期日
- **大商场 / 超市 / 银行 / 邮局都关**(马德里巴塞中心例外)
- **博物馆 / 景点照开**
- **西班牙人周日:家庭午餐 + 散步 + 咖啡 + 家人相聚**
- **购物行程不要放周日**

### 酒店
- **空调**:很多欧洲老酒店没空调(尤其安达露西亚 / Sevilla 夏天),预订时查清
- **洗漱用品自带**:欧洲环保,酒店不送牙膏牙刷(拖鞋也常不送)
- **一楼不在一楼**(上条)
- **退房时间**(check-out)常 11-12 点,Check-in 常 14-16 点

### 语言
- **英文覆盖**:马德里 / 巴塞 / 塞维利亚 / 巴塞主要游客区 = OK;小镇 / 本地餐厅 = 可能只西语
- **会几个词 = 态度到了,服务会明显变好**:
 - Hola(你好)/ Gracias(谢谢)/ Por favor(请)/ Perdón(对不起)/ La cuenta(账单)/ No entiendo(我不懂)
- **区域专属**(文案红线,同 R1):
 - 加泰:**bon dia / gràcies / adéu**
 - 巴斯克:**kaixo / eskerrik asko / agur**

### 问路
- **问本地人,不问路人游客**
 - 百年老店的老爷爷 / 加油站柜员 / 出租车司机 / 警察
 - KFC/星巴克员工靠不住(有时瞎指)

## 七、签证篇(中国护照独家)

### 基本
- 西班牙 = 申根区,**申根签可进**
- 中国护照必须申请签证
- **至少提前 1 个月准备材料**
- 办理周期 15 工作日,旺季更长
- **可在上海 / 北京 / 广州 / 沈阳 / 成都 / 武汉 / 重庆 / 昆明 / 杭州 / 济南 / 福州 / 西安 / 长沙 / 南京 BLS 中心申请**

### 材料(核心)
- 护照(有效期 > 申根旅程结束 + 3 个月)
- 机票订单 + 酒店订单(全程)
- 旅行保险(**30,000 欧医疗保险**强制)
- 资金证明(银行流水近 3 个月 + 冻结证明)
- 在职证明 + 营业执照 / 学生证
- 签证申请表 + 照片(白底)

### 踩坑
- **保险金额 30,000 欧**(低于不过)
- **机票订单**:可用"不出票订单"(去哪儿 / 携程可帮订可退),**不要真出票**再被拒签
- **酒店订单**:Booking 可免费取消,全程订够再打印
