## 1. 灵魂 / day_mood

**一句话**:摩尔最后的叹息 + 洞穴吉普赛 + 免费 tapas — 安达露西亚最"异域"的城市。

Granada 是**基督教收复失地运动的终点**(1492 年),最后一个摩尔王国的都城。阿尔罕布拉宫是"摩尔建筑的巅峰也是终章"。整个城市的灵魂在三条线上:**山顶的摩尔宫殿 + 半山的吉普赛洞穴 + 老城里消失中的免费 tapa 文化**。

情绪基调:**比塞维利亚更静、更异域、更山城**(海拔 700m,背靠 Sierra Nevada 雪山)。**不是"西班牙的西班牙",是"还留着摩尔血液的西班牙"**。

**感官主导**:视觉(红墙 + 雪山 + 瓷砖)+ 听觉(zambra 的阿拉伯音阶 flamenco)+ 味觉(每杯酒送一盘 tapa)。

**和其他城市的分工**:
- Sevilla = 安达露西亚灵魂的正面(热 + 舞 + Flamenco puro)
- Granada = 安达露西亚灵魂的背面(冷 + 静 + 摩尔遗绪)
- Córdoba = 三教合一的中间态

**中国用户加分点**:韩剧《阿尔罕布拉宫的回忆》让亚洲游客对这里有情感预设。

## 2. 骨架 / 主线路线(2 日 + 可选夜游)

### Day 1 · Alhambra 全日(这是 Granada 的存在理由)

- **09:00-13:00 · Alhambra 主访问**
 - 开门就进(**Nasrid Palaces 时段票严格,过时不候**)
 - 顺序:Alcazaba(军事要塞,最早建)→ Palacios Nazaríes(主皇宫,票面时间进)→ Partal 花园 → Palacio de Carlos V(基督教国王加盖)→ 博物馆
- **13:30-15:00 · 午餐**(Alhambra 山下 Plaza Nueva 或 Realejo 区)
- 15:00-17:00 · **Generalife**(夏宫 + 花园)—— 可以拆到下午,Alhambra 票通常含全天
- **17:30-18:30 · Paseo 时段** — Carrera del Darro 河岸街(Alhambra 正下方最美街之一)
- 19:00-20:30 · **Mirador de San Nicolás 日落**(见第 3 节)
- 21:00-23:30 · 免费 tapas 跳店(见第 6 节)

### Day 2 · Albaicín + Sacromonte + 夜游 Alhambra(可选)

- 9:30-12:00 · **Albaicín** 白天走 — UNESCO 世遗,比 Alhambra 还早
 - 从 Plaza Nueva 沿 Carrera del Darro 上去
 - 清真寺改教堂、白墙 + 瓷砖、cármenes(带花园的宅子)
- 12:30-14:00 · **Mirador 早场**(San Miguel Alto 是本地人点,比 San Nicolás 安静)
- 14:00-16:00 · 午餐 + sobremesa(Albaicín 老城或 Plaza Nueva)
- 16:00-17:30 · **Capilla Real**(天主教国王夫妇 Ferdinand + Isabella 陵墓,1492 收复 Granada 那两人)
- 17:30-19:00 · Cathedral + 老城漫游(Alcaicería 旧丝绸市场)
- 19:00-21:00 · **Sacromonte** 洞穴区(见第 4 节)
- **22:00-23:30 · Alhambra 夜游 Nasrid Palaces**(全年开放,另外买票)或 Sacromonte zambra flamenco

## 3. 热门景点"怎么玩对"

### Alhambra(全西班牙最难订的景点)

| 维度 | 做对了 | 做错了 |
|------|-------|-------|
| **订票渠道** | **官网 tickets.alhambra-patronato.es**(唯一官方);电话 +34 858 889 002 | 其他网站多是加价或套路票 |
| **预订窗口** | **3 个月-1 年前订**;Semana Santa / 春-秋(4-9 月)/ Sierra Nevada 滑雪周末必须超早 | 3 周前订 = 旺季已无票 |
| **每天配额** | **每天 6500-8500 张**;Nasrid Palaces 每 30 分钟 300 人 | 不知道配额紧张 |
| **Nasrid Palaces 时段票** | **票面上的时间是硬的,过时不候,一秒钟都不让进** | 迟到 = 买了也进不去 |
| **取票** | **购票人本人携带支付用的银行卡 + 护照**才能取;不能别人代取 | 家人代取被拒 |
| **入场必带** | **护照或身份证原件**(数字扫描不够)+ 票(二维码或 A4 打印) | 忘带证件被拒 |
| **推荐时段** | **开门就进**(9:00)或接近关门(Nasrid 最后一场) | 中午-下午 2 点最挤 |
| **游览时长** | 3-4 小时全面看,不要赶 | 1.5 小时浪费票 |
| **票种选择** | "General"(含 Nasrid + Generalife + Alcazaba,全日)—— **确认含 Nasrid!** | 买错"Gardens only"进不了主宫殿 |
| **备用方案** | **Granada Card**(2-3 日版,含 Alhambra 白天票 + 11-15 个景点)/ 官方合作导游团 | 路边小旅行社"含 Alhambra"但可能不含 Nasrid |
| **夜游选项** | Nasrid Palaces 夜游(全年);Generalife 夜游(4-11 月);**Alhambra Experience**(夜游 Nasrid + 次日花园 combo) | 以为只有白天 |
| **最后手段** | 售完后 2 小时前窗口可能补放少量票(**赌博**,信用卡/现金) | 依赖这个 = 空手而归 |

### Alhambra 夜游(Granada 独家体验)

- **Nasrid Palaces 夜游**:灯光下的香桃木庭院(Patio de Arrayanes)+ 狮子庭院(Patio de los Leones)氛围完全不同
- **Generalife 夜游**(4-11 月):花园 + 音乐
- **Alhambra Experience combo**:夜游主宫殿 + 次日花园日游(两个时段,避开最挤的白天主宫殿时段)

### Capilla Real(皇家礼拜堂)
- Ferdinand + Isabella 夫妇陵墓 — **1492 收复 Granada + 送哥伦布出航 + 驱逐犹太人** 三件事同一对夫妇
- 历史重要性高,时间不够可作为 30 分钟快速参观

## 4. Mirador 日落 + Sacromonte 洞穴 + Flamenco

### Mirador de San Nicolás — Debussy 说"欧洲最美日落"

| 维度 | 做对了 | 做错了 |
|------|-------|-------|
| **时段** | **日落后留 30-60 分钟**——Alhambra 亮灯后景象完全变——大多数团在日落就走了 | 日落就离开 |
| **季节** | **11-4 月**:Sierra Nevada 雪山当背景 | 夏季看不到雪山 |
| **安全** | **扒手盯紧**,手机钱包贴身 | 忘形拍照被偷 |
| **替代视角(人少)** | **Mirador de San Miguel Alto**(Granada 最高 mirador,本地人+学生)/ Mirador de Carvajales / Mirador de San Cristobal | 只知道 San Nicolás |
| **餐厅包场** | Restaurante Huerto de Juan Ranas 正下方,同景无人挤 | 拍完就走没坐下 |

### Sacromonte — 洞穴区 + zambra Flamenco

Sacromonte 的身份清楚:**16 世纪犹太人和穆斯林被驱逐后,Romany(吉普赛)人从欧洲各地来,在 Albaicín 上方山壁凿洞居住,Flamenco 在这里诞生**。

**3,000+ 洞穴**,从 Plaza Nueva 步行 20-25 分钟(上坡),也可坐 **C34 小巴**(€1.40,10 分钟)。

**洞穴 Flamenco(Zambra)**:
- 30-50 人小场
- **洞穴拱顶自然放大脚踩声**,和 Sevilla tablao 的共鸣完全不同
- **吉他音阶偏小调 + 阿拉伯音阶**,不是 Sevilla 的 soleá/bulerías
- 主要场所:Venta El Gallo / Cueva de la Rocio / Jardines de Zoraya / Cuevas los Tarantos

**着装 / 鞋**:**绝不穿高跟鞋**,坡陡 + 鹅卵石路

### Flamenco 三选一决策

| 场所类型 | 气质 | 适合谁 |
|---------|------|-------|
| **Sacromonte 洞穴(Zambra)** | 阿拉伯音阶 + 洞穴声学,最异域 | 第一次来 Granada 的首选 |
| **Albaicín 小场 Tablao** | 室内 / 安静 / 传统 | 想专心看"纯"flamenco |
| **Peña flamenca**(居民社团) | 非商业 / 原始 | 真 flamenco 控 |

### 一天节奏(安达露西亚最晚型 + 山地修正)
- 早餐 8-9h
- **9-13h:Alhambra / 景点**(Nasrid 早场最松)
- 午餐 14-16h
- Siesta / 室内 16-18h(夏季 40°C+)
- Merienda 17-18:30h
- **日落时段 19-21h(Mirador)**
- 晚餐 21-23h(免费 tapas 跳店 / 坐餐)
- Flamenco 20:00 / 22:00 两场

### 季节
- **甜蜜点**:4-5 月(春花 + Sierra Nevada 雪尚在)/ 9-10 月
- **冬季 12-2 月**:**Alhambra 最松** + **Sierra Nevada 雪山背景** + 比马德里暖
- **夏季 6-8 月**:40°C+,Alhambra 清晨最对
- **Semana Santa**(3 月底-4 月初):浓但极挤,Alhambra 票极难订

### 一周
- Alhambra 每日开(**周一不关**,和大多数博物馆不同)
- Flamenco 洞穴每晚营业
- 多数景点周日缩短时间,老城商店周日关(参考 R1)

## 6. 餐饮骨架(免费 tapa 制度是核心)

### 免费 tapa 制度 — Granada 独家,全西班牙仅存

**规则**(写成简化版给用户):
1. 点一杯酒(caña / tinto de verano / 软饮多数也算) → **送一盘 tapa**
2. **不同的杯量送不同大小的 tapa**——小 caña 送小盘,doble 送大盘
3. **第二杯在同一家,tapa 会升级**(本地共识:到第三杯是最好的)
4. **跳店 vs 坐店**:一家一轮 hop 更"游客";本地人也常坐一家连点
5. **烈酒(rum / vodka)一般不送**
6. 厨房休息时段(16-20h)可能只送薯片 / 橄榄
7. 若忘记送了,说 **"Perdone, nos faltan las tapas"**(我们还没有 tapa)
8. **一杯 + tapa ≈ €2-3**,**一晚三轮就吃饱**
9. **别点 à la carte** — 免费送的已经够,点多了又贵又吃不完
10. 不能挑剔 — 送什么吃什么,素食 / 过敏第一杯就告诉服务员

### 最佳 tapa 街区 / 酒吧
| 区域 | 气质 | 代表店 |
|------|------|-------|
| **Cathedral 周边** | 本地餐桌 / 混合风 | Bar Fede / El Bar de Eric / La Loca de Gandoca |
| **Calle Elvira** | 中心半隐藏,竞争激烈 | 多家传统 |
| **Calle Navas** | tapas 主街之一,人气稳 | 多家 |
| **Calle Alhamar** | **大学生挑战路线**,一天走完全街 = 传奇 | — |
| **Albayzín 区** | 氛围 > 食物,瓷砖 + 白墙 + 传统 tapa | — |
| **Realejo 区(老犹太区)** | 本地日常,游客少 | — |
| **Plaza Albert Einstein** | 大学生区,超本地 | — |

### 具体必去酒吧
- **Bodegas Castañeda**(Plaza Nueva 附近)— 悬挂 jamón 腿 + flamenco 海报 + 气氛炸裂;站着吃;传统
- **Los Diamantes**(多家分店)— 炸海鲜强项,calamares / gambas / boquerones,问本地第一个说的就是这家
- **Casa Julio**(Plaza Nueva 附近)— 小 / 站着 / 混乱 / 招牌炸海鲜,出什么吃什么
- **La Blanca Paloma** — **berenjenas con miel**(蜂蜜炸茄子)的传奇
- **Chikito** — Federico García Lorca(格拉纳达本地诗人,1936 被法西斯杀害)最爱店,1909 开业

### 必吃 core_local_experience
- **Berenjenas con miel** — 蜂蜜炸茄子,Granada 招牌 tapa
- **Tortilla Sacromonte** — 猪 / 牛 / 羊的内脏煎蛋(猎奇但传统)
- **Remojón Granaíno** — 鳕鱼 + 洋葱 + 橄榄 + 橙子 + 石榴的冷沙拉
- **Rabo de toro** — 炖牛尾(Andalucía 共有)
- **Piononos** — Santa Fe 镇的小甜点(糖 + 奶油 + 海绵蛋糕,柱形)

## 8. 硬约束 / 游客坑

### 订票硬窗口(Granada 最严的一节)
- **Alhambra General**:**3 个月前订**(旺季 Semana Santa / 4-9 月应 4 个月前);官网 tickets.alhambra-patronato.es
- **Alhambra 夜游**:同样官网,分 Palacios / Generalife 两种
- **Granada Card**:官方卡含 Alhambra + 多景点,淡季次日可订,旺季需提前
- **Semana Santa**(2026:3 月 29 - 4 月 5 日):住宿翻倍,Alhambra 票极难
- **韩剧《阿尔罕布拉宫的回忆》后亚洲游客涌入** — 订票竞争更激烈(中文源独家)

### Alhambra 入场硬规则
- **取票必须购票人本人**,带支付用的**信用卡 + 护照**
- 入场必带**护照原件**(数字扫描不够)
- **Nasrid Palaces 时段票过时不候**(最大踩坑)
- **私家车禁入市中心** — 开车须走 Ronda Sur,推荐走路 / C3 或 C4 巴士 / 出租(Plaza Isabel 15-20 分钟)

### dress code
- Alhambra 无宗教要求,合理穿着即可
- Cathedral / Capilla Real:过膝 / 非无袖
- Sacromonte 山路:**坚决不穿高跟鞋**,鹅卵石 + 陡坡
- Mirador / Paseo:smart casual

### 安全
- **Mirador San Nicolás 日落扒手多**——手机钱包贴前袋
- Albaicín / Sacromonte 深夜小巷少亮 — 人多时走主路
- Granada 整体比巴塞马德里**安全得多**,但仍要警觉

### 日历
- **Alhambra 周一通常开**(反向红利)
- **Corpus Christi**(5 月底-6 月初)— Granada 最大节日,**整周庆典 + 斗牛节**,热但极浓
- **Día de la Toma**(1 月 2 日)— 纪念 1492 收复 Granada,有政治争议(穆斯林社区不认)
- **Semana Santa**(3 月底-4 月初)— 游行,圣像抬过窄巷很壮观
- **Festival Internacional de Música y Danza**(6 月下-7 月上)— Alhambra 内庭院音乐会(超稀缺票)

### 雨天 / 高温 Plan B
- Capilla Real / Cathedral / Alcaicería 旧市场
- Museum of Sacromonte Caves(洞穴博物馆)
- Carmen de los Mártires 花园(Alhambra 旁,免费,很少游客)
- 免费 tapas 酒吧(坐下慢吃)

### 信息分层
- **主线**:Alhambra(全日)/ Albaicín / Sacromonte / Mirador San Nicolás / 免费 tapas / Capilla Real
- **次级**:Cathedral / Sacromonte Abbey / Casa Museo Federico García Lorca / Carmen de los Mártires
- **附录**:Sierra Nevada 日归 / Santa Fe 镇 Piononos / 大学生 Calle Alhamar 挑战

### 中文源独家踩坑补充
- 韩剧《阿尔罕布拉宫的回忆》亚洲游客井喷 = 订票更卷
- 部分中文攻略仍在推 Ticketmaster 购票(**已失效**,只有官网)
- "一般票 ≠ 夜游票,夜游还分宫殿和花园"—— 买错票类型
- **开车进市中心 = 罚款** — 中国用户如自驾容易踩
