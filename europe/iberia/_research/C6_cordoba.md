## 1. 灵魂 / day_mood

**一句话**:世上唯一清真寺+大教堂融为一体的奇迹 + 百花庭院。

Córdoba 在西班牙的定位不同于前 5 城 — 它是**一座"浓缩的"城市**,主干只有 4 条:
1. **Mezquita-Catedral(清真寺-大教堂)** — 独一无二,850 根大理石柱 + 365 扇红白马蹄拱
2. **Judería(犹太区)** — 白墙小巷 + 百花 + Synagoga
3. **Alcázar de los Reyes Cristianos** — 基督教国王宫殿
4. **Roman Bridge + Guadalquivir 河** — 古罗马桥日落

整个老城 1994 年列入 UNESCO 世遗。规模小,**一日充足、两日不累、三日不够塞**。

**和其他城市的分工**:
- Sevilla / Granada = 住几天
- **Córdoba = 半日到一日游**,AVE 从 Sevilla 40 分钟,Madrid 1.5 小时
- **最佳用法:Madrid → Córdoba(stopover)→ Sevilla** 或 **Sevilla → Córdoba 日归**

情绪基调:**三教叠合的神秘感 + 小城慢节奏**。Córdoba 10-11 世纪是 Córdoba 哈里发国首都,占据整个伊比利亚半岛,**仅次于麦加和耶路撒冷的穆斯林朝圣地**——这座城"曾经是全世界的中心"。

**感官主导**:视觉(马蹄拱 + 百花庭院 + 白墙)+ 味觉(Salmorejo + Montilla-Moriles 酒)+ 空间感(庭院比街道凉 10-15°C)。

## 2. 骨架 / 主线路线(1 日为主,有 2-3 日选项)

### 1-日经典(从 Sevilla 或 Madrid 来的最常见用法)

- **9:00 AM 前抵达 Córdoba**(Sevilla 早班 AVE / Madrid 早班)
- 9:00-11:00 · **Mezquita**(主锚点,2 小时慢走)
 - 或 **8:30-9:20 免费时段**(周一-六)—— 但无 cathedral 访问权 + 无导览
- 11:00-12:30 · **Judería 犹太区 + Calleja de las Flores(百花巷)+ Synagoga**
- 13:00-14:30 · **午餐 Salmorejo + Rabo de toro**(老城 taberna)
- 15:00-16:30 · **Alcázar de los Reyes Cristianos** — 基督教国王宫 + 花园
- 16:30-17:30 · **Puente Romano 古罗马桥 + 河岸**
- 17:30-19:00 · **Patios de Córdoba(百花庭院)** — 5 月庭院节期间是主角
- 19:30+ · **河对岸看 Mezquita 日落亮灯**(摄影最佳)
- 20:30-22:00 · 晚餐(Plaza del Potro 或 Judería)
- **22:30 · 晚班回 Sevilla 或 Madrid**(或留宿)

### 2-日深度(加入)

- **Medina Azahara** — 10 世纪 Córdoba 哈里发城遗址,UNESCO 世遗,市郊 8 km
- **Palacio de Viana** — **12 个不同风格庭院**,"Córdoba 的 12 个庭院"
- **Flamenco 场次**(比 Sevilla 场数少,但有本地味)

### 3-日(延伸)

- **Montilla-Moriles 酒庄日归**(类 sherry 不强化的本地酒)
- **Almodóvar del Río** 城堡
- **Zuheros / Priego de Córdoba** 白村

## 3. 热门景点"怎么玩对"

### Mezquita-Catedral(主锚点)

| 维度 | 做对了 | 做错了 |
|------|-------|-------|
| **免费时段** | **周一-六 8:30-9:30 免费**(9:20 停止入场)—— **无导览团,无 Cathedral 通达,50 分钟探索** | 不知道这个花全价 |
| **最佳时段(体验)** | **16:00 之后进**(早班日归游客已走)或 **开门 10 AM**(Sevilla / Madrid 日归团 11-16h 高峰) | 中午 11-16h 高峰 |
| **周日特殊** | **上午 11:20 清场**,不可下午再进 | 周日上午买票 |
| **票价** | 成人 **€13**(官网);减价 €12(65+/学生 15-26)/ €8(10-14 岁);10 岁以下免 | 买贵票 |
| **Bell Tower** | **额外 €3 登塔,30 分钟时段制** | 不订错过 |
| **Cathedral 夜游 "Soul of Córdoba"** | €20,光影秀 + 最多 100 人,全年运营,需提前订 | 不知道有夜场 |
| **时长** | **1 小时最快,1.5-2 小时合理** | 45 分钟匆匆 |
| **Bonus** | 日票含 **"Fernandine Churches Route" 10 座教堂**(或另 €5) | 不知道赠送 |
| **着装** | **过膝 + 不无袖** | 被拒 |
| **体温** | **内部凉**(无窗),夏天要加一件 | 夏天只穿短袖冻 |
| **包** | **不能带大包**,没有寄存 | 背大包被拒 |

### Judería(犹太区)
- 免费进入,街道穿行
- **Calleja de las Flores(百花巷)** — 科尔多瓦最出片的巷子,蓝花盆 + 白墙 + Mezquita 钟楼背景
- **Sinagoga**(14 世纪)— 西班牙现存仅 3 座犹太堂之一
- **Puerta de Almodóvar** — 犹太区入口旧城门
- 晚上灯光下最美

### Alcázar de los Reyes Cristianos
- 基督教国王(Fernando + Isabel)宫 — 哥伦布曾在这里觐见他们
- 花园 + 喷泉 + 罗马马赛克
- **比 Sevilla 的 Alcázar 小得多**,不要期待同等级
- €5 左右

### Puente Romano + Torre de la Calahorra
- 古罗马桥建于公元前 1 世纪
- **日落时南端看 Mezquita 亮灯倒影** — Córdoba 最经典明信片
- 对岸 Calahorra 塔是博物馆(三教文化史)

### Medina Azahara(需日归或第二天)
- Abd ar-Rahman III 为宠妃建的哈里发城,10 世纪
- UNESCO 2018 世遗
- 8 km 郊外,需公交或打车
- **免费(欧盟居民)/ 非 EU 收费**(查官网最新)
- 3-4 小时

### Palacio de Viana
- **12 个不同风格庭院**(14-20 世纪渐次扩建)
- 可以只买庭院票不进建筑
- 是 Córdoba "百花庭院"概念的浓缩版,即使不在 5 月节也开放

## 4. 庭院节(Fiesta de los Patios)— 5 月专属

### 概况
- **2026 年:5 月 4 日 - 5 月 17 日**(一般是 5 月前两周)
- **UNESCO 2012 非遗**
- **50+ 私人庭院开放** — 免费进(导游团付费)
- 从 1921 年市长 Francisco Fernández de Mesa 开始的庭院比赛
- 6 条路线穿过历史老城

### 路线
- **Judería(犹太区)** — 最有名 + 最挤(和 Mezquita 同区,包 Mezquita 已经走过一遍)
- **San Basilio** — **早上 9 AM 前人少**(大多数团默认去 Judería)
- **Santa Marina** / **San Agustín** / **San Lorenzo** / **Centro**

### 逆向玩法
- **9 AM 前:San Basilio**(等待时间极短)
- **下午平日:Judería**(团散后)
- 相同的美,时间省一半

### 开放时段(庭院节期间)
- 11:00-14:00 + 18:00-22:00
- 早上早去最松

### 庭院凉爽原理
- 罗马时代的 peristylium → 摩尔的 riad → Cordobés 的 patio
- 夏天 40°C+ 高温,**庭院比街道凉 10-15°C**
- 中央喷泉 + 砖地 + 植物遮阳
- 这不是装饰,是气候功能

### 庭院节附加
- 歌手 + 舞者表演(Flamenco)
- Montilla-Moriles 葡萄酒随意流 + tapas

## 5. 最佳游玩时间 / 时段工程

### 一天节奏(安达露西亚,和 Sevilla 同步,晚型)
- 早到(日归客 9-10 AM 抵)
- Mezquita 10-12h(或 8:30 免费)
- 午餐 13-15h
- 午后犹太区 + Alcázar 15-17h
- 日落 Puente Romano 19-20h
- 晚餐 21-23h
- 回程或夜游

### 季节
- **甜蜜点 1**:**5 月(庭院节)** — 非去不可的季节,但人多 + 住宿贵
- **甜蜜点 2**:4 月 / 9-10 月 — 花开 / 温和
- **夏季 6-8 月**:**45°C+ 极热**(比 Sevilla 更热)— 严苛,必须庭院 / Mezquita(凉)避中午
- **冬季 11-2 月**:最松 + 温和 15°C+,Mezquita 免费时段最舒服

### 一周
- Mezquita 每天开
- **周日上午 11:20 清场** — 主要游客陷阱
- **博物馆大多周一关**(但 Mezquita / Alcázar 开)

## 6. 餐饮骨架

### 必吃 core_local_experience

| 菜 | 说明 | 哪里吃 |
|----|------|-------|
| **Salmorejo** | **Córdoba 招牌** — 比 gazpacho 更浓更奶(番茄+面包+橄榄油+大蒜,冷汤,撒火腿丁和蛋) — "喝一升都不够" | 任何本地 taberna |
| **Flamenquín** | 火腿 + 里脊卷炸 | 同上 |
| **Rabo de toro** | 炖牛尾(Andalucía 共享,Córdoba 也强) | |
| **Berenjenas con miel** | 蜂蜜炸茄子(Andalucía 共享) | |
| **Salmorejo 的表亲 "ajoblanco"** | 杏仁冷汤 | |
| **Alcachofas en Montilla** | Montilla 酒炖洋蓟 | |
| **Boquerones en vinagre** | 醋腌凤尾鱼 | |
| **Montilla-Moriles 酒** | Córdoba 本地雪莉风格 — **不强化**(不加烈酒),只有 Córdoba 省产 | 餐厅配菜 |

### 必去 taberna
- **Bodegas Mezquita Céspedes / Corregidor / Ribera** — Judería 区连锁,质量稳
- **Taberna Casa Pedro Ximénez** — 名副其实的 Pedro Ximénez 雪莉文化
- **Taberna El Abanico**
- **La Posada del Caballo Andaluz**
- **Mercado Victoria** — Andalucía 第一美食厅,~30 个摊位,从 salmorejo 到阿根廷 empanadas 到生蚝香槟

### 晚餐氛围
- **Plaza del Potro** — 小广场,几家酒馆 + 灯光好,氛围赞(La Siesta / Taberna La Alquería)
- **La Regadera** — 角落的现代精致餐,升档选择

### 时段红线
- **晚餐 9 PM 后再吃** — 本地人这时还在第一轮 vermouth(中文源独家)

## 7. Córdoba 的产品定位(最关键的一节)

### 用法一:Sevilla 日归(最高频)
- AVE 40 分钟,€11-34
- 早班 8-9h 出,晚班 20-22h 回
- 单日经典 + 晚餐回 Sevilla
- **提前订 2 张不同座位票**(来回 2 段需分别订)

### 用法二:Madrid → Córdoba → Sevilla 中转(高性价比)
- Madrid 早班(8 AM)→ Córdoba(9:30)→ 主体 1 日 → 夜班 20+ → Sevilla(21-22h)
- 寄包:**Córdoba 火车站对面的巴士站锁柜**(但常满,有风险)
- **最佳用法:两大城之间的"文化中转"**
- 一张旅程变 3 地,不额外住宿

### 用法三:1 晚住宿(深度)
- 只想在 Mezquita 待久一点 + 第二天 Medina Azahara / Palacio de Viana
- 淡季酒店便宜
- 更轻松的节奏,可以看夜场 "Soul of Córdoba"

### 用法四:不来(异端但值得考虑)
- 有一位英文博主说"如果重来会跳过 Córdoba,把时间给 Sevilla 或 Madrid"
- **对于 6-7 天紧凑行程 + 不关心伊斯兰建筑的用户**,这个判断可能成立
- 项目层面:Córdoba 不应是强制模块,**是用户勾选"想看三教融合"时的 opt-in**

## 9. 硬约束 / 游客坑

### 订票
- **Mezquita**:网上提前订,尤其 5 月庭院节 / 旺季
- **"Soul of Córdoba" 夜场**:100 人上限,提前订
- **Bell Tower**:30 分钟时段票,和 Mezquita 一起订
- **Palacio de Viana**:现场一般不紧,庭院节期间排队
- **5 月庭院节住宿**:**几个月前订,价格翻**

### dress code
- Mezquita:过膝 + 不无袖(活跃宗教场所)
- Alcázar / 犹太会堂:一般合理即可

### 安全
- **全西班牙旅游城市里最安全之一**(扒手比巴塞马德里轻多)
- 老城巷子白天 + 大多数晚上安全
- 夏天中午 40-45°C**中暑比扒手更危险**

### 日历
- **5 月 4-17 日**(2026):庭院节 — 最美但最挤
- **Semana Santa**(3 月底-4 月初):游行
- **Feria de Córdoba(5 月下旬)**:本地节庆
- **周日上午 11:20 清场**(Mezquita)
- **夏季 7-8 月**:过热,中午必须室内

### 雨天 / 高温 Plan B
- **Mezquita 室内凉爽**(+ 一件加衣)
- Palacio de Viana 有顶庭院
- Mercado Victoria 美食厅(有顶)
- Museo Arqueológico / Museo Julio Romero de Torres

### 中文源独家
- **"喝一升都不够"的 Salmorejo** — 比 gazpacho 浓奶
- **橘园中庭(Patio de los Naranjos)** — Mezquita 正门即见,穆斯林礼拜前净礼用水池
- **850 根大理石柱 + 365 扇马蹄拱** — 数字记忆 anchor
- **Mezquita 是仅次于麦加和耶路撒冷的穆斯林朝圣地**(历史 framing)

### 信息分层
- **主线**:Mezquita / Judería / Alcázar / Puente Romano / Salmorejo 一顿
- **次级**:Palacio de Viana / Medina Azahara / Synagoga / Calleja de las Flores
- **附录**:"Soul of Córdoba" 夜场 / Montilla 酒 / 5 月庭院节深度路线
