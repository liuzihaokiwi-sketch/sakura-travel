## 1. 灵魂 / day_mood

**一句话**:一座岛 = 一座城 + 一座山 + 一片海 —— 西班牙最紧凑的"三合一"目的地。

Mallorca 是巴利阿里群岛最大的岛(3640 km²,南北 75km × 东西 100km),首府 **Palma de Mallorca**。别把它当"纯沙滩度假岛"——Palma 本身是**一座 13 世纪起就有哥特大教堂 + 摩尔皇宫 + 完整老城**的城市,完全可以和 Córdoba / Granada 平级。加上西北侧 **Serra de Tramuntana(特拉蒙塔纳)山脉 2011 年 UNESCO 世遗**,一次旅行里同时拿到:

- **老城文化**(Palma Old Town + La Seu)
- **山区小镇**(Valldemossa / Deià / Sóller)
- **海岸**(Sa Calobra / Cala Llombards / Cap Formentor / Es Trenc)

对你给我的用户画像("时间有限、一次性、多城市")这是西班牙产品里最划算的岛。**Palma + 3 天 Mallorca 可以和 Barcelona 前后拼接**(巴塞飞 Palma 50 分钟 / 船 7 小时过夜)。

**感官主导**:视觉(43m 玫瑰窗 + 地中海蓝 + 石头山村)+ 味觉(sobrasada + ensaïmada + 橄榄油)+ 身体(自驾山路 / 复古小火车)。

**中国用户 anchor**:1946 年肖邦和 George Sand 在 Valldemossa Cartuja 修道院的冬天——最浪漫的音乐史场景。

## 2. 骨架 / 主线路线(4 天 Palma + Mallorca 建议)

### Day 1 · Palma Old Town 一日(主城)

- 9:00-11:30 · **La Seu 大教堂**(开门就进避邮轮潮)
- 11:30-12:30 · **Almudaina 皇宫**(隔壁,西班牙王室 Mallorca 正式官邸,14 世纪)
- 12:30-13:30 · **Parc de la Mar + S'Hort del Rei 王园**(免费,中世纪)
- 14:00-16:00 · **Santa Catalina 午餐 + 市场**(Mercat de Santa Catalina 1920)
- 16:00-18:00 · **Old Town 漫游** — Arab Baths / Basílica de Sant Francesc / Santa Eulàlia / 隐藏 patios
- 18:00-19:30 · **Passeig del Born + La Lonja** 傍晚
- 19:30-20:30 · **Paseo + 海边 / Parc de la Mar 看 La Seu 倒影**
- 21:00-23:00 · 晚餐(La Lonja 港口区 或 Santa Catalina)

### Day 2 · Valldemossa + Deià + Sóller(山村日)

- 9:00 · Palma 租车出发(或 Sóller 百年火车:Palma → Sóller 18€,35 分钟,1912 木制)
- 9:45-12:00 · **Valldemossa** — Chopin + George Sand 故居 Real Cartuja(1838 冬)+ 石头小巷
- 12:30-14:00 · **Deià** — Robert Graves 墓 + Cala Deià 小海湾午餐
- 14:30-17:00 · **Sóller** — 现代主义建筑 + 新鲜橙汁 + **Ferrocarril de Sóller 有轨电车** 到 Port de Sóller(7€)
- 17:00-19:00 · **Port de Sóller** 港口 + 海滨休息
- 19:00-21:00 · 回 Sóller 老城晚餐 或 Fornalutx 小村(全西班牙最美村之一)
- 夜宿 Sóller 或回 Palma

### Day 3 · Sa Calobra + 北岬(自驾挑战日)

- **早出发 / 或下午 3 PM 后出发**(旅游大巴 3 PM 后不能上路,之字路少很多危险)
- **Mirador Coll de Reis** 观景台看 Sa Calobra 之字路
- **Sa Calobra 海滩**(两崖之间,Torrent de Pareis 峡谷出口)
- 回程经 **Lluc 修道院**(Tramuntana 圣地)+ **Pollença** + **Formentor 灯塔日落**
- 或改走海路:Sóller 港坐渡轮到 Sa Calobra(50 分钟,一天 4 班)

### Day 4 · 南侧网红海滩(悠闲日)

- **Caló des Moro + Cala S'Almunia + Cala Llombards**(三连海滩,南侧)
- Moro 要走山路(未铺),停车紧张
- Almunia 相对人少
- Llombards 躺椅 €7.5 不限时 + 冰淇淋 + 日落
- 晚回 Palma 或继续住岛

### 替代北线(如不去 Sa Calobra)
- **Alcúdia 古城**(老罗马城墙 + 中世纪)→ **Port d'Alcúdia** → **Port de Pollença** → **Pollença**
- 岛东北角,比西南侧人少,适合不想开山路的

### 替代海线(奢华度假)
- **Es Trenc**(南侧加勒比色沙滩)
- **Cala Mondragó**(自然公园)
- **Cala d'Or** 别墅区

## 3. 热门景点"怎么玩对"

### La Seu(Palma 大教堂)

| 维度 | 做对了 | 做错了 |
|------|-------|-------|
| **时段** | **开门就进**避邮轮潮(4 艘邮轮同靠时队伍沿教堂绕一圈) | 中午到 |
| **玫瑰窗** | **早晨阳光透过东侧窗**是"Gothic Eye"(43m 直径,1200+ 块玻璃,世界最大之一) — 上午看最亮 | 下午看光不对 |
| **Gaudí 痕迹** | **铁制荆棘冠**悬在祭坛上方 — Gaudí 1903-1914 参与修复留下 | 不知道 Gaudí 碰过 |
| **现代艺术** | **Miquel Barceló 陶瓷装置**(Santísimo 礼拜堂) | 漏掉 |
| **高度** | 主殿 44m,全球最高哥特之一 | |
| **dress code** | **短裤 + 无袖均禁** | 被拒 |
| **票** | €10(教堂 + Sacred Art 博物馆) | |
| **屋顶 terrace** | 需 guided tour;有陡梯、恐高 / 儿童不适合 | 随便上 |

### Almudaina 皇宫
- 14 世纪阿拉伯堡改
- **西班牙王室 Mallorca 官方住所**(现任国王)
- 有 Flemish 挂毯 + 古董家具 + 皇家花园
- **花园看 La Seu 的角度最棒**

### Castillo de Bellver(贝尔维尔城堡)
- 14 世纪哥特,罕见**圆形城堡**
- Palma 西侧山头,公交或步行 40 分钟
- 全岛景观最佳之一(适合日落)

### Santa Catalina 区(Palma 最好吃的街区)
- 一条老渔村区改造,地中海+国际混合
- **Mercat de Santa Catalina**(1920,Mallorca 最老市场) — 50 摊,早上 7 AM 开 / 下午 5 PM 关 / **周日关**
- **Carrer de la Fàbrica** = "tapas 街"
- **Es Jonquet**(风车小区)— 老磨坊风车 + 海湾景

### Ferrocarril de Sóller(百年木制小火车)
- **1912 开通,木制车厢原件**
- Palma → Sóller → Port de Sóller(火车 + 有轨电车)
- 火车单程 €18 / 电车段 €7
- 本身就是景点,穿 Tramuntana 山脉隧道 + 橘林

## 4. Tramuntana 山脉(UNESCO 2011 世遗)

### 定位
- 西北侧长 90 km 山脉
- **梯田橄榄林 + 石墙 + 风车 + 老磨坊**
- 文化景观世遗(人与自然千年互动)
- GR221(Ruta de Pedra en Sec / 干石路)— 全欧经典徒步路线

### 三村精华
| 村 | 角色 | Anchor |
|----|------|--------|
| **Valldemossa** | **Chopin + George Sand 冬营**(1838)| Real Cartuja 修道院 — 他们住的房间还原 |
| **Deià** | 波希米亚艺术家村(石墙)| Robert Graves 故居 + 墓 |
| **Sóller** | "Golden Valley 金色谷"首城 | 现代主义建筑 + 橙子 + 火车终点 |

### Sa Calobra 之字路(西班牙最刺激之一)
- 从 Sóller 到 Sa Calobra 海湾
- **Mirador Coll de Reis 观景台**看之字路全貌(俯瞰)
- **限速 70,很多 hairpin**,**大巴 3 PM 后禁行**
- **下午 3 PM 后自驾**最佳(无大巴)
- 或走海路:Sóller 港 → Sa Calobra 渡轮(50 分钟,4 班/天)
- 海滩被两座石灰岩崖夹,**Torrent de Pareis 峡谷**出海口

### Cap Formentor(岛最北端"天涯海角")
- 需开 1 小时山路(小心驾驶)
- **灯塔日落**是全岛经典
- 免费参观,但停车紧
- **旺季大巴禁入** — 清晨或 3 PM 后自驾

### Lluc 修道院
- 山区圣地,13 世纪
- **卢克圣母(Moreneta)**— Mallorca 守护神
- 山中清静,全西班牙少有

## 6. 最佳游玩时间 / 时段工程

### 一天节奏(地中海岛型,比安达露西亚早一点)
- 早餐 8-10h(ensaïmada + 咖啡)
- 景点 10-13h(La Seu / Old Town)
- 午餐 14-16h(Santa Catalina)
- Siesta / 海 / 博物馆 16-18h
- **Paseo 18:30-20h**(Parc de la Mar 或 Es Jonquet)
- 晚餐 21-23h
- 夜 Santa Catalina 酒吧

### 季节
- **甜蜜点**:**4-6 月** + **9-10 月** — 温和 + 花 + 海温足以下水 + **非人潮高峰**
- **夏季 7-8 月**:**地狱人潮 + 酒店翻倍 + 本地人厌倦**(过度旅游是岛政治议题);海水最好
- **冬季 11-3 月**:**多数度假酒店关**,Palma 城市段开;温和 14-18°C 可以纯城市游
- **Mallorca 一年 300+ 晴天**

### 一周
- **La Seu**:周日关
- **Mercat de Santa Catalina**:周日关
- **多数景点周一开**
- **自驾山路周末 + 夏季**:大巴 + 游客车堵,**尽量平日**

## 8. 硬约束 / 游客坑

### 订票 / 租车
- **La Seu**:可现场,也可网上;邮轮日排队最长
- **租车**:**几个月前订**,Mallorca 租车旺季极度紧张
- **Sóller 火车**:旺季建议提前订,淡季现场
- **Sa Calobra 渡轮**:旺季需订
- **酒店**:4-10 月旺季酒店紧,7-8 月涨 50-100%

### 自驾
- **山路窄 + hairpin + 悬崖**,信心不足不要开
- **大巴 3 PM 后禁行 Sa Calobra** — 自驾最佳时段
- **Cap Formentor 旺季大巴禁入**
- **Palma 市区不建议租车开** — 停车位少 + 罚单多
- **加油**:Tramuntana 山上加油站少,Palma 加满再出发

### dress code
- La Seu:过膝 + 不无袖
- Valldemossa Cartuja:合理即可
- 海滩 → 市内:套一层(参考 R1)

### 安全
- **岛上总体很安全**,扒手比巴塞马德里轻非常多
- 海浪:Sa Calobra 峡谷 / Cap Formentor 有离岸流,注意旗帜
- 山路晒伤 + 脱水

### 日历
- **La Seu / Mercat / 多数商店周日关**
- **8 月 15 日**:Mare de Déu d'Agost(Mallorca 主要节)
- **圣诞节教堂**:La Seu 圣诞弥撒很浓
- **Cycling season**(1-5 月):职业车队训练,岛上自行车客暴增

### 中文源独家
- **公交分两种**:Palma 市内 = 现金 €2 / 长途横岛 = 刷卡有优惠
- **Sa Bodegueta 海鲜饭**(中国人点名)
- **橄榄市场物价便宜**,海鲜市场可帮处理
- **SUP 租板 €20 / 2 小时**(海上运动)
- **Llombards 躺椅 €7.5 不限时**
- **Calo des Moro 需走未开发山路**,停车极紧张

### 环境 / 政治红线
- **Mallorca 本地人已对游客过载不满**(over-tourism)— 7-8 月尤烈
- 不要在老城高声喊 / 吵闹;尊重 paseo 和 siesta
- 不买来路不明的"迷迭香"(吉普赛骗局类似 R3)

### 雨天 Plan B
- La Seu / Almudaina / Bellver Castle 室内
- Mercat de Santa Catalina(有顶)
- Museu Fundación Joan March / Palau March(现代艺术)
- CaixaForum Palma

## 9. 行程拼接建议

### 用法 A:Barcelona + Palma 拼接(最推荐)
- Barcelona 5 天 → 飞 50 分钟 / 夜船 7h → Palma + Mallorca 3-4 天
- 一趟旅行同时拿到:加泰设计 + 海岛文化 + 山脉 + 海滩
- **8-10 天行程最佳组合**
- 渡轮(Trasmediterranea)夜船省一晚酒店 + 醒来就在 Palma 港

### 用法 B:独立 Mallorca 5-7 天
- 适合**度假型用户**
- 2 天 Palma + 3-4 天自驾环岛 + 1 天海滩躺平

### 用法 C:不去
- 6-7 天西班牙紧凑行程 + 非度假型 → 跳过
