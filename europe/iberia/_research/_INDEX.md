## 文案红线总清单

### 区域身份
- **加泰 / 巴斯克 / 加利西亚 不自认 Spanish** — 禁用"西班牙人",用"加泰人 / 巴斯克人 / galego"或"这里的人"
- 安达露西亚 / 马德里 可以叫"西班牙人"

### 地名双语(4 区)
- San Sebastián = **Donostia** / Bilbao = **Bilbo** / Vitoria = **Gasteiz** / Basque Country = **Euskal Herria**
- Catalunya(不是 Cataluña)
- Galicia = **Galicia**(Galego 和西语同形)

### 问候词 4 套
- 标准西语:hola / gracias / adiós
- 加泰:**bon dia / gràcies / adéu**
- 巴斯克:**kaixo / eskerrik asko / agur / osasun / on egin**
- 加利西亚:**ola / graciñas / deica logo**

### 吃 / 喝
- Sangría + Paella 在 La Rambla / Plaza Mayor = 游客陷阱双坑
- 本地正解:Tinto de Verano / Cava / Vermut / Txakoli / Albariño / Caña / Zurito
- Paella 是午餐,不是晚餐
- Paella Valenciana ≠ 海鲜 paella(前者正宗,鸡兔豆)
- "挂满斗牛/flamenco/吉他装饰 + 英文菜单 + 门口拉客" = 游客餐厅(避)
