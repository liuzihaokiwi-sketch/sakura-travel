## 1. 灵魂 / day_mood

**一句话**:全球酒吧密度最高的湾 — 不在西班牙,在 Euskal Herria。

San Sebastián 官方有两个名字:西班牙语 San Sebastián / 巴斯克语 **Donostia**。两个名字都要认。它是**巴斯克自治区**的海滨首府,和 Madrid / Sevilla / Barcelona 气质完全不同:**欧洲化、克制、山海包围、美食密度冠全欧**。

三个层面的叠加:
- **自然底**:La Concha 湾(壳形完美湾)+ Monte Urgull + Monte Igueldo + Zurriola 冲浪海滩
- **文化底**:巴斯克语言 + 身份 + Franco 时代被镇压史
- **美食底**:**老城 Parte Vieja 246 家酒吧 = 全球酒吧密度最高**;25 公里半径内 19 家米其林星餐厅

情绪基调:**优雅 + 慢 + 有骨气**。不像 Andalucía 的热,不像 Madrid 的乱,不像 Barcelona 的游客化——**"小城的自尊"**。

**感官主导**:味觉(pintxos 密度)+ 视觉(壳形海湾)+ 风与浪。

**和其他城市的分工**:
- 南方(Sevilla/Granada)= 安达露西亚灵魂
- Barcelona = 地中海设计
- Madrid = 首都夜
- **San Sebastián = 巴斯克美食精致**

**中国用户的 anchor 点**:**Basque Burnt Cheesecake(巴斯克燃焦芝士蛋糕)发源地 La Viña** — 小红书/抖音上芝士蛋糕热的源头就在这里。

## 3. 热门景点"怎么玩对"

### Monte Urgull(老城后山)

| 维度 | 做对了 | 做错了 |
|------|-------|-------|
| **登上方式** | 从 Basílica de Santa María 后 或 San Telmo 博物馆旁 | 找不到入口走冤枉路 |
| **时长** | 30 分钟上;逛顶 + 下山 ≈ 1.5-2h | 低估陡坡 |
| **时段** | **午后 siesta 时段(14-16h)** — 城里酒吧全关,上山最舒服 | 晚上关(夏 21h / 冬 19:30) |
| **付费** | **免费** | 以为要买票 |

### Monte Igueldo(湾另一端)

- **百年木制 funicular**(1912),本身是景点
- 每 15 分钟一班
- 往返 €4.50
- **181m 高,San Sebastián 最佳全景**
- 山顶有 Mercure Monte Igueldo 酒店 + 老式游乐场(本身是童话感加分)

### San Telmo Museoa

- 巴斯克文化 + 历史
- **周二免费**(但要早)
- 16 世纪多米尼克修道院改建 + 现代翼
- 适合想深入理解巴斯克身份的用户,不是走过场

### Peine del Viento(风之梳)

- Eduardo Chillida 钢雕,Ondarreta 海滩尽头
- **海浪大时最震撼**(吹气孔会喷水)
- **日落时段 + 西风**是黄金组合

### San Martín 市场 + Bretxa 市场

- 两个本地市场都值得走
- 二楼看海鲜 / 火腿切片的节奏
- 可直接在市场里买 pintxo 坐着吃

## 4. Pintxos 深度(C5 最硬核的一节)

### Pintxos ≠ Tapas 的 5 条硬机制

1. **Pintxo 小,牙签(或 spike)固定**,和 Andalucía 的 tapa 长得不一样
2. **冷 vs 热两种服务**:
 - **冷 pintxos**:吧台玻璃柜下整排,自取,**牙签留着**,结账时数
 - **热 pintxos**:黑板手写,**厨房点单**——**游客 vs 本地的分界线**。只吃摆在吧台上的冷 pintxos = 错过这家的精华
3. **Honor System(荣誉制)**:吃完 pintxo 告诉吧台"我吃了什么 + 喝了什么",**一次性结账**。每吃一盘就付钱 = 游客暴露
4. **跳店 cardinal rule**:**一家吃 1-2 个就走**,不在一家把所有都点一遍。这是 pintxos 文化的核心,把一家吃死 = touristy mistake
5. **先问招牌**:"¿Cuál es la especialidad de la casa?" — 每家至少一样招牌,其他的可能平庸

### 时段
- **Hamaiketako**(11 AM 左右)— 巴斯克版 elevenses,小 pintxo + zurito
- **午餐 pintxos** — 12-14h
- **Siesta** 14-16h(大多数酒吧休)
- **晚间 pintxos** — 19-23h(高潮)
- **周一周日很多店关**,最佳是**周二-周四晚**

### 点什么喝的
| 饮品 | 量 / 价 | 说明 |
|------|--------|------|
| **Zurito** | 小杯鲜啤(比 caña 还小)~ €1.5 | pintxos 跳店首选,不会喝多 |
| **Kaña** | 标准小杯鲜啤 ~ €2 | 巴斯克也叫 kaña 或 caña |
| **Txakoli** | **当地白葡萄酒,微气泡** | **从高处倒入激气泡**(仪式感,出片)~ €2-3 |
| **Sidra** | 苹果酒 | 秋季新酒季最对,Astigarraga 村是圣地 |
| **Vermut** | | 一部分人 pintxos 前来一杯 |

### 必去老城酒吧(密度最高的 Calle 31 de Agosto)

| 店名 | 招牌 | 性质 |
|------|------|------|
| **La Viña** | **Basque Burnt Cheesecake 发源地**(浓奶香 + 淡奶酪味 + 松软) + 螃蟹浓汤 | **不能跳过**(中国用户 anchor 点) |
| **Ganbara** | **Funghi 野菌 pintxo**(金针菇+蛋黄)+ 海鲜 | 最受欢迎,排队常态 |
| **La Cuchara de San Telmo** | 热 pintxos(小羊排 / 伊比利亚猪)— **必须厨房点** | 无吧台陈列,最考验"点热 pintxo"门槛 |
| **Borda Berri** | 热 pintxos,肉炖 / risotto 风 | 拥挤 + 大声喊点 + 组织混乱但食物一流 |
| **Bar Sport** | | 老派 |
| **Txepetxa** | **凤尾鱼(anchoas)**各种法,一盘十几种 | 凤尾鱼控朝圣 |
| **Goiz Argi** | **烤串 / 虾串** | 烤海鲜精 |
| **Gandarias Jatetxea** | Brioche 串 + 伊比利亚 | 中文源点名 |

### A 轨 + B 轨(套 meal_role)

- **A**(第一晚):**La Viña + Ganbara + Goiz Argi** 三家跳店
 - 冷 pintxo(Ganbara)+ 热 pintxo 厨房点(La Cuchara de San Telmo)+ 凤尾鱼专题(Txepetxa)+ 甜点(La Viña 的 Basque Burnt)
- **B**(第二晚 surprise):**Gros 区(Zurriola 一侧)**——年轻 / 冲浪客 / 低游客密度
 - 或 **Bergara / La Espiga** 这些在 Gros 区的 pintxo 新派

### meal_role 配置(San Sebastián 专属)
| 餐 | 角色 | 说明 |
|----|------|------|
| Hamaiketako(11h) | 新增 `hamaiketako` / 归 `merienda` | 小 pintxo + zurito |
| 午餐 | 两选一:`pintxo_crawl` 或 `menu_del_dia` 餐厅 | 14-16h siesta 要尊重 |

## 5. 米其林 + 高端餐厅(San Sebastián 的另一面)

**25km 半径 19 家米其林星**,这是一个必须提到的维度。但不是主线——项目"降档也是体验"原则下,**pintxos 跳店 > 米其林**。只有高档用户 + 人群日晚餐 +1/+2 档才触发。

| 顶级 | 地位 | 价 |
|------|------|----|
| **Arzak**(Juan Mari Arzak) | 3★,"新巴斯克料理"教父 | €240+ |
| **Akelarre**(Pedro Subijana) | 3★,Monte Igueldo 山侧看海 | €240+ |
| **Martín Berasategui**(Lasarte-Oria,10km) | 3★,世界顶级之一 | €300+ |
| **Mugaritz**(Errenteria,10km) | 2★(曾 3★),先锋实验 | €280+ |

这些不是手账本主线,是用户勾选"体验 Basque 美食极限"时的升档选项。**订票 2-3 个月前**。

## 6. 最佳游玩时间 / 时段工程

### 一天节奏(巴斯克 = 西班牙北型,比安达露西亚早 30-60 分钟)
- 早餐 8-9h
- **Hamaiketako** 11h(小吃早场)
- 景点 9-13h(Monte Urgull / La Concha / 市场)
- **午餐 13-15h**(巴斯克比 Andalucía 早一小时)
- Siesta / 海 / 博物馆 15-17h(酒吧关)
- Merienda 17-18:30h
- Paseo 18:30-20h
- **晚 pintxos 高潮 20-23h**
- 夜 23-01h

### 季节
- **甜蜜点**:5-6 月 / 9 月 — 温 20-25°C + Txakoli 收成季 + 苹果酒季
- **夏季 7-8 月**:海滩季高峰,住宿贵,老城挤
- **冬季 12-2 月**:**酒店最便宜**(中文源省钱信号)但阴雨 + 海滩季结束
- **9 月**:**San Sebastián 国际电影节** — 9 月下旬,明星涌入
- **苹果酒季(Sidra,1-4 月)**:Astigarraga 村 sidrerias 打开大木桶,倒酒仪式 + 全牛餐

### 一周
- **周一周日**:**大量 pintxo 店关**(最大坑)
- **周二-周四**:空气最好
- **周五-周六**:挤但氛围最浓
- **周二 San Telmo 博物馆免费**

## 7. 巴斯克身份 + 语言(C5 文案红线)

### 身份红线
- **不是"西班牙人",是 Basque / vasco / euskaldun**
- **不叫"西班牙",叫 Euskal Herria / País Vasco**
- 中文里叫"西班牙北部海滨城市"= 失礼;叫"巴斯克自治区首府"= 对
- Franco 时代(1939-1975)Euskara 被禁,儿童在地下 ikastolak 学校学,**这段历史需要被尊重**——用户理解这一层会让他们更珍惜这趟

### 关键语 5 词(中国用户最低门槛)
| 巴斯克语 | 中文 | 场景 |
|---------|------|------|
| **Kaixo** | 你好 | 进店说第一句 |
| **Eskerrik asko** | 谢谢 | 离开时说 |
| **Agur** | 再见 | 离开 |
| **Osasun!** | 干杯 | 酒桌 |
| **On egin!** | 好胃口 / Buen provecho | 开吃 |

用得不对不要紧,**用 = 心意到了**。服务员听到游客说 Euskara,基本 100% 会笑。

### 地名双语
| 西班牙语 | 巴斯克语 |
|---------|---------|
| San Sebastián | **Donostia** |
| Bilbao | **Bilbo** |
| Vitoria | **Gasteiz** |
| Basque Country | **Euskal Herria** |

## 9. 硬约束 / 游客坑

### 订票
- **米其林餐厅(Arzak / Akelarre / Berasategui / Mugaritz)**:2-3 个月前订
- **9 月国际电影节**:住宿飙升,明星涌入
- **顶级 pintxo 店**:不订位,先到先得,**高峰期站着吃**
- **Monte Igueldo funicular**:不用订,每 15 分钟一班

### dress code
- 老城教堂:过膝 / 无无袖
- 老城 pintxo 店:随意,但不穿泳装进市内(R1 规则)
- 海滩 → 老城:泳装外要套一层
- 米其林:smart casual(不要牛仔)

### 安全
- **全西班牙大城市里最安全的之一** — 扒手比马/巴塞轻很多
- 老城深夜:大多数时候安全,但注意醉客区
- 海浪:Zurriola 冲浪区浪大,La Concha 最温和

### 日历
- **周一 / 周日**:**大量 pintxo 店关** — 最大坑
- **7 月第三周 Semana Grande**:城市最大节,烟火赛 + 游行 + 全城爆满
- **9 月下旬国际电影节**
- **1 月 20 日 Día de San Sebastián / Tamborrada**:24 小时全城击鼓,从 0 点到 24 点
- **8 月 14-15 日 Aste Nagusia**:周年庆
- **11-3 月**:阴雨多,海滩季结束

### 雨天 / 高温 Plan B(San Sebastián 是北型,**阴雨常态**)
- San Telmo Museoa
- Mercado San Martín(有顶)
- Tabakalera 艺术中心
- **Pintxo 跳店本身就是全天候**(室内)
- Monte Urgull(伞)

### 语言
- **老城游客多,英语通** — 但老派店只会 Euskara / 西班牙语
- **开口说 Kaixo / Eskerrik asko** — 态度比发音重要

### 信息分层
- **主线**:La Concha / Monte Urgull / Parte Vieja pintxos / San Telmo / Monte Igueldo / Peine del Viento
- **次级**:Zurriola / San Martín Market / La Viña / Miramar 宫 / Basílica de Santa María
- **附录**:Tabakalera / Zumaia / Getaria / Gaztelugatxe / Astigarraga sidreria / 米其林 3 星

### 中文源独家补充
- **Basque Burnt Cheesecake 发源地就在 La Viña** — 小红书打卡源头
- **Pintxos 单价 2-3 欧**(精致 4-5 欧),酒 2 欧以内 — 性价比
- **权游梗 2 处**:Zumaia Flysch 龙石岛 / Gaztelugatxe 另一处龙石岛取景
- **Tabakalera 是小红书较少提及的宝藏地** — 烟厂改文化中心 + 2015 新
